# Claude Code 教程课程章节目录

## 课程概述
本课程系统讲解 Anthropic 公司推出的 Claude Code 工具，帮助学习者掌握如何利用大模型辅助编程，实现从代码探索、功能添加、测试调试到项目重构的完整工作流程。

---

## 第一章：导论与基础认知
本章要解决什么问题：帮助学习者理解 Claude Code 是什么、它能做什么、以及如何正确评估其在实际开发中的价值

- **1.1 课程介绍** [P1]  
  了解课程目标、学习路径和前置要求

- **1.2 Claude Code 概述** [P2]  
  明确 Claude Code 的定位、适用场景与局限性

---

## 第二章：环境配置与项目理解
本章要解决什么问题：掌握 Claude Code 的正确启动方式，学会让 AI 高效理解陌生代码库的结构

- **2.1 设置和代码库理解** [P3]  
  环境初始化配置与向 Claude 传递项目上下文的最佳实践

---

## 第三章：核心功能开发流程
本章要解决什么问题：运用 Claude Code 完成实际的代码编写、测试和重构任务，建立与 AI 协作的工程思维

- **3.1 功能添加与上下文管理** [P4]  
  如何在保持上下文连贯性的前提下让 Claude 添加新功能

- **3.2 测试驱动调试与代码重构** [P5]  
  利用 Claude 编写测试用例、定位 bug 并进行代码优化

- **3.3 多会话并行与 Git 工作树** [P6]  
  同时处理多个功能需求，以及借助 Git 分支管理 AI 生成代码的技巧

---

## 第四章：高级集成与协作机制
本章要解决什么问题：扩展 Claude Code 的使用边界，将其深度融入现有开发工具链和工作流

- **4.1 GitHub 集成与 Hooks 机制** [P7]  
  将 Claude Code 接入 GitHub 实现自动化代码审查与提交检查

---

## 第五章：实战项目拓展
本章要解决什么问题：通过端到端的完整项目案例，演示如何将 Claude Code 应用于数据分析和前端开发场景

- **5.1 Jupyter 笔记本重构与 Streamlit 仪表盘** [P8]  
  使用 Claude Code 重构数据分析笔记本并快速构建可视化仪表盘

- **5.2 Figma 原型到 Web 应用** [P9]  
  将设计稿描述转化为完整的 Web 应用代码

---

## 第六章：总结与展望
本章要解决什么问题：回顾核心知识点，明确 Claude Code 在个人开发流程中的最佳使用位置

- **6.1 课程回顾与展望** [P10]  
  梳理课程体系，展望 AI 辅助编程的未来趋势
