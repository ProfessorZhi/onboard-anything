# 课程练习题

## 第一章：导论与基础认知

### 一、结构识别题

**1.1** 填空题  
Claude Code 是由 __________ 公司推出的 AI 编程辅助工具，旨在帮助开发者完成从代码探索、功能添加、测试调试到项目 __________ 的完整工作流程。

**1.2** 单选题  
以下哪个不是 Claude Code 的核心定位？
- A. 代码自动补全工具
- B. 代码库理解助手
- C. 功能开发协作者
- D. 自动化测试生成器

**1.3** 判断题  
Claude Code 只能处理小型项目，不适合大型代码库的理解和维护。（ ）

### 二、机制理解题

**1.4** 简答题  
课程中提到的 AI 辅助编程工具（如 Claude Code）与传统 IDE 智能补全（如 Copilot）有何本质区别？请从交互模式和任务范围两个维度进行分析。

**1.5** 论述题  
为什么说 Claude Code 更适合“协作式”而非“自动化”的编程场景？请结合课程中提到的适用场景和局限性进行阐述。

### 三、最小实现题

**1.6** 实践操作题  
假设你是一名后端开发者，需要使用 Claude Code 辅助开发一个新的 REST API 端点。请列出你计划如何向 Claude 描述这个任务，至少包含：需求描述、预期输入输出、技术栈说明。

## 第二章：环境配置与项目理解

### 一、结构识别题

**2.1** 填空题  
Claude Code 的环境初始化配置主要涉及两个层面：__________ 配置和向 Claude 传递 __________ 的最佳实践。

**2.2** 多选题  
在让 Claude 高效理解陌生代码库时，以下哪些做法是正确的？
- A. 提供项目目录结构概览
- B. 说明使用的技术栈和框架
- C. 告知项目的入口文件和主要模块

### 二、机制理解题

**2.3** 简答题  
课程 P3 中提到的“上下文传递”机制是什么？请解释为什么在多轮对话中保持上下文连贯性对代码开发至关重要。

**2.4** 对比分析题  
比较以下两种向 Claude 描述代码库的方式，说明哪种更有效及其原因：  
方式 A：直接提问“我需要添加一个登录功能”  
方式 B：先描述项目结构，再说明“我正在使用 React + Node.js 构建电商后台，需要在 /api/auth 路径下添加 JWT 登录功能”

### 三、最小实现题

**2.5** 实践操作题

```
├── app.py

├── models/
│   ├── __init__.py
│   └── user.py
└── routes/
    ├── __init__.py
    └── api.py
```
请设计一段向 Claude 传递项目上下文的 prompt，要求包含：技术栈说明、关键文件作用描述、当前任务目标。

## 第三章：核心功能开发流程

### 一、结构识别题

**3.1** 填空题

- A. 编写测试 → 运行测试 → 让 Claude 根据失败测试生成代码
- B. 让 Claude 生成代码 → 编写测试 → 运行测试验证
- C. 编写部分代码 → 编写测试 → 让 Claude 补全剩余代码

**3.3** 判断题

**3.4** 多选题

- A. Git 分支管理
- B. 多会话并行
- C. 代码自动生成
- D. 工作目录隔离

### 二、机制理解题

**3.5** 简答题

**3.6** 机制分析题  
课程 P5 中提到的“测试驱动调试”工作流中，测试用例扮演了什么角色？请分析这种模式相比直接调试的优势。

**3.7** 设计思考题  
在多会话并行处理多个功能时，为什么需要使用 Git 分支？这样做可以避免哪些问题？如果不使用分支直接在不同功能间切换，可能产生什么风险？

### 三、最小实现题

**3.8** 编程实践题  
假设你需要在 Node.js Express 项目中添加一个分页查询接口。请完成以下任务：

1. 用自然语言描述这个功能需求，包括：路由路径、请求参数、响应格式
2. 设计至少两个边界测试用例（正常数据、空数据、非法页码）
3. 写出你将用于向 Claude 发送的完整 prompt

**3.9** 代码重构题  
以下是一段存在问题的 JavaScript 代码，请说明你会如何向 Claude 描述重构需求：

```javascript
function processData(data) {
  var result = [];
  for (var i = 0; i < data.length; i++) {
    if (data[i].active == true) {
      result.push({id: data[i].id, name: data[i].name.toUpperCase()});
    }
  }
  return result;
}
```

请列出三个具体的重构目标，并设计相应的 prompt。

**3.10** 工作流设计题  
假设你需要使用 Claude 完成一个用户注册功能，包括：表单验证、数据库存储、邮件通知。请设计一个完整的多会话工作流程，标明每个会话的目标和使用的 Git 分支命名。
