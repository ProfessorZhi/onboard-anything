# 知识框架

## 课程定位与目标

本课程聚焦 Anthropic 公司推出的 Claude Code 工具，目标是掌握大模型辅助编程的完整工作流程，覆盖从代码探索、功能添加、测试调试到项目重构的全链路能力。

## 能力边界

Claude Code 的能力范围：

能做：
- 代码理解与解读
- 代码生成与修改
- 测试用例编写
- Bug 定位与修复建议
- 代码重构与优化
- Git 操作辅助
- 项目上下文管理

不能做或有限制：
- 完全替代开发者的架构决策
- 处理极复杂的业务逻辑理解
- 保证生成的代码完全无错误
- 替代对业务需求的深度理解
- 替代代码审查中的人类判断

## 学习路径建议

1. 起点：导论与基础认知（理解工具定位）
2. 入门：环境配置与项目理解（学会正确启动）
3. 核心：功能添加 → 测试调试 → 多会话管理 → GitHub 集成（递进掌握开发流程）
4. 实战：Jupyter 重构 → Figma 到 Web（综合应用）
5. 收尾：总结与展望（形成完整认知）
