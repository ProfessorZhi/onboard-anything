# P71 资源清单 - 【网络爬虫】requests基本使用
---

## 1. requests

- **类型**：Python库 / HTTP请求库
- **在本P中的作用**：核心工具，用于发送HTTP请求（GET/POST等），获取网页、图片、音频等网络资源
- **来源**：字幕直接提到
- **建议方向**：深入学习requests的高级特性，包括Session对象、代理设置、请求重试、超时控制、文件上传下载等

---

## 2. Kenneth Reitz

- **类型**：人物 / 开源开发者
- **在本P中的作用**：requests库的作者，被称为Python语言的总架构师
- **来源**：字幕直接提到
- **建议方向**：了解其其他开源作品（如python-guide、requests源码架构），学习其代码设计风格

---

## 3. pip

- **类型**：命令 / Python包管理工具
- **在本P中的作用**：通过`pip install requests`命令安装requests库
- **来源**：字幕直接提到
- **建议方向**：学习pip的其他常用命令（pip list、pip show、pip uninstall、pip freeze等），了解虚拟环境管理（venv/virtualenv）

- **类型**：软件 / 集成开发环境（IDE）

- **来源**：字幕直接提到
- **建议方向**：熟悉PyCharm的调试功能、代码补全、HTTP Client工具链，学习使用远程解释器和Docker集成

- **建议方向**：掌握命令行操作、切换工作目录、激活虚拟环境等

## 12. urllib

- **类型**：Python标准库
- **在本P中的作用**：Python内置的HTTP请求库，requests是对其的封装，作为知识对比延伸
- **来源**：基于上下文推断（提到requests是urllib的封装）
- **建议方向**：了解urllib的使用方法，对比requests与urllib的API差异和性能区别

---

## 13. HTTP协议

- **类型**：网络协议
- **在本P中的作用**：requests库操作的底层协议基础，理解请求方法（GET/POST）、请求头、响应状态码等概念
- **来源**：基于上下文推断
- **建议方向**：深入学习HTTP协议细节（缓存机制、压缩传输、HTTPS/TLS握手过程）

---

## 14. Unicode / UTF-8 / GBK

- **类型**：字符编码标准
- **在本P中的作用**：.decode()解码时的字符集选择，解决中文乱码问题的关键知识
- **来源**：字幕直接提到
- **建议方向**：深入理解字符编码原理（ASCII、Unicode、UTF-8关系），学习处理更多编码类型（如GB2312、Big5）

---

## 15. User-Agent

- **类型**：HTTP请求头字段 / 反爬概念
- **在本P中的作用**：识别客户端身份的关键字段，设置浏览器UA伪装成正常浏览器访问是第一种反爬应对手段
- **来源**：字幕直接提到
- **建议方向**：学习构建User-Agent池实现随机切换，了解更多反爬机制（IP限制、验证码、Cookie验证等）

---

## 16. cookies

- **类型**：HTTP机制 / 会话概念
- **在本P中的作用**：response.cookies和response.request.cookies属性用于获取和传递会话信息
- **来源**：字幕直接提到
- **建议方向**：理解Cookie与Session的机制差异，学习在爬虫中维护登录状态和处理Cookie过期

---

## 17. GitHub

- **类型**：平台 / 代码托管网站
- **在本P中的作用**：requests库源码托管地址，可查阅官方文档和源码
- **来源**：基于上下文推断（开源Python库的常规托管平台）
- **建议方向**：阅读requests源码了解其架构设计，查阅官方文档学习完整API，参考其他优秀爬虫开源项目

---

## 18. response对象

- **类型**：Python对象 / 数据结构
- **在本P中的作用**：requests.get()返回的对象，包含text、content、status_code、headers、cookies等属性
- **来源**：字幕直接提到
- **建议方向**：深入了解response对象的更多方法（如json()、iter_content()大文件分块读取）
