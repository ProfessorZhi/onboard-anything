# P75 资源清单 - 【网络爬虫】jsonpath数据提取和案例
---

## 一、核心依赖库

**jsonpath**

- 类型：Python 第三方库
- 作用：实现 JSONPath 路径表达式语法，从复杂嵌套 JSON 数据中精准定位并提取目标字段
- 来源：字幕直接提到，命令 `pip install jsonpath`
- 建议方向：结合本地 JSON 文件或公开 API（如 JSONPlaceholder、豆瓣 API）练习不同路径表达式的写法与效果

**Python json 模块**

- 类型：Python 内置标准库
- 作用：提供 JSON 字符串与 Python 对象之间的双向转换（loads/dumps），以及文件级读写操作（load/dump）
- 来源：字幕直接提到，直接 `import json` 使用
- 建议方向：练习处理包含嵌套对象、数组、布尔值、空值、中文等复杂结构的 JSON 数据；对比 `ensure_ascii` 和 `indent` 参数的效果

**requests 库**

- 类型：Python 第三方库
- 作用：发送 HTTP GET/POST 请求，获取 API 响应数据，本 P 中用于抓取小饭桌接口
- 来源：基于上下文推断（本 P 依赖但非重点，属于前置知识）
- 建议方向：配合本 P 的 JSON 解析流程，练习对真实 API 接口的完整抓取与解析闭环

## 二、工具与环境

**浏览器开发者工具（DevTools）Network 面板**

- 类型：浏览器内置工具（Chrome / Firefox / Edge 等）
- 作用：拦截并查看网页发出的 AJAX 请求，获取真实数据接口的 URL、请求参数、响应格式
- 来源：字幕直接提到，“在开发者工具 Network 中抓包”
- 建议方向：用 DevTools 分析任意一个动态加载数据的网页（如新闻列表、商品列表页），找出数据接口并手动构造请求

**Python 运行环境**

- 类型：开发环境
- 作用：执行本 P 中的所有代码演示，包括 json 模块调用、jsonpath 提取、requests 请求
- 来源：字幕直接提到
- 建议方向：建议搭配 Jupyter Notebook 或 VS Code 调试模式，便于逐行观察变量值和断点排查

## 三、目标网站与数据源

**小饭桌网站（xiaofanzhuo.com）**

- 类型：目标网站（金融投资资讯平台）
- 作用：本 P 实战演示对象，数据通过 AJAX 异步加载，返回深层嵌套的 JSON 结构，用于练习 JSONPath 提取和翻页抓取
- 来源：字幕直接提到，“打开小饭桌网站”
- 建议方向：在遵守 robots.txt 和网站使用条款的前提下，练习抓取其融资资讯、热门文章等列表数据；注意部分字段可能缺失，需处理异常情况
