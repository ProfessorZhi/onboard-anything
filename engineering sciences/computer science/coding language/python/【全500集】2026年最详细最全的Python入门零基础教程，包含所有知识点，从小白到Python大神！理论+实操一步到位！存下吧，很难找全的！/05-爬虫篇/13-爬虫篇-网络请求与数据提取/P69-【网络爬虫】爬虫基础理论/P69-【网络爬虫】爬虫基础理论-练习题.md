## 结构识别

**题目 1（爬虫类型识别）**  
以下场景分别对应哪种爬虫类型？请说明判断理由。

1. 百度搜索引擎定期抓取全网网页，建立索引供用户搜索
2. 某电商平台抓取竞品商品的价格和销量数据
3. 新闻网站每隔30分钟检查并抓取最新发布的文章
4. 抓取一个通过JavaScript动态加载内容的页面

---

**题目 2（概念正误判断）**  
判断正误，并说明理由。

1. 通用爬虫和聚焦爬虫是按照实现技术分类的。
2. 增量式爬虫属于聚焦爬虫的优化变体。
3. 只要网站根目录有robots.txt文件，就必须遵守其中的规定。

## 最小实践

**题目 6（场景方案设计）**  
某自媒体运营者希望每天自动获取"豆瓣电影TOP250"榜单中新增或排名变化的电影信息。请回答：

1. 应选择哪种爬虫类型？为什么？
2. 请用箭头表示完整的工作流程。

---

**题目 7（robots.txt分析）**  
假设某网站 robots.txt 内容如下：

```
User-agent: *
Disallow: /user/
Disallow: /admin/
Allow: /public/
Sitemap: https://example.com/sitemap.xml
```

请回答：
1. 哪些路径明确禁止爬虫访问？
2. `/public/` 路径的爬虫访问权限是什么？
3. 搜索引擎爬虫必须遵守以上规则吗？
