- **类型**：搜索引擎

### 百度 robots.txt
- **类型**：协议文件
- **在本P中的作用**：展示Robots协议的标准格式（User-agent、Disallow等字段）
- **来源**：字幕直接提到

- **后续可练习方向**：访问 taobao.com/robots.txt，观察禁止声明与实际数据流通的反差

### 谷歌 / 搜狗 / Bing

- **在本P中的作用**：列举通用爬虫的典型应用场景，补充说明搜索引擎的工作原理
