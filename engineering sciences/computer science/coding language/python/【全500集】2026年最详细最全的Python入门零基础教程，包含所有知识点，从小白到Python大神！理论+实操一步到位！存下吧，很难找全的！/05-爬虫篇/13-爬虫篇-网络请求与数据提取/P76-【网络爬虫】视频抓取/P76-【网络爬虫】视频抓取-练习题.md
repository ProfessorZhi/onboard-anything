## 机制理解

**Q3：为什么 B 站必须携带 Referer 和 Cookie？**

从服务端身份验证的角度解释：如果请求缺少这两个 Header，B 站会如何判断该请求，并产生什么响应？如果只缺少其中一个，结果是否相同？请说明你的推断依据。

**Q4：音视频分离的后果**

假设你只下载了 B 站的视频流（.m4s），没有下载音频流，也没有进行合成，直接将该文件重命名为 .mp4 并尝试播放，会出现什么现象？请解释原因，并说明 moviepy 合成步骤在其中解决了什么问题。
