## 七、配置文件与参数

### 7.1 白名单配置

| 属性 | 内容 |
|------|------|
| **名称** | 白名单（Whitelist） |
| **类型** | 平台配置项 |
| **在本P中的作用** | 代理平台的安全机制，将本机出口IP加入白名单后才能调用API获取代理；未配置时会报错"当前IP不在白名单内" |
| **来源** | 字幕直接提到 |
| **后续可练习方向** | 建议方向：了解动态IP（拨号VPS）场景下白名单的管理策略；学习通过API动态添加/删除白名单，实现自动化运维 |

### 7.2 proxies参数

| 属性 | 内容 |
|------|------|
| **名称** | proxies参数（requests库） |
| **类型** | API参数 |
| **在本P中的作用** | requests.get()的proxies参数接收字典，指定HTTP和HTTPS代理，将请求路由到代理服务器再转发给目标 |
| **来源** | 字幕直接提到 |
| **后续可练习方向** | 建议方向：理解HTTP代理与HTTPS代理的区别（隧道 vs 直连）；学习在Scrapy/Playwright等框架中配置代理；尝试使用socks5代理 |

### 7.3 response.cookies

| 属性 | 内容 |
|------|------|
| **名称** | response.cookies |
| **类型** | API属性（requests库） |
| **在本P中的作用** | 登录请求响应后，从response.cookies提取Cookie对象并存入Cookie池，供后续请求复用 |
| **来源** | 字幕直接提到 |
| **后续可练习方向** | 建议方向：对比RequestsCookieJar与字典格式的Cookie；学习Cookie的过期时间管理；实现Cookie池的自动刷新与淘汰机制 |
