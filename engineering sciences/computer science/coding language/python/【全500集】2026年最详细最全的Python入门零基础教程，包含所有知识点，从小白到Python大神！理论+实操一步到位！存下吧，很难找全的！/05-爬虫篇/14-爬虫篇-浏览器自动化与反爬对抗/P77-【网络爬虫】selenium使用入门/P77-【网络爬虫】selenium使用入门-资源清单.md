# P77 资源清单 - 【网络爬虫】selenium使用入门
---

## 一、开发库与依赖

### 1. selenium
- **类型**：Python 第三方库
- **在本 P 中的作用**：核心爬虫库，提供 `webdriver` 接口模拟浏览器操作，可获取 JS 渲染后的完整页面源码，绕过穿云盾等反爬机制
- **来源**：字幕直接提到（pip install selenium）
- **建议方向**：阅读官方文档了解完整 API；尝试控制 Firefox、Edge 等其他浏览器

### 2. requests
- **类型**：Python 第三方库
- **在本 P 中的作用**：对比演示对象，说明当 requests 被反爬拦截后，Selenium 作为兜底方案的必要性
- **来源**：字幕直接提到（Python 已有生态）
- **建议方向**：回顾 requests 的 POST 请求构造方法（URL、请求头、Cookie），作为与 Selenium 方案的对比练习

---

## 二、驱动与浏览器

### 3. chromedriver
- **类型**：浏览器驱动（可执行文件）
- **在本 P 中的作用**：Selenium 与 Chrome 之间的通信桥梁，版本必须与 Chrome 浏览器完全一致；通过 `webdriver.Chrome()` 调用
- **来源**：字幕直接提到
- **建议方向**：记录当前 Chrome 版本并下载对应驱动；尝试将驱动放到不同路径测试加载方式

### 4. Chrome 浏览器
- **类型**：软件
- **在本 P 中的作用**：Selenium 操控的目标浏览器；通过开发者工具（F12）辅助分析页面结构、XPath 表达式调试
- **来源**：字幕直接提到
- **建议方向**：检查当前版本号；熟悉 F12 开发者工具的 Elements、Network 面板

### 5. Chrome 官方驱动下载页面
- **类型**：网站
- **在本 P 中的作用**：chromedriver 官方下载地址，下载与 Chrome 版本完全匹配的驱动
- **来源**：字幕直接提到
- **建议方向**：收藏该页面；验证"版本匹配"机制（故意下载错误版本观察报错）

### 6. 淘宝 chromedriver 镜像
- **类型**：网站（镜像源）
- **在本 P 中的作用**：Chrome 官方地址访问慢时的备选下载地址，提供 chromedriver 镜像
- **来源**：字幕直接提到
- **建议方向**：测试镜像下载速度；对比官方源与镜像源的版本齐全程度

---

## 三、实战案例网站

### 7. 领导留言板网站
- **类型**：网站（政府公开平台）
- **在本 P 中的作用**：反爬实战演示案例，数据通过 Ajax 异步加载，演示 requests 被拦截后 Selenium 成功获取数据的过程
- **来源**：字幕直接提到
- **建议方向**：访问该网站观察留言数据加载方式；尝试用 Selenium 等待并提取具体留言内容

### 8. 穿云盾
- **类型**：反爬虫防护平台/服务
- **在本 P 中的作用**：本 P 案例中拦截 requests 请求的防护系统，说明为何需要 Selenium 兜底；老师强调该防护破解成本高
- **来源**：字幕直接提到
- **建议方向**：了解穿云盾的防护机制；搜索相关反穿云盾的技术讨论

---

## 四、辅助工具与技能

### 9. 开发者工具（F12）
- **类型**：浏览器内置工具
- **在本 P 中的作用**：分析网页结构、定位元素、调试 XPath 表达式、查看网络请求（Ajax 接口）
- **来源**：字幕直接提到（按 F12 打开）
- **建议方向**：熟练使用 Elements 面板 + Ctrl+F 搜索框验证 XPath；用 Network 面板观察 Ajax 请求

### 10. XPath
- **类型**：定位语言/表达式
- **在本 P 中的作用**：万能元素定位方式，本 P 强调"只需掌握 XPath 一种"；示例：`//input[@id="kw"]` 可定位百度搜索框
- **来源**：字幕直接提到
- **建议方向**：练习常见 XPath 表达式写法；用 F12 开发者工具验证表达式的唯一匹配性

---

## 五、关联工具（基于上下文推断）

### 11. BeautifulSoup
- **类型**：Python 解析库
- **在本 P 中的作用**：Selenium 获取页面源码后，可配合 BeautifulSoup 进行结构化数据提取；老师提到 Selenium 不像 BeautifulSoup 那样可以用 CSS 选择器直接提取
- **来源**：基于上下文推断（爬虫工具链关联）
- **建议方向**：了解 BeautifulSoup 的 CSS 选择器语法，作为 Selenium 数据提取的补充方案

### 12. pip
- **类型**：命令（包管理器）
- **在本 P 中的作用**：Python 包管理命令，用于安装 selenium 等第三方库：`pip install selenium`
- **来源**：字幕直接提到
- **建议方向**：确认 pip 可用；尝试 pip list 查看已安装的包
