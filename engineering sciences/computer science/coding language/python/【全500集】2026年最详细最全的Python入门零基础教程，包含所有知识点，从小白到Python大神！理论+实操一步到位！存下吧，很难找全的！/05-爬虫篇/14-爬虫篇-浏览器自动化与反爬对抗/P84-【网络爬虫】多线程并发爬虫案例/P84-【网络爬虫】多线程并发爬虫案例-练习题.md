## 参考答案要点

> 以下仅为答题方向的简要提示，非标准答案。

**1.1**：B、D（解析阶段负责数据提取和结构转换，不涉及网络请求和文件写入）

**1.2**：B（三阶段模型的核心就是队列连接各线程）

**2.1**：线程不能直接调用其他线程的方法（跨线程调用不直接），且解耦后各阶段可独立调试和替换

**2.2**：B（Queue 的 get() 在队列为空时会阻塞，这是生产者-消费者模式的基础）

**2.3**：请求阶段的网络 I/O 等待最长；并发让多个线程在不同请求的等待间隙中工作

**3.1**：`t = threading.Thread(...)` / `t.start()`

**3.2**：`os.makedirs(song_dir, exist_ok=True)` / `os.path.join(song_dir, song_name + ".txt")` / `"w", "utf-8"`（或直接写入字符串）

**3.3**：应使用 `"wb"` 模式写入二进制数据，`"w"` 会导致编码错误

**4.1~4.2**：开放性题目，核心在于理解并发效率瓶颈（网络带宽、CPU、I/O）与反爬机制

**5.1~5.2**：实操题目，无固定答案，完成过程即完成练习
