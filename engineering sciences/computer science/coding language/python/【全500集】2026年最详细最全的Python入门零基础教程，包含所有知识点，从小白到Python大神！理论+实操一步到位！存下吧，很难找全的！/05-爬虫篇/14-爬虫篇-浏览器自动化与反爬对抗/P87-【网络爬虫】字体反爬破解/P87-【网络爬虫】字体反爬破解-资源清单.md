## 软件 / 工具

### FontCreator

- **类型**：桌面软件（Windows）
- **在本P中的作用**：演示开发者视角的字体反爬实现。用于创建、编辑自定义字体文件，包括设置字形、配置UNICODE字符码映射、导出字体文件（TTF/WOFF/WOFF2格式）
- **来源**：字幕直接提到
- **建议方向**：下载安装后尝试制作一个简单的自定义字体（如用符号替换数字），理解字形与字符码的映射关系

---

### Fiddler

- **类型**：抓包工具
- **在本P中的作用**：拦截HTTP请求，获取目标网站返回的字体文件（woff/ttf）。是破解字体反爬时获取字体文件的常用工具
- **来源**：字幕直接提到（与BurpSuite、Charles并列提及）
- **建议方向**：配合练习目标网站，尝试抓取并保存页面引用的字体文件

---

### BurpSuite

- **类型**：抓包工具 / Web安全测试平台
- **在本P中的作用**：与Fiddler功能类似，用于拦截HTTP请求、捕获字体文件。是专业爬虫工程师常用的工具
- **来源**：字幕直接提到（与Fiddler、Charles并列提及）
- **建议方向**：学习其代理配置和请求拦截功能，用于获取字体文件

---

### Charles

- **类型**：抓包工具
- **在本P中的作用**：跨平台抓包工具，用于macOS/Windows/Linux环境下拦截HTTP/HTTPS请求，获取字体文件
- **来源**：字幕直接提到（与Fiddler、BurpSuite并列提及）
- **建议方向**：若使用macOS系统，可优先选择Charles进行抓包练习

---

## Python 库

### fonttools

- **类型**：Python库（核心依赖）
- **在本P中的作用**：解析woff/ttf等字体文件，提取cmap映射表。可通过saveXML()方法将二进制字体转换为可读的XML格式，便于分析映射关系。破解字体反爬的必用库
- **来源**：字幕直接提到（pip install fonttools）
- **建议方向**：结合实际字体文件练习解析、提取cmap表、建立映射字典的全流程

---

## 文件格式

### WOFF (Web Open Font Format)

- **类型**：字体文件格式
- **在本P中的作用**：网页中常用的字体格式，体积相对TTF更小（已压缩）。是字体反爬的主要载体之一，存储自定义的字符码到字形映射关系
- **来源**：字幕直接提到
- **建议方向**：注意区分WOFF与WOFF2的兼容性，根据目标网站实际情况选择解析

---

### TTF (TrueType Font)

- **类型**：字体文件格式
- **在本P中的作用**：兼容性最好的字体格式，各浏览器和平台均支持。同样可作为字体反爬的载体
- **来源**：字幕直接提到
- **建议方向**：fonttools对TTF和WOFF的解析方式类似，可统一练习

---

### XML (字体中间格式)

- **类型**：配置文件 / 中间格式
- **在本P中的作用**：fonttools.saveXML()方法将二进制字体文件转换为可读的XML格式。XML中的cmap标签记录了字符码（code属性）与字形名称（name属性）的对应关系，是建立映射字典的关键数据源
- **来源**：字幕直接提到（fonttools.saveXML('output.xml')）
- **建议方向**：练习解析XML结构，重点关注cmap节点，理解code到name的映射逻辑

### ord() / hex()

- **类型**：Python内置函数
- **在本P中的作用**：ord()用于获取字符的Unicode码；hex()用于将整数转换为16进制字符串。这两个函数是建立HTML字符与Unicode码映射的关键工具
- **来源**：字幕直接提到（课程示例代码中演示）
- **建议方向**：结合具体字符练习Unicode码的获取与转换，理解字符编码的基本概念

## 补充说明

1. **FontCreator** 主要服务于“学习攻击”阶段，用于理解字体反爬的原理。若仅需破解，无需安装此软件。

2. **抓包工具三选一**即可，建议根据自身操作系统选择：Windows用户优先Fiddler，macOS用户优先Charles，Linux用户可两者皆用或选Fiddler + Mono。

3. **fonttools** 是破解的必用库，建议优先深入学习其API文档，掌握除saveXML外的直接读取cmap方法。

4. 本P未涉及动态字体（每次请求返回不同映射）的处理，属于进阶方向，可在掌握基础破解后进一步研究。
