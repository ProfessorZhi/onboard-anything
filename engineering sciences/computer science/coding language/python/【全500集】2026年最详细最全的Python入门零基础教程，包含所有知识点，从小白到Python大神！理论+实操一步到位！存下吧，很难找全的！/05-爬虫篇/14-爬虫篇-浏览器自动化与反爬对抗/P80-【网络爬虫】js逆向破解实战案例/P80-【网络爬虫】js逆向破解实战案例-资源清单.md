## 一、开发工具

### 1. 开发者工具（DevTools）
- **类型**：浏览器内置开发工具
- **在本P中的作用**：JS逆向分析的核心工作台，用于抓包分析（Network面板）、断点调试（Sources面板）、调用栈追踪、变量值观察。是定位加密函数、提取JS代码的主要手段。
- **来源**：字幕直接提到
- **建议方向**：深入学习Sources面板的调试功能，包括条件断点、监视表达式、调用栈导航；掌握XHR/Fetch断点的使用方法；练习快速定位混淆JS中的目标函数

## 二、运行环境

### 2. Node.js
- **类型**：JavaScript运行时环境
- **在本P中的作用**：pyexecjs库的底层依赖，为Python提供JS代码执行能力。安装后需配置环境变量，使Python能够调用Node.js的execjs引擎执行JS代码。
- **来源**：字幕直接提到
- **建议方向**：完成官方安装包安装，配置系统环境变量PATH；学习使用node -v验证安装；了解npm全局模块路径的概念及配置方法

---

## 三、Python库

### 3. pyexecjs
- **类型**：Python第三方库
- **在本P中的作用**：核心桥接工具，将JS代码编译为上下文对象后，通过ctx.call()方法调用JS函数并传递参数，返回JS函数的执行结果。用于在Python中执行提取的JS加密函数。
- **来源**：字幕直接提到（pip install pyexecjs）
- **建议方向**：掌握execjs.compile()的完整流程；学习处理函数依赖较多时的代码组织方式；了解不同JS引擎（Node.js、QuickJS等）的切换配置

### 4. hashlib
- **类型**：Python标准库
- **在本P中的作用**：提供MD5、SHA1、SHA256等多种哈希算法实现。在JS未混淆时，用于Python原生还原JS中的加密逻辑（如MD5加盐哈希）。
- **来源**：字幕直接提到
- **建议方向**：熟练使用hashlib.md5()进行加盐哈希；对比hashlib与JS CryptoAPI的输出差异；扩展学习AES等对称加密算法的Python实现（需安装pycryptodome）

### 5. requests
- **类型**：Python第三方库
- **在本P中的作用**：发送HTTP/HTTPS请求，构造伪造的登录请求或翻译接口请求。配合逆向得到的加密参数，实现对目标接口的模拟访问。
- **来源**：字幕直接提到
- **建议方向**：掌握POST请求的构造方法；学习请求头（Headers）的完整复制；了解Cookie和Session的处理方式；结合加密还原实现完整的登录或接口调用流程

## 四、命令工具

### 6. npm root -g
- **类型**：命令
- **在本P中的作用**：查看Node.js全局模块的安装路径。该路径需要加入系统环境变量，使Python的pyexecjs能够找到Node.js的模块。
- **来源**：字幕直接提到
- **建议方向**：了解npm作为包管理工具的其他常用命令；学会通过该路径判断模块是否正确安装

### 9. 登录接口密码加密案例
- **类型**：实战案例

- **来源**：字幕直接提到
