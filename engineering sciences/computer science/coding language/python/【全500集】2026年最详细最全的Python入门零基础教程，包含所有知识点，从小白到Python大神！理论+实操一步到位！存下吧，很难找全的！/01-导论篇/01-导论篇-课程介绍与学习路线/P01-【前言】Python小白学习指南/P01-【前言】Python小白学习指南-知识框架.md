# P01 知识框架 - 【前言】Python小白学习指南
## 本P位置
- 课程导论（P1/500）
- 功能：建立学习地图，非具体技术知识
- 定位：为"基础→进阶→实战"三阶段提供路线导航

## 核心问题
- Python学习如何从零基础系统化到达实战能力？
- 三阶段板块之间的递进关系是什么？
- 每个阶段需要培养哪些核心能力？

## 知识骨架

**三板块递进结构**：

基础篇（根基）
- 软件安装与环境配置
- 变量、数据类型、运算符
- 控制流、函数
- 目标：代码规范 + 逻辑思维，能写简单代码

进阶篇（体系）
- 面向对象编程（OOP）核心理念
- 继承与多态
- 正则表达式
- 目标：构建完整编程知识体系，提升复杂问题解决能力

实战篇（变现）
- 爬虫（图片、视频抓取）
- 数据存储、数据分析与挖掘
- 自动化办公
- 目标：实操能力 + 项目经验积累，适配大数据时代需求

## 关系与依赖

基础篇 → 进阶篇 → 实战篇（单向线性依赖）

基础篇输出的"简单代码编写能力"是进阶篇的输入
进阶篇构建的"完善编程体系"是实战篇的输入
后一阶段以前一阶段能力为前提，不可跳过

## 输入 / 输出 / 前置 / 支撑

前置：无（零基础友好）
输出：每个阶段的明确能力目标
输入：各阶段依赖前一阶段的输出
支撑：资料包（开发环境指南、课件、练习素材、电子书）、思维导图、学习计划表

## 易错点

- 跳过基础篇直接学框架/库
- 忽视代码规范和编程习惯的早期养成
- 将实战案例当作孤立技能，忽略其对基础和进阶知识的依赖

## 外部资源与对象

- **资料包**：课程配套资源集合（环境配置指南、课件、练习素材、电子书），通过评论区留言"111"获取
- **思维导图**：可视化知识梳理工具，用于辅助理解三板块结构
- **学习计划表**：时间管理工具，支撑学习进度规划
- **Python 2026课程**：本教程实体，承载三板块内容的学习平台
