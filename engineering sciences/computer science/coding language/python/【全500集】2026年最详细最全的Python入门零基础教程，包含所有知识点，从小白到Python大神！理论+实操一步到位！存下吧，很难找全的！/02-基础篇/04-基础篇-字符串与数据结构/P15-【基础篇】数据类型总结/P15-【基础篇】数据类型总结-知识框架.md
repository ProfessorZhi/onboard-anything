# P15 知识框架 - 【基础篇】数据类型总结
## 本P位置

Python基础篇 · 数据类型专题总结课。前置：各基本数据类型的分节讲解。本P承上启下，将十种数据类型横向整合，为后续函数、文件操作等章节奠定类型操作基础。

## 核心问题

1. Python数据类型分为哪两大类，各包含哪些具体类型？

3. 哪些操作可跨类型通用，它们的用法和限制是什么？
4. 类型之间如何互相转换，转换规则和风险是什么？

## 知识骨架

```
Python数据类型
├── 基本数据类型（处理单个数据）
│   ├── int     —— 整数，无小数点
│   ├── float   —— 浮点数，带小数点
│   ├── bool    —— 布尔值，True/False
│   ├── complex —— 复数，实部+虚部J
│   ├── str     —— 字符串，不可修改
│   └── None    —— 空值，无数据标记
└── 复合数据结构（处理多个数据）
    ├── list    —— 有序可修改，允许重复
    ├── tuple   —— 有序不可修改
    ├── dict    —— 键值对映射，键唯一
    └── set     —— 无序自动去重

通用操作（跨越数据类型）
├── in / not in     —— 成员判断
├── len()           —— 长度统计
├── del             —— 删除
└── sum / max / min —— 数值聚合

类型转换
├── int() / float() / bool() / str()
├── eval()          —— 执行字符串表达式（安全风险）
└── list() / tuple() / set() / dict()
```

## 关系与依赖

- **基本 → 复合**：六种基本类型处理单个值，四种容器类型处理批量数据；从单到多，是认知递进关系
- **通用操作层**：in/not in、len()、del、sum/max/min 独立于类型系统，可作用于 str、list、tuple、dict、set；不作用于数值类型和 None
- **转换层**：建立在类型系统之上，提供类型间互转能力；dict() 转换需要嵌套元组提供键值对，是唯一有结构要求的转换
- **核心对立轴**：可修改 vs 不可修改、有序 vs 无序、唯一 vs 可重复——三条轴线区分四种容器类型

## 易错点

- **str 不可修改**：字符串切片返回新字符串，原字符串不变
- **dict in/not in 判断键名**：用 .values() 判断值是否存在
- **集合不能用 del 删除元素**：集合无索引，只能 del 整个变量
- **len() 字符统计**：中文字符按 UTF-8 占 2-4 字节，但 len() 统计字符个数而非字节
- **bool 转换口诀**："非零非空即为 True"；空格字符非空 → True；空容器 → False
- **eval() 安全风险**：禁止执行未知来源字符串，可注入恶意代码
- **int() 截断而非四舍五入**：int(1.9) → 1
- **sum/max/min 仅对数值有效**：max/min 对字符串可按 ASCII 排序，sum 不行
- **dict 转换结构要求**：必须提供键值对结构，dict([1,2,3]) 报错
