## 二、Python 内置函数

### type()

| 字段 | 内容 |
|------|------|
| 名称 | `type()` |
| 类型 | Python 内置函数 |
| 在本P中的作用 | 检测变量或表达式的数据类型；在演示类型转换时，用于验证转换前后的类型变化 |
| 来源 | 字幕直接提到（老师建议用 `type()` 确认转换结果） |
| 建议方向 | 对本P所有十种类型逐一调用 `type()`，将结果整理为对照表；在类型转换练习中形成"转换前 → type() 验证 → 转换后 → type() 验证"的习惯 |

---

### int()

| 字段 | 内容 |
|------|------|
| 名称 | `int()` |
| 类型 | Python 内置类型转换函数 |
| 在本P中的作用 | 将浮点数或字符串转换为整数；演示截断行为（`int(1.9)` → `1`，不四舍五入） |
| 来源 | 字幕直接提到 |
| 建议方向 | 分别测试 `int(1.9)`、`int(-1.9)`、`int('123')`、`int('12.3')`（会报错），观察截断规则与报错边界；与 `round()` 对比理解截断和四舍五入的区别 |

---

### float()

| 字段 | 内容 |
|------|------|
| 名称 | `float()` |
| 类型 | Python 内置类型转换函数 |

| 来源 | 字幕直接提到 |
| 建议方向 | 测试 `float('abc')` 等非法输入的报错信息；结合 `int()` 理解两者互转的规则与损失 |

---

### bool()

| 字段 | 内容 |
|------|------|
| 名称 | `bool()` |

| 来源 | 字幕直接提到 |

### str()

| 字段 | 内容 |
|------|------|
| 名称 | `str()` |
| 类型 | Python 内置类型转换函数 |
| 在本P中的作用 | 将任意数据类型（数字、列表、字典等）转换为字符串；老师称其为"万能的转文字工具" |
| 来源 | 字幕直接提到 |

---

### eval()

| 字段 | 内容 |
|------|------|
| 名称 | `eval()` |
| 类型 | Python 内置函数 |
| 在本P中的作用 | 执行字符串中的表达式，将字符串还原为对应数据类型（本质是"去除引号"）；支持还原整数、浮点数、列表、元组、集合；老师同时明确指出其安全风险 |
| 来源 | 字幕直接提到 |
| 建议方向 | 在受控环境中练习合法字符串的 `eval()` 还原；了解 `ast.literal_eval()` 作为更安全的替代方案（只解析字面量，不执行任意代码）；不要将 `eval()` 用于 `input()` 的直接处理 |

---

| 来源 | 字幕直接提到 |
| 建议方向 | 测试对字符串、元组、集合、range 对象分别调用 `list()` 的结果；观察顺序和元素数量的变化 |

---

### tuple()

---

### set()

| 字段 | 内容 |
|------|------|
| 名称 | `set()` |
| 类型 | Python 内置类型转换函数 / 空集合构造方式 |
| 在本P中的作用 | 将可迭代对象转换为集合（自动去重）；`set()` 是创建空集合的唯一合法方式（直接用 `{}` 会创建空字典）；示例 `set('冰冰')` → `{'冰'}` |
| 来源 | 字幕直接提到 |
| 建议方向 | 对含重复元素的列表调用 `set()` 做去重练习；测试 `{}` vs `set()` 的类型差异（用 `type()` 验证） |

---

### dict()

| 字段 | 内容 |
|------|------|
| 名称 | `dict()` |
| 类型 | Python 内置类型转换函数 |
| 在本P中的作用 | 将嵌套元组结构的列表转换为字典；演示了结构要求：每个子元素必须是包含两个值的元组（键值对），否则报错 |
| 来源 | 字幕直接提到 |
| 建议方向 | 练习 `dict([('A', 1), ('B', 2)])` 的合法格式；测试非法格式（`dict([1, 2, 3])`）的报错信息；后续结合 `zip()` 函数理解更灵活的字典构造方式 |

---

## 三、Python 内置运算符 / 关键字

### in / not in

| 字段 | 内容 |
|------|------|
| 名称 | `in` / `not in` |
| 类型 | Python 内置成员运算符 |
| 在本P中的作用 | 判断元素是否存在于字符串、列表、元组、字典（默认判断键名）、集合中；返回布尔值；不适用于数值类型和 None |
| 来源 | 字幕直接提到 |
| 建议方向 | 分别在五种支持类型中练习；重点练习字典的键判断与 `.values()` 值判断的区别；测试对整数调用的报错行为 |

---

### del

| 字段 | 内容 |
|------|------|
| 名称 | `del` |
| 类型 | Python 内置关键字 |
| 在本P中的作用 | 删除变量（从内存移除）、删除列表中指定索引的元素、删除字典中指定键的键值对；集合因无索引而不支持 `del` 删除元素 |
| 来源 | 字幕直接提到 |
| 建议方向 | 分三个场景独立练习（变量、列表元素、字典键值对）；观察删除变量后再引用的 `NameError` 报错；对比集合使用 `.remove()` / `.discard()` 方法删除元素的方式 |
