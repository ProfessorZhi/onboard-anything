**题目 2**：以下代码的输出结果分别是什么类型？

```python
s = "Hello World"
a = s.find("o")
b = s.replace("World", "Python")
c = s.startswith("He")
d = s.encode()
print(type(a), type(b), type(c), type(d))
```

## 二、机制理解

**题目 3**：判断正误（正确打 ✓，错误打 ✗ 并说明原因）

- （ ）调用 `replace()` 后，原字符串的内容会发生改变。
- （ ）`find()` 和 `index()` 的功能完全相同。
- （ ）`"你好".islower()` 的返回值是 `True`。
- （ ）`split()` 不指定分隔符时，会按所有空白字符（空格、换行、制表符）分割。
- （ ）`"hello".encode()` 得到的字节串长度是 5。

**题目 4**：解释为什么以下代码的输出是 `"Hello World"` 而不是 `"hello World"`。

```python
s = "hello World"
s.upper()
print(s)
```

## 四、扩展思考

**题目 8**：

> 需求：判断用户输入的密码是否同时满足以下条件——以字母开头、长度至少 8 位、包含数字。

请分析以下代码能否正确实现该需求，并指出问题所在。

```python
password = input("请输入密码：")
if password.startswith(str.isalpha) and len(password) >= 8 and any(c.isdigit() for c in password):
    print("密码符合要求")
else:
    print("密码不符合要求")
```

## 五、参考答案（要点）

> **题目 1**：查找类 → find；修改类 → replace、split、encode；判断类 → isupper、startswith
>
> **题目 3**：(1) ✗ 字符串不可变 (2) ✗ 功能相同但找不到时行为不同 (3) ✓ (4) ✓ (5) ✗ 中文占3字节，英文占1字节
>
> **题目 4**：修改类方法返回新字符串，不修改原对象
>
> **题目 6**：`text.split()` 后过滤含 `@` 的部分
>
> **题目 8**：`startswith()` 参数应为字符串而非函数，应改为 `password[0].isalpha()`
