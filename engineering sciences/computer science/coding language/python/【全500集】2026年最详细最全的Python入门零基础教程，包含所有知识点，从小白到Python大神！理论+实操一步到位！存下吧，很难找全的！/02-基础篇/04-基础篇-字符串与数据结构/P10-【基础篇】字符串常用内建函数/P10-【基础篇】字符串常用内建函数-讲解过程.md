# P10 讲解过程 - 【基础篇】字符串常用内建函数
## 本P定位

本P是"字符串常用内建函数"章节的核心内容。老师在引入查找类函数时，先概述了其功能，随后逐一演示find()、index()、count()三个查找方法的用法与差异，并特别强调find()未找到时返回-1、index()未找到时会报错这一核心区别。接着进入修改类函数的讲解，涉及replace()替换、split()分割、strip()去除首尾字符、lower()与upper()大小写转换，每种方法都配合代码演示和运行结果展示。最后补充判断类函数（startswith、endswith、isupper、islower）的用法，以及encode()和decode()编码解码的原理与实际应用。课程末尾以五点总结串联了字符串章节的核心知识点。

## 讲解过程

### 一、引入：字符串内建函数的作用

老师开篇指出字符串常用内建函数能够高效处理各种文字问题，表示这是本次课这个小节的最后一个部分。接下来将内容分为三大类：查找类函数、修改类函数、判断类函数。

### 二、查找类函数（find / index / count）

老师先介绍三个查找方法的共同特点，再逐一演示差异。

**find() 方法**：查找子串首次出现的下标位置，未找到返回-1。老师定义变量 `name = "冰冰"`，演示 `name.find("G")` 返回3，即"G"在字符串中的下标；`name.find("NG")` 返回2，即"N"的下标。随后演示查找不存在的子串 `name.find("SSS")`，运行结果显示返回-1，表示不存在。

**index() 方法**：功能与find()完全相同，区别在于未找到时会直接报错。老师演示 `name.index("G")` 返回3，`name.index("NG")` 返回2，与find()结果一致。但当查找 `name.index("SSS")` 时，运行后会报错。这是find与index唯一的区别。

**count() 方法**：统计子串出现的次数，未找到返回0。老师演示 `name.count("B")` 返回2（因为"冰冰"中有两个"B"字符）；`name.count("BIN")` 返回2（"冰冰"中"BIN"出现两次）；`name.count("NG")` 或 `name.count("GB")` 返回1；`name.count("SSS")` 返回0。

### 三、修改类函数（replace / split / strip / lower / upper）

老师先说明修改类函数的特点：它们会生成新的字符串，因此必须用print输出结果，不能直接调用方法后置之不理。

**replace() 方法**：替换字符串中的内容。老师演示 `name.replace("B", "D")`，将"冰冰"中的所有"B"替换为"D"，输出"丁丁"。老师还演示了可选的第三个参数——替换次数：如果指定 `name.replace("BING", "DDD", 1)`，则只替换第一个匹配项，其余保持不变。没有指定次数时默认全部替换。

**split() 方法**：按分隔符分割字符串，返回列表。老师先定义 `st = "hello\nword gun 冰冰"`，演示 `st.split()` 不指定分隔符时，默认使用空白字符（空格、`\n`、`\t` 等）作为分隔符进行分割。老师用图示模拟分割过程：遇到空格切开、遇到`\n`切开，最终得到包含4个元素的列表 `['hello', 'word', 'gun', '冰冰']`。

老师随后演示指定分隔符的场景：`st.split(" ")`（指定空格为分隔符），由于字符串中不含`\n`且多个空格连续，处理逻辑略有不同，最终得到3个元素的列表。老师还演示了可指定的第二个参数——分割次数：`st.split(" ", 1)` 表示只分割一次，只切出前两个部分，其余合并为一个字符串。

**strip() 方法**：去除首尾指定的字符，默认去除空格。老师定义 `st = "  xxx冰冰xxx  "` 或类似带多余字符的字符串，演示 `st.strip()` 不指定参数时默认去除首尾空格。随后演示指定去除字符：`st.strip("X")` 去除首尾的"X"字符（会将所有连续的"X"都去掉）。老师强调strip()只会去除首尾的字符，中间的字符不会受影响。

**lower() 和 upper() 方法**：大小写转换。老师定义 `st = "hello Python"`，演示 `st.upper()` 将所有字母转为大写输出"HELLO PYTHON"；`st.lower()` 将所有字母转为小写输出"hello python"。

### 四、判断类函数（startswith / endswith / isupper / islower）

**startswith() 和 endswith() 方法**：判断字符串是否以指定内容开头或结尾。老师定义 `name = "冰冰"`，演示 `name.startswith("BIN")` 返回True（因为确实以"BIN"开头）；`name.startswith("I")` 返回False（首字符不是"I"）。演示 `name.endswith("G")` 返回True（末字符确实是"G"）；`name.endswith("BIN")` 返回False（因为最后三个字符是"ING"而不是"BIN"）。

**isupper() 和 islower() 方法**：判断字符串是否全部由大写字母或全部由小写字母组成。老师强调这两个方法要求字符串中所有字母字符必须同为大写或同为小写，否则返回False。例如对于 `name = "冰冰"`（包含中文，中文不算字母），`name.islower()` 返回True（因为没有大写字母），但 `name.isupper()` 返回False。

### 五、编码与解码（encode / decode）

老师先铺垫背景：字符串是人类能理解的字符信息，计算机存储数据以字节为单位（binary data），需要通过编码将字符串转换为字节串（bytes），或者通过解码将字节串还原为字符串。老师用现实例子类比：桌面新建文本文档写入"冰冰"两个字，实际文件大小为6个字节——说明计算机存储的确实是字节信息。

**encode() 方法**：编码，将字符串转换为字节串。老师定义 `st = "今天真开心"`，调用 `st.encode()`，方法默认使用UTF-8编码，返回带 `b` 前缀的字节串（人类看不懂的形式）。返回值应赋值给变量保存。

**decode() 方法**：解码，将字节串转换回字符串。老师演示将上述字节串变量调用 `.decode()` 方法，解码后输出"今天真开心"。老师指出这种字符串与字节串的相互转换在网络传输、文件写入、爬虫数据处理等场景中非常常见。

### 六、课程总结

老师最后归纳了五个核心要点：

1. **字符串定义与运算**：单引号、双引号、三引号界定；加号拼接、星号重复；in/not in 判断包含关系。

2. **索引与切片**：从左往右从0开始，从右往左从-1开始；切片遵循包前不包后原则；步长为负数反向切，为正数正向切。

3. **转义字符**：\n 表示换行，\t 表示制表符（对齐文本）；文件路径优先用原始字符串r""取消转义。

4. **格式化**：推荐f-string，简洁、支持运算和排版、可读性好。

5. **常用内建函数**：查找用find/index/count，修改用replace/split/strip，判断用startswith/endswith/isupper/islower。**重要提醒**：字符串不可修改，操作后得到的是新字符串。

老师建议课后多结合实际场景练习，如格式化输出个人信息或处理文案。

## 提到的外部资源与对象

- **UTF-8**：encode()方法的默认编码格式，属于Python内建行为，无需额外安装。
- **f-string**：Python 3.6+引入的字符串格式化方式，属于Python内建语法特性。
- **字节串（bytes类型）**：Python内置的数据类型，表示二进制数据，带 `b` 前缀，如 `b'\xe4\xbb\x8a\xe5\xa4\xa9'`。
- **列表（list）**：split()方法的返回值类型，本P中老师预告列表将在下一节详细讲解。

## 本P结论

- **查找类函数**：find() 未找到返回-1，index() 未找到会报错，count() 未找到返回0。
- **修改类函数**：replace() 支持指定替换次数，split() 支持指定分隔符和分割次数（默认空白字符分割），strip() 默认去除首尾空格，lower()/upper() 进行大小写转换。修改类函数均返回新字符串，原字符串保持不变。
- **判断类函数**：startswith() 和 endswith() 用于判断开头结尾，isupper() 和 islower() 要求字符串中所有字母字符必须同为大写或同为小写。
- **编码解码**：encode() 将字符串转为字节串（默认UTF-8），decode() 将字节串还原为字符串，二者常用于网络传输和文件操作场景。
- **核心性质**：字符串是不可变（immutable）对象，所有修改类操作都生成新字符串。
