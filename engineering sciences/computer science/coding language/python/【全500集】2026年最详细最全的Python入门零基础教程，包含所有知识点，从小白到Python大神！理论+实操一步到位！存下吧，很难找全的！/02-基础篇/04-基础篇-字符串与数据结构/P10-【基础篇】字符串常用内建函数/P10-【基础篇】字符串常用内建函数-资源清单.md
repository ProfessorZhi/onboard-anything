# P10 资源清单 - 【基础篇】字符串常用内建函数
## 资源说明

本 P 聚焦 Python 字符串内建函数，属于纯语法讲解，未涉及外部工具、第三方库或在线平台。资源类型以 Python 内置概念、数据类型为主，辅以与本 P 直接关联的官方文档和开发工具建议。

---

## 一、核心数据类型（内置）

### 1. str（字符串类型）

- **类型**：内置数据类型
- **在本 P 中的作用**：所有内建函数的调用主体。本 P 涉及的 find()、replace()、split()、encode() 等方法均依附于 str 对象。
- **来源**：字幕直接提到
- **建议方向**：对比 str 与其他序列类型（如 list、tuple）的通用方法（索引、切片、len()），加深对序列共性的理解。

### 2. bytes（字节串类型）

- **类型**：内置数据类型
- **在本 P 中的作用**：encode() 方法的输出类型，decode() 方法的输入类型。表示二进制数据，带 `b` 前缀，如 `b'\xe4\xbb\x8a\xe5\xa4\xa9'`。
- **来源**：字幕直接提到
- **建议方向**：在文件中实际写入中文内容，观察文件字节大小；用 Python 读取二进制文件（如图片），理解 bytes 在实际场景中的应用。

### 3. list（列表类型）

- **类型**：内置数据类型
- **在本 P 中的作用**：split() 方法的返回类型。本 P 仅提及 split() 返回列表，未展开列表操作。
- **来源**：字幕直接提到（老师预告下一节详述列表）
- **建议方向**：学习列表的索引访问、追加、删除、切片等操作，为后续处理 split() 的返回值做准备。

---

## 二、编码与字符集

### 4. UTF-8

- **类型**：字符编码格式
- **在本 P 中的作用**：encode() 方法的默认编码格式。中文字符在 UTF-8 下占 3 字节。
- **来源**：字幕直接提到
- **建议方向**：对比 GBK、UTF-16 等编码格式，了解不同编码对中文字符的字节占用差异；尝试用不同编码 encode() 同一字符串，观察输出差异。

## 三、字符串方法清单

以下方法均为 Python str 类型的内置方法，无需导入可直接调用。

### 查找类方法

| 方法 | 类型 | 在本 P 中的作用 | 来源 | 建议方向 |
|------|------|----------------|------|----------|
| find() | 内建方法 | 查找子串首次出现的下标，未找到返回 -1 | 字幕直接提到 | 在长文本中查找关键词；结合循环实现查找所有出现位置 |
| index() | 内建方法 | 功能同 find()，区别在于未找到时抛出 ValueError | 字幕直接提到 | 对比 find() 的容错处理，理解异常机制的适用场景 |

| 方法 | 类型 | 在本 P 中的作用 | 来源 | 建议方向 |
