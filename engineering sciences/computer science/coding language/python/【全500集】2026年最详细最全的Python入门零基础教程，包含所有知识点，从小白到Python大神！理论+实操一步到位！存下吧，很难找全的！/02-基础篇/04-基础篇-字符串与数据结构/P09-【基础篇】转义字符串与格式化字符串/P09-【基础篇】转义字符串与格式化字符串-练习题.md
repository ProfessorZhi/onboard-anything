# P09 练习题 - 【基础篇】转义字符串与格式化字符串
---

## 结构识别

**题目 1.1** 下列哪些是 Python 合法的转义字符序列？（多选）

- A. `\t`
- B. `\x`
- C. `\\`
- D. `\u0061`
- E. `\#`

**答案**：`A` `C` `D`

---

**题目 1.2** 下面三段代码的输出结果分别是？

```python
print("A\tB")
print("C\\nD")
print(r"E\nF")
```

**答案**：
- 第一行：`A    B`（`\t` 转换为制表符，AB之间有空隙）
- 第二行：`C\nD`（`\\` 输出单个反斜杠，保留字面 `\n`）
- 第三行：`E\nF`（`r` 前缀使反斜杠失去转义意义，原样输出）

## 机制理解

**题目 2.1** 请解释 `print(f"{25:05.2f}")` 中每个部分的含义：

| 占位符部分 | 含义 |
|---|---|
| `25` | |
| `05` | |
| `.2` | |
| `f` | |

**答案**：

| 占位符部分 | 含义 |
|---|---|
| `25` | 要格式化的数值 |
| `05` | 总宽度为5位，不足时用 `0` 补齐 |
| `.2` | 保留2位小数 |
| `f` | 以定点小数形式输出（非科学计数法） |

**输出结果**：`25.00`

---

**题目 2.2** 以下三种对齐方式 `"{:<5}"`、`"{:>5}"`、`"{:^5}"` 分别用于数字 `7` 时，输出效果是？

**答案**：
- `<` 左对齐：`7    `（7在前，空格在后）
- `>` 右对齐：`    7`（空格在前，7在后）
- `^` 居中：`  7  `（7在中间，两侧各2空格）

## 最小实践

**题目 3.1** 用一行代码实现：输出变量 `name = "小明"` 和 `score = 87.5`，使其显示为以下格式（总宽度固定，成绩保留1位小数）：

```
姓名：小明   成绩：87.5
```

**答案**：

```python
print(f"姓名：{name:<5}   成绩：{score:.1f}")
```

---

**题目 3.2** 在 Windows 系统中有一个文件路径 `D:\workspace\notes\readme.txt`，请用原始字符串正确定义它。

**答案**：

```python
path = r"D:\workspace\notes\readme.txt"
```

如果不用原始字符串，需要写成：
```python
path = "D:\\workspace\\notes\\readme.txt"
```

**题目 3.3** 用 f-string 在一行内完成：输出当前变量的类型（使用 `type()` 函数）和值。

```python
x = "hello"
# 期望输出类似：类型：<class 'str'>  值：hello
```

**答案**：

```python
print(f"类型：{type(x)}  值：{x}")
```

## 扩展思考

**题目 4.1** 为什么 Python 的 f-string 直到 3.6 版本才引入？在此之前 Python 使用什么方式实现类似功能？

**答案要点**：
- f-string 依赖 **Python 3.6 引入的 PEP 498**（Literal String Interpolation）
- 更早版本使用 `.format()` 方法或 `%` 格式化运算符
- f-string 在语法层面直接嵌入表达式，比前两者更简洁，但需要运行时解析器支持新语法

---

**题目 4.2** 如果需要在 f-string 的输出中显示花括号本身（例如打印数学集合表示），应该怎么写？

**答案**：

```python
print(f"集合 A = {{{1, 2, 3}}}")
# 输出：集合 A = {1, 2, 3}
```

**原理**：双层花括号 `{{` 转义为单个 `{`，同理 `}}` 转义为单个 `}`

## 最小替代练习

**题目 5.1（外部工具验证）** 在 Windows 资源管理器中新建一个文本文件 `D:\test\demo.txt`，然后用 Python 原始字符串读取并打印该路径：

```python
import os
path = r"D:\test\demo.txt"
print(os.path.exists(path))
```

**答案**：输出应为 `True`，确认路径字符串构造正确后文件可被操作系统识别。

> **注意**：如果不用 `r` 前缀，Python 会将 `\t` 解释为制表符、`\d` 触发语法警告，导致路径指向完全错误的位置。
