# P11 资源清单 - 【基础篇】列表及其常见操作
## 资源总览

本P教学完全依赖Python标准库，未使用任何第三方库或外部工具。所有资源均为Python内置对象或交互环境自带组件。

---

## 核心数据类型

**Python 解释器**
- 类型：运行环境
- 在本P中的作用：承载所有代码演示的执行环境，提供 list 类型及其所有内建方法
- 来源：字幕直接提到（交互式环境演示）
- 后续可练习方向：搭配 IDE（如 VS Code + Python 插件）练习脚本文件编写，或使用 Jupyter Notebook 分步调试观察列表状态变化

**list 类型**
- 类型：数据结构
- 在本P中的作用：列表是本P的核心教学对象，作为有序、可变、能装任意类型数据的容器，是Python第一种可变序列
- 来源：字幕直接提到
- 后续可练习方向：用 `[]` 创建空列表、`list()` 将其他可迭代对象转为列表，对比两者的差异与适用场景

## 内建函数

**type()**
- 类型：内建函数
- 在本P中的作用：验证列表的类型，返回 `list`，帮助建立"列表是一种独立数据类型"的概念
- 来源：字幕直接提到
- 后续可练习方向：对比 `type([])`、`type({})`、`type(())` 的返回值，加深对Python内置类型的认知

**print()**
- 类型：内建函数
- 在本P中的作用：打印列表内容、元素值、索引结果等，是演示代码输出的主要手段
- 来源：字幕直接提到
- 后续可练习方向：练习格式化输出，如打印列表时带上索引编号，模拟表格或菜单展示效果

---

## 列表方法

**list.append()**
- 类型：列表方法
- 在本P中的作用：在列表末尾追加单个元素，演示了成绩列表追加100分的场景
- 来源：字幕直接提到
- 后续可练习方向：模拟购物车添加商品、待办事项列表追加任务，理解"单元素尾追加"的使用场景

**list.extend()**
- 类型：列表方法
- 在本P中的作用：将可迭代对象拆分后逐个追加到列表末尾，演示了对字符串（拆成单字符）和列表的批量追加
- 来源：字幕直接提到
- 后续可练习方向：对比 `append([1,2])`（整体追加）和 `extend([1,2])`（拆分追加）的结果差异；用 `extend()` 合并多个列表

**list.insert()**
- 类型：列表方法
- 在本P中的作用：在指定下标位置插入单个元素，原位置及后续元素自动后移
- 来源：字幕直接提到
- 后续可练习方向：模拟在有序队列中插入优先级的场景，如成绩表中插入补考成绩到特定位置

**list.index()**
- 类型：列表方法
- 在本P中的作用：返回元素第一次出现的下标索引，找不到则抛出 `ValueError`，演示了查找数值和字符串下标的用法
- 来源：字幕直接提到
- 后续可练习方向：在包含重复元素的列表中定位目标，理解"返回首次出现"这一特性；结合循环实现多元素查找

**list.count()**
- 类型：列表方法
- 在本P中的作用：统计指定元素在列表中出现的次数，元素不存在返回0（不报错），与 `index()` 形成对比
- 来源：字幕直接提到
- 后续可练习方向：统计一段文本中各字符出现的频率（转字符列表后统计）、统计考试成绩分布

**list.remove()**
- 类型：列表方法
- 在本P中的作用：移除列表中第一个匹配到的元素，不存在则抛出 `ValueError`，演示了删除数值和字符串的场景
- 来源：字幕直接提到
- 后续可练习方向：模拟任务列表删除已完成项、购物车移除指定商品；注意其"只删第一个"的特性，理解与"删除所有"需求的区别

**list.sort()**
- 类型：列表方法
- 在本P中的作用：对列表进行原地排序，默认升序，传入 `reverse=True` 实现降序；演示了数值列表和字符串列表的排序（ASCII码比较）
- 来源：字幕直接提到
- 后续可练习方向：对字典列表按指定键排序（`sorted()` 函数用法迁移）、对中文列表排序观察 Unicode 编码顺序

**list.reverse()**
- 类型：列表方法
- 在本P中的作用：将列表顺序直接翻转（原地操作），演示了数值列表的反转效果
- 来源：字幕直接提到
- 后续可练习方向：结合 `sort()` 实现降序排列的另一种方式；模拟栈的出栈顺序（先入后出）

## 错误类型

**IndexError**
- 类型：异常类型
- 在本P中的作用：当列表索引超出范围时触发，报错信息 `List index out of range`，演示了 `LI[8]` 超出长度为5的列表的下标范围
- 来源：字幕直接提到
- 后续可练习方向：编写代码时主动触发并捕获此异常，理解边界检查的重要性

**TypeError**
- 类型：异常类型
- 在本P中的作用：当 `extend()` 传入非可迭代对象（如整数100）时触发，报错信息 `'int' object is not iterable`
- 来源：字幕直接提到
- 后续可练习方向：编写防御性代码，区分 `append()` 和 `extend()` 的参数类型要求

**ValueError**
- 类型：异常类型
- 在本P中的作用：当 `index()` 或 `remove()` 查找不存在的元素时触发，`count()` 找不到元素时返回0不报错，形成行为对比
- 来源：字幕直接提到
- 后续可练习方向：在使用 `index()` 和 `remove()` 前先用 `in` 操作符检查元素是否存在，养成安全编程习惯
