# P11 知识框架 - 【基础篇】列表及其常见操作
## 本P位置

- **章节归属**：Python基础语法 — 数据结构篇
- **前置章节**：字符串基础（P10）
- **后续章节**：元组、字典、集合（P12-P14）
- **课程地位**：列表是Python第一种**可变序列**，承上启下，后续所有数据结构均以列表为基础扩展
- **核心角色**：从"不可变"的字符串过渡到"可变"的列表，建立**可修改数据结构**的认知起点

---

## 核心问题

- **列表是什么**：有序、可变、能装任意类型数据的容器
- **列表与字符串的根本区别**：可变 vs 不可变（列表可直接修改元素，字符串不行）
- **列表能做什么**：批量存储、动态增删改查、二维数据表示

---

## 知识骨架

```
列表（list）
├── 定义与创建
│   └── [] 包裹，逗号分隔，可存任意类型
├── 访问
│   ├── 索引（正向0起，负向-1起）
│   └── 切片 [start:stop:step]（包前不包后）
├── 可修改性
│   └── 通过索引重新赋值
├── 嵌套
│   └── 外层索引 + 内层索引
├── 常用方法
│   ├── 添加：append() / extend() / insert()
│   ├── 查找：index() / count()
│   ├── 删除：remove()
│   └── 排序反转：sort() / reverse()
└── 类型
    └── type() 返回 list
```

---

## 关系与依赖

| 关系类型 | 描述 |
|---------|------|
| **索引规则继承** | 列表索引与切片规则完全继承自字符串，可直接迁移 |
| **可变 vs 不可变** | 列表与字符串的最大区别，列表操作直接修改原对象 |
| **extend() 的约束** | 依赖可迭代协议，只接受可迭代对象（字符串/列表/元组/字典/集合） |
| **嵌套访问逻辑** | 外层索引定位子列表 → 内层索引定位子元素，二级寻址 |
| **方法返回值** | append/extend/insert/remove/sort/reverse 返回 None（原地操作） |

---

## 输入 / 输出 / 前置 / 支撑

| 类型 | 内容 |
|------|------|
| **输入** | 可迭代对象（用于 extend）、下标值、元素值、布尔值（用于 sort reverse） |
| **输出** | 元素值、索引值、次数统计、None（原地修改的方法） |
| **前置知识** | 字符串的索引与切片规则、变量赋值、基本数据类型 |
| **支撑工具** | Python 解释器交互环境（用于 type() 验证类型） |

## 易错点

| 错误场景 | 正确认知 |
|---------|---------|
| 索引超出范围 | 触发 `IndexError: list index out of range`，列表长度为5时最大正向索引为4 |
| extend() 传整数 | 触发 `TypeError: 'int' object is not iterable`，必须是可迭代对象 |
| index() 找不到元素 | 触发 `ValueError`，元素不存在时报错 |
| remove() 找不到元素 | 触发 `ValueError: x not in list` |
| count() 找不到元素 | 返回 0，**不报错**，与 index() 行为相反 |
| 修改后检查类型 | 列表始终是 list 类型，可修改不等于类型变化 |
| 切片步长为负 | 从右往左切，start 默认为 -1，stop 默认为 -len-1 |
| 字符串列表排序 | 按 ASCII 码逐字符比较，首字符相同比较第二个字符 |

## 关键结论

1. 列表是**有序 + 可变 + 任意类型**的序列，核心价值在于支持原地修改
2. 索引与切片规则与字符串完全一致，可直接类比迁移
3. 添加方法按功能区分：append（单尾）、extend（多尾）、insert（指定位）
4. 查找方法按返回值区分：index（位置，报错不存在）、count（次数，返回0不存在）
5. **所有原地修改方法返回 None**，返回值不是修改后的列表
