# P11 讲解过程 - 【基础篇】列表及其常见操作
## 本P定位

本P属于Python基础语法部分，承接字符串的概念引入，讲解第二种核心数据结构——列表（list）。老师通过类比"万能储物框"建立直观认知，随后逐一展开列表的定义、访问、修改、嵌套以及常用内建函数。整P教学逻辑为：概念引入 → 定义与类型 → 索引与切片 → 可变性 → 嵌套 → 常用操作函数 → 总结。

## 讲解过程

### 一、列表的概念引入

老师先通过对比引出列表：如果说字符串是存储文字的专属容器，那列表就像是一个万能的储物框。它能够一次性装下数字、文字甚至是另一个列表这些不同类型的数据，而且里面的内容可以随时添加、修改或删除。老师强调，在日常编程中列表的使用频率很高，学会运用列表能够快速实现数据的批量管理和灵活调整，显著降低基础数据处理的难度，是入门Python必须掌握的核心技能。

### 二、列表的定义与类型验证

老师首先明确列表的定义方式：列表是一种有序的数据结合，定义时用方括号包裹元素，元素之间用逗号隔开。老师边讲边演示，在交互式环境中直接写入方括号并放入不同类型的数据进行展示。

**演示代码（直接打印不赋值）：**

```python
print(["A", "B", "C"])
print(type(["A", "B", "C"]))
```

老师演示了两种用法：不定义变量直接打印列表内容，以及使用 `type()` 函数检测类型。结果显示列表的类型为 `list`。随后老师解释了为什么要定义变量保存列表——如果一个数据需要使用多次（如既打印内容又打印类型），就定义变量保存，这样能提高数据的复用性和可维护性；如果数据只需使用一次，直接使用即可，无需定义变量占用内存。

### 三、列表的索引

老师指出列表的索引规则与字符串完全一致，可以举一反三。编号规则为：从左往右从零开始，从右往左从负一开始。通过列表名加方括号加下标值的方式就能取出对应元素。

**演示代码：**

```python
LI = ["A", "B", "C", "D", "E"]
print(LI[0])   # 输出 A
print(LI[3])   # 输出 D
print(LI[-1])  # 输出 E
```

老师特别演示了索引越界的情况：

```python
print(LI[8])
```

运行结果报错：`List index out of range`，说明下标超出范围会触发索引错误。老师提醒编号不能超出列表的长度，列表长度为5时，最大正向编号是4。

### 四、列表的切片

老师介绍切片的语法结构为 `列表[start:stop:step]`，其中 start 是起始索引（包含），默认为零；stop 是结束索引（不包含），即包前不包后原则；step 是步长，默认为1。老师强调正数从左往右切，负数从右往左切。

**演示代码：**

```python
LI = [1, 2, 3, 4, 5, 6, 7]
print(LI[:5])      # [1, 2, 3, 4, 5] 从零开始到下标5（不包含）
print(LI[::3])     # [1, 4, 7] 步长为3，从零开始正向切
print(LI[::2])     # [1, 3, 5, 7]
```

老师详细讲解了步长为负数的反向切取逻辑：`LI[::-2]` 从下标6（值为7）开始，每次减2，依次取下标6、4、2，最终得到 `[7, 3]`。

**演示三个参数全指定的例子：**

```python
print(LI[1:5:2])   # [2, 4] 从下标1开始到下标5（不包含），步长2
```

老师解释：起始下标1对应值2，下一个下标1+2=3对应值4，再下一个1+4=5不包含，所以结果为 `[2, 4]`。

### 五、列表的可修改特性

老师指出列表有一个与字符串截然不同的关键特性——可修改。字符串定义后无法直接改其中的字符，但列表的元素能够轻松修改。修改方法是通过索引找到值，再重新赋值。

**演示代码：**

```python
LI = ["A", "B", "C", "D", "E"]
LI[1] = "冰冰"      # 把 B 修改为 冰冰
print(LI)           # ['A', '冰冰', 'C', 'D', 'E']
LI[3] = "你好"      # 把 D 修改为 你好
print(LI)           # ['A', '冰冰', 'C', '你好', 'E']
```

老师强调这个特性使得列表在需要频繁调整数据的场景中非常实用。

### 六、列表嵌套

老师介绍列表嵌套就是列表里面再装一个列表，好比储物框里套小储物框，适合存储表格、矩阵等二维数据。老师用学生成绩举例，每个子列表代表一个同学的语文和数学成绩。

**演示代码：**

```python
grade = [[90, 96], [87, 68], [8, 89]]
```

老师详细讲解嵌套列表的访问逻辑：

```python
print(grade[0])      # [90, 96] 获取第一个子列表
print(grade[0][0])   # 90 获取第一个子列表的第一个元素
print(grade[0][1])   # 96 获取第一个子列表的第二个元素
print(grade[2][0])   # 8 获取第三个同学的第一门成绩
```

老师总结：列表嵌套的访问逻辑是"外层加内层"，先通过外层索引找到对应的小列表，再通过内层索引取出小列表中的元素。

### 七、添加元素的方法

老师介绍三个常用的添加元素方法：`append()`、`extend()`、`insert()`，按功能分类讲解便于记忆。

#### 7.1 append()——追加单个元素到末尾

```python
grade = [90, 87, 98, 78]
grade.append(100)
print(grade)  # [90, 87, 98, 78, 100]
```

#### 7.2 extend()——追加多个元素（必须是可迭代对象）

老师首先演示传非可迭代对象会报错：

```python
grade.extend(100)  # TypeError: 'int' object is not iterable
```

报错原因：整型不是可迭代对象。然后演示传字符串和列表：

```python
grade.extend("ABCD")   # 拆分成 'A','B','C','D' 逐个添加
grade.extend([11, 22, 33])  # 拆分成 11,22,33 逐个添加
```

老师解释：`extend()` 会将可迭代对象拆分成一个个元素，再逐一添加到列表中。字符串拆分成单个字符，列表/元组/字典/集合拆分成各自元素。

#### 7.3 insert()——在指定位置插入单个元素

```python
grade = [90, 87, 98, 78]
grade.insert(1, 100)  # 在下标1位置插入100，原元素后移
print(grade)  # [90, 100, 87, 98, 78]
```

### 八、查找元素的方法

老师介绍两个查找类方法：`index()` 和 `count()`，用法与字符串一致。

#### 8.1 index()——查找元素第一次出现的索引

```python
LI = [1, 2, 1, 2, 13, 5, 7, 2]
print(LI.index(2))    # 1 返回第一个2的下标
print(LI.index(8))    # ValueError 元素不存在会报错
```

#### 8.2 count()——统计元素出现的次数

```python
print(LI.count(2))    # 3 统计2出现的次数
print(LI.count(1))    # 2 统计1出现的次数
print(LI.count(5))    # 1
print(LI.count(8))    # 0 元素不存在返回0，不报错
```

老师对比强调：`index()` 找不到元素会报错，`count()` 找不到元素返回0不报错。

### 九、删除元素的方法

老师介绍 `remove()` 方法，用于移除列表中第一个匹配到的元素。

```python
LI = [1, 2, 1, 2, 13, 5, 7]
LI.remove(5)
print(LI)  # [1, 2, 1, 2, 13, 7]
LI.remove(2)  # 只删除第一个2，第二个2保留
LI.remove(8)  # ValueError: list.remove(x): x not in list
```

老师演示了删除不存在的元素会触发 `ValueError`，错误信息为 `x not in list`。

### 十、排序与反转

老师介绍两个方法：`sort()` 用于排序，`reverse()` 用于反转。

#### 10.1 sort()——排序

```python
LI = [2, 1, 3, 1, 5, 7, 9]
LI.sort()              # 升序（默认）
print(LI)              # [1, 1, 2, 3, 5, 7, 9]
LI.sort(reverse=True)  # 降序（设置reverse=True）
print(LI)              # [9, 7, 5, 3, 2, 1, 1]
```

#### 10.2 reverse()——直接反转列表顺序

```python
LI = [2, 1, 3, 1, 5, 7, 9]
LI.reverse()
print(LI)  # [9, 7, 5, 3, 1, 2]
```

老师演示了字符串列表也可以排序，会根据ASCII码逐字符比较：

```python
LI = ["B", "F", "D", "A", "C", "E"]
LI.sort()
print(LI)  # ['A', 'B', 'C', 'D', 'E', 'F']
```

对于首字符相同的字符串，会继续比较第二个字符。老师指出通常不会对字符串列表排序，一般只针对数值类型数据。

### 十一、课堂总结

老师最后梳理本P重点内容：

- **列表的定义**：有序的万能储物框，用方括号定义，逗号隔开元素，能装任意类型数据
- **访问方式**：索引和切片规则与字符串一致
- **可修改性**：列表支持直接修改元素
- **列表嵌套**：通过外层索引和内层索引的方式访问元素
- **常用函数**：添加用 `append()`（单元素）、`extend()`（多元素）、`insert()`（指定位置）；查找用 `index()`（找位置）、`count()`（统计次数）；删除用 `remove()`（删除第一个匹配值，不存在则报错）；排序用 `sort()`（默认升序，`reverse=True` 为降序）；反转用 `reverse()`

老师特别提醒：列表操作后会直接改变原列表；使用 `index()` 或 `remove()` 时要确保元素存在，避免报错。

## 本P结论

1. **列表的定义**：使用方括号 `[]` 包裹元素，逗号分隔，可存储任意类型数据，是有序且可修改的数据结构
2. **索引规则**：从零开始，支持负索引，访问越界会报 `IndexError`
3. **切片语法**：`[start:stop:step]`，包前不包后，正步长从左往右，负步长从右往左
4. **列表可修改**：通过索引重新赋值即可修改元素，这是与字符串的本质区别
5. **嵌套列表**：通过外层索引加内层索引逐层访问，适合表示二维数据
6. **添加元素**：`append()` 追加单元素到末尾，`extend()` 拆分可迭代对象后批量追加，`insert()` 在指定位置插入
7. **查找元素**：`index()` 返回首次出现的索引（不存在报错），`count()` 统计出现次数（不存在返回0）
8. **删除元素**：`remove()` 删除第一个匹配值，不存在则报 `ValueError`
9. **排序反转**：`sort()` 默认升序排列（`reverse=True` 为降序），`reverse()` 直接翻转列表顺序，两者均会修改原列表
10. **重要提示**：列表操作直接修改原对象，使用查找和删除方法前需确认元素存在，避免运行时错误
