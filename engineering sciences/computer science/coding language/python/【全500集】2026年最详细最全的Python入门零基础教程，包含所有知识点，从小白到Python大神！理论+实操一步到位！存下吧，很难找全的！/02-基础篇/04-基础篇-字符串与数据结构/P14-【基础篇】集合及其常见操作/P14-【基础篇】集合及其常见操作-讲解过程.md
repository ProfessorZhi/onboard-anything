## 讲解过程

### 一、引入集合：为什么需要集合？

**时间区间：[1.4-47.6]**

- 开门见山指出：集合是 Python 中"很有特色"的数据容器，核心特点鲜明——**无序且元素唯一**
- 用生活类比："就像是一个去重的篮子，不管往里面放多少重复数据，它都会把重复的自动去除掉"
- 点明应用场景：快速做交集、并集等集合运算，在**去重**或**数字对比**场景中特别好用
- 学习目标：学会运用集合后，能够**高效解决重复数据处理问题**，**简化数据对比逻辑**

### 二、认识集合：什么是集合？

**时间区间：[47.6-174.1]**

#### 1. 定义语法

- 集合是**存储唯一无序元素**的数据结构
- 定义语法：**大括号 `{}` 包裹元素，元素之间用逗号隔开**
- 最直观特点：**自动去重**

#### 2. 代码演示：验证自动去重

```python
s = {'A', 'b', 'A', 'C', 'A'}
print(s)
print(type(s))
```

- 运行后发现：原数组有 5 个元素（3 个 'A' + 'b' + 'C'），但打印结果只保留了 3 个
- 原因：重复的 'A' 被自动去除，**元素必须唯一**
- **额外发现**：每次打印结果顺序可能不同 → 引出下一特性

#### 3. 与字典的区分

- 字典也用大括号 `{}`，但字典是**键值对映射**关系（key: value）
- 集合只是**单一元素**，类似列表/元组，只是符号不同
- **易错点提醒**：花括号 `{}` 默认创建的是字典，不是集合

### 三、避坑：空集合的创建方式

**时间区间：[194.3-294.4]**

#### 1. 空花括号 `{}` 是字典

```python
data = {}
print(data)
print(type(data))  # 输出 <class 'dict'>
```

- 空花括号创建的是**空字典**（dictionary）
- 这是新手极易踩的坑

#### 2. 空集合必须用 `set()`

```python
data = set()
print(data)
print(type(data))  # 输出 <class 'set'>
```

- **正确做法**：`set()` 小括号创建空集合
- 强调：这个细节不要搞错

### 四、集合的三个核心规则

**时间区间：[297.7-371.7]**

#### 规则一：元素唯一（自动去重）

- 核心特性，无论添加多少重复数据，最终只保留一个
- 重复数据会被**自动过滤**

#### 规则二：元素无序（没有下标/索引）

- 集合**没有下标、没有索引**
- 不能像列表、元组那样按位置访问元素

```python
data = {1, 2, 3}
print(data[0])  # 报错：无法通过下标访问
```

- 无序 → 没有固定顺序 → 每次运行顺序可能不同 → 无法按位置查找
- 因为集合本身无序，没有下标，所以无法根据下标去"找"

#### 规则三：元素类型限制（必须是不可变类型）

- 集合内只能放**不可修改的数据**：
  - 数值类型（int、float）
  - 字符串（str）
  - 元组（tuple）
- **不能放**可变类型：列表、字典、集合
- 否则会报错

### 五、集合的常用内建函数

**时间区间：[397.6-693.3]**

#### 1. 添加元素

##### add()：整体添加一个元素

```python
s = {'A', 'B', 'C'}
s.add('D')
print(s)  # 添加了 D

s.add(12)
print(s)  # 添加了数字 12
```

- 一次添加**一个元素**，整体放入

##### update()：分散添加多个元素

```python
s = {'A', 'B', 'C'}
s.update('DEF')  # 字符串会被拆成 'D'、'E'、'F' 三个字符逐一添加
print(s)

s.update([1, 2, 2, 3, 3, 4, 5, 5])  # 列表也会被拆分
print(s)  # 重复的 2、3、5 会被去重
```

- 一次添加**多个元素**，但必须是**可迭代对象**（字符串、列表、元组、字典、集合）
- 注意：数值类型不能放进去，因为它不能被拆分
- update 会把可迭代对象**拆分**成单个元素再逐一添加
- 如果有重复元素，会自动去重

#### 2. 删除元素

##### remove()：删除指定元素

```python
s = {1, 2, 3, 4}
s.remove(3)  # 正常删除
s.remove(8)  # 报错：元素不存在
```

- 如果元素**不存在**，程序**会报错**

##### discard()：删除指定元素（更推荐）

```python
s = {1, 2, 3, 4}
s.discard(3)  # 正常删除
s.discard(10)  # 不存在也不会报错，程序继续运行
```

- 功能与 remove 相同
- **优点**：元素不存在时**不会报错**，更友好
- 老师**更推荐使用 discard**

##### clear()：清空所有元素

```python
s = {1, 2, 3, 4}
s.clear()
print(s)  # 输出 set()，空集合
```

- 语法简单，调用时无参数
- 执行后集合变成**空集合**（仍是 set 类型，只是没有元素了）
- 集合对象本身仍然存在，可以继续进行增删改查操作

### 六、集合的运算（重点功能）

**时间区间：[695.2-951.4]**

集合最强大、最实用的功能：**集合运算**，能快速对比两个集合

#### 1. 交集（Intersection）

- **定义**：找两个集合的**共同元素**
- **符号**：`&`（麻花符号）
- **方法**：`intersection()`
- **图示说明**：A 和 B 共有的元素组成交集

```python
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

# 方式一：符号
result = s1 & s2
print(result)  # {3, 4}

# 方式二：方法
result = s1.intersection(s2)
print(result)  # {3, 4}
```

- s1 有 s2 也有 → 只有 3 和 4 → 结果就是 {3, 4}
- 快速找出两个集合的**重叠部分**

#### 2. 并集（Union）

- **定义**：合并两个集合的**所有元素**
- **符号**：`|`（竖杠）
- **方法**：`union()`
- **图示说明**：A 有的 + B 有的，所有元素整合到一起

```python
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

# 方式一：符号
result = s1 | s2
print(result)  # {1, 2, 3, 4, 5, 6}

# 方式二：方法
result = s1.union(s2)
print(result)  # {1, 2, 3, 4, 5, 6}
```

- 合并两个集合的所有元素，**重复的 3、4 自动去重**
- **支持多个集合**：

```python
s3 = {5, 6, 7, 8}
result = s1.union(s2, s3)
print(result)  # {1, 2, 3, 4, 5, 6, 7, 8}
```

### 七、实用技巧

**时间区间：[1057.6-1192.8]**

#### 技巧一：去重优先使用集合

列表去重用集合转换，效率远高于手动写逻辑：

```python
li = [1, 1, 5, 1, 8, 7, 9, 2, 3, 5]
# 如果用列表操作：要写 if-else 判断元素是否存在，很麻烦

# 快速去重：列表 → 集合 → 列表
result = list(set(li))
print(result)  # 重复元素自动去除
```

- 列表转集合：自动去重
- 集合转列表：恢复列表形式
- 省略大量去重逻辑，效率更高
- 老师提示：类型转换是下一小节内容，这里先预告

#### 技巧二：数据对比用集合运算

- 找两个列表的共同元素 → 用**交集**
- 合并两个列表去重 → 用**并集**
- 比手写复杂逻辑更高效

### 八、课堂总结

**时间区间：[956.3-1199.1]**

- **本质**：唯一无序的去重数据结构，核心作用是去重
- **定义**：使用 `{}`，空集合用 `set()`
- **三大特性**：
  1. 自动去重
  2. 没有索引（不能按位置访问）
  3. 支持集合运算
- **常用操作**：
  - 添加：add() 添加一个，update() 添加多个（可迭代对象）
  - 删除：discard() 更推荐（不存在不报错），remove() 会报错，clear() 清空
- **集合运算**：交集 `&` / intersection()，并集 `|` / union()
- **实用价值**：去重和数据对比场景优势明显，能**少写很多代码**

## 本P结论

1. **集合定义**：`{}` 包裹元素，空集合必须用 `set()`
2. **三大核心特性**：元素唯一（自动去重）、无序（无索引）、元素必须是不可变类型
3. **添加操作**：add() 添加一个元素，update() 添加多个（可迭代对象）
4. **删除操作**：discard() 比 remove() 更安全（不存在不报错），clear() 清空所有元素
5. **集合运算**：交集 `&` 或 intersection()，并集 `|` 或 union()
6. **核心应用**：列表去重（list → set → list）和数据对比（找共同元素/合并去重）
7. **易错点**：空 `{}` 是字典不是集合，空集合要用 `set()`
