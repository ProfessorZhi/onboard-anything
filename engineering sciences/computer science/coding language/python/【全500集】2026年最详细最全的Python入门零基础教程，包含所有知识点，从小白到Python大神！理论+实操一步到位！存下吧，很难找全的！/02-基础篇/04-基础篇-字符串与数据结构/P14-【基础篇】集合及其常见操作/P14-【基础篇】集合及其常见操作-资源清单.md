## 一、Python 内建类型与构造函数

### 1. `set` 类型

| 字段 | 内容 |
|------|------|
| **名称** | `set`（内建集合类型） |
| **类型** | Python 内建数据类型 |
| **在本P中的作用** | 本节核心对象；承载"无序、唯一"语义，用于自动去重与集合运算 |
| **来源** | 字幕直接演示；Python 标准语言规范内建 |
| **建议方向** | 阅读 Python 官方文档中 `set` 类型页面，对比 `frozenset`（不可变集合）的使用场景；尝试在实际去重任务中替换手写 for 循环 |

---

### 2. `set()` 构造函数

| 字段 | 内容 |
|------|------|
| **名称** | `set()` |
| **类型** | Python 内建构造函数 |
| **在本P中的作用** | 创建空集合的唯一正确方式；区别于空花括号 `{}` 默认生成字典的陷阱 |
| **来源** | 字幕直接演示 |
| **建议方向** | 练习用 `set(iterable)` 将列表、字符串、元组直接转换为集合；与 `list()`、`tuple()` 对比理解类型转换统一规律（该主题在下一节展开） |

---

## 二、集合实例方法

### 3. `set.add()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.add(elem)` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 向集合中整体添加一个元素；演示了添加字符串和整数两种类型 |
| **来源** | 字幕直接演示 |
| **建议方向** | 测试向集合中 add 一个列表会发生什么（预期报 TypeError），加深对"元素必须不可变"规则的理解 |

---

### 4. `set.update()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.update(iterable)` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 将可迭代对象逐一拆分后批量添加到集合；演示了传入字符串和列表两种场景；自动去重效果同步体现 |
| **来源** | 字幕直接演示 |
| **建议方向** | 分别传入字符串、列表、元组、字典、集合，观察各类型拆分行为差异；特别测试传入字典时 key 还是 value 被加入 |

---

### 5. `set.remove()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.remove(elem)` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 删除指定元素；演示了元素不存在时抛出 `KeyError` 的报错行为，作为反面参照引出 `discard` |
| **来源** | 字幕直接演示 |
| **建议方向** | 用 `try / except KeyError` 包裹 `remove`，练习异常捕获基础用法；理解为何生产场景更倾向 `discard` |

---

### 6. `set.discard()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.discard(elem)` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 安全删除指定元素；元素不存在时静默忽略，不抛出异常；讲师明确标注为推荐方式 |
| **来源** | 字幕直接演示 |
| **建议方向** | 对比 `remove` 与 `discard` 的异常行为，总结"需要报错感知时用 remove，容错场景用 discard"的选用原则 |

---

### 7. `set.clear()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.clear()` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 清空集合所有元素，保留空集合对象（结果为 `set()`，类型仍是 `set`） |
| **来源** | 字幕直接演示 |
| **建议方向** | 对比 `clear()` 与 `del s` 的区别（一个清空内容，一个销毁变量）；与列表的 `clear()` 行为横向对比 |

---

### 8. `set.intersection()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.intersection(*others)` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 返回两个集合的共同元素（交集）；与 `&` 运算符等价；演示了找共同数字的场景 |
| **来源** | 字幕直接演示 |
| **建议方向** | 对比方法调用与运算符两种写法在可读性上的差异；尝试传入三个集合求多集合交集 |

---

### 9. `set.union()`

| 字段 | 内容 |
|------|------|
| **名称** | `set.union(*others)` |
| **类型** | 集合实例方法 |
| **在本P中的作用** | 返回多个集合合并后去重的并集；演示了双集合和三集合两种调用方式；与 `|` 运算符等价 |
| **来源** | 字幕直接演示 |
| **建议方向** | 实验 `s1 | s2 | s3` 与 `s1.union(s2, s3)` 的结果是否一致；思考为何 union 支持多参数而 `|` 需链式调用 |

---

## 三、集合运算符

### 10. `&` 运算符（交集）

| 字段 | 内容 |
|------|------|
| **名称** | `&`（集合交集运算符） |
| **类型** | Python 内建运算符 |
| **在本P中的作用** | `s1 & s2` 求交集，与 `intersection()` 方法等价；讲师称为"麻花符号" |
| **来源** | 字幕直接演示 |
| **建议方向** | 查阅 Python 数据模型文档，了解 `&` 对应的魔术方法 `__and__`；尝试用于 `frozenset` |

---

### 11. `|` 运算符（并集）

| 字段 | 内容 |
|------|------|
| **名称** | `\|`（集合并集运算符） |
| **类型** | Python 内建运算符 |
| **在本P中的作用** | `s1 \| s2` 求并集，与 `union()` 方法等价；演示了合并两集合并自动去重 |
| **来源** | 字幕直接演示 |
| **建议方向** | 扩展了解差集运算符 `-`（`difference()`）和对称差 `^`（`symmetric_difference()`），构成完整集合运算体系 |
