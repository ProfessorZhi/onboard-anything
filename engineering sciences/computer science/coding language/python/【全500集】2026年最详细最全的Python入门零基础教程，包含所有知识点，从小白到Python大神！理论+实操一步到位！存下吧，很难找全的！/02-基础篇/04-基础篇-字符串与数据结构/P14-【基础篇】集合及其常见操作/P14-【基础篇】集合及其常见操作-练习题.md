### 1. B、C
`{}` 创建的是空字典，不是空集合。必须用 `set()` 或 `set([])` 创建空集合。

### 2. C、D
集合元素必须是不可变类型。列表、字典、集合本身都是可变的，不能作为集合元素；元组（不可变）和字符串可以。

### 3. B
集合要求元素唯一，当插入已存在的元素时，实际上是用新值覆盖旧值，从而实现去重。这是集合的数学定义决定的，不是 Python 的特殊算法。

### 4. B
`add(x)` 将 x 作为整体添加；`update(iterable)` 会先拆分 iterable，再逐一添加元素。

### 5. C
`remove()` 删不存在元素会抛出 KeyError。若想安全删除，应使用 `discard()`。

### 8. 
```python
def unique_list(lst):
    seen = set()
    result = []
    for item in lst:
        if item not in seen:
            seen.add(item)
            result.append(item)
    return result
```

### 9. 
集合使用哈希表实现，成员检测时间复杂度为 O(1)；列表需要逐个遍历，时间复杂度为 O(n)。当数据量大时，集合的查找效率远高于列表。

### 10.

# 优势：
# 1. 内存占用降低（重复ID只保留一份）
# 2. 后续查找效率高（O(1) 成员检测）
# 3. 集合运算方便（快速计算用户间的共同购买商品等）
