## 核心问题
如何对字符串执行运算、访问单个字符、截取子串？

---

## 知识骨架

**字符串运算符**
- `+` → 拼接，两个操作数必须同为str，否则TypeError
- `*` → 重复，让字符串重复指定次数
- `in` / `not in` → 成员检查，返回bool

**索引（Indexing）**
- 正向索引：从0开始，依次0, 1, 2...
- 负向索引：从-1开始（倒数第一），依次-1, -2...
- 越界访问 → IndexError

**切片（Slicing）**
- 语法：`str[start:stop:step]`
- 核心原则：包前不包后
- 三参数均可省略，使用默认值（start=0, stop=末尾, step=1）
- step为负：从右往左
- 三参数全省略时冒号不可省

---

## 关系与依赖

```
数据类型基础（str概念）
    ↓
字符串运算（+ * in/not in）→ 输出：str / bool
    ↓
索引（正向/负向）→ 输出：单个字符（str）
    ↓
切片（基于索引的扩展）→ 输出：子串（str）
```

- 索引是切片的基础
- 字符串运算是最基础的组合操作
- 切片语法后续被列表、元组等复用

## 易错点

- `str + int` → TypeError（类型不匹配）
- `str[1]` 取的是**第二个字符**（索引从0开始）
- 切片 `[2:14]` 不包含索引14的字符
- `in` / `not in` 是整体子串匹配，不是逐字符随机组合
- `[:]` 省略全部参数时冒号必须保留，否则被识别为索引操作
- 负向索引从-1开始，不是从-0开始
