### 二、字符串的运算符

#### 1. 字符串拼接（+）

- 讲解思路：先回顾数值相加时 `+` 的意义（算术加法），再切换到字符串场景，类比为"一卷胶带"，能将多个字符串"牢牢粘在一起"。
- 演示代码：

```python
# 字面量拼接
print("A" + "B")  # 输出 "AB"

# 变量拼接
str1 = "美女"
str2 = "你好呀"
str3 = "!"
print(str1 + str2 + str3)  # 输出 "美女你好呀!"
```

- 强调一个**常见错误**：

```python
# 错误写法：字符串和整数直接拼接会报 TypeError
age = "年龄"
print(age + 18)  # TypeError: can only concatenate str (not "int") to str
```

- 解释报错原因：类型不一致，就像"一个人加一头猪"，无法确定结果是什么类型。如果要拼接，必须先将数字转为字符串。

#### 2. 字符串重复（*）

- 讲解思路：类比算术运算符中的乘法（`3 * 2 = 6`），但在字符串中 `*` 表示**重复**，即让字符串重复指定次数。
- 演示代码：

```python
print("美女" * 3)  # 输出 "美女美女美女"
print("-" * 10)   # 输出 "----------"
```

- 实用场景：快速生成分隔线，例如在输出内容之间插入 `"=" * 20` 或 `"-" * 10`。

#### 3. 成员运算符（in / not in）

- 讲解思路：将其比喻为"侦探"，专门检查一个子串是否存在于另一个字符串中，返回布尔值 `True` 或 `False`。
- 演示代码：

```python
name = "冰冰"
print("冰" in name)       # True
print("冰" not in name)   # False
print("BIN" in name)      # True（整体匹配，"冰冰"中确实包含"BIN"）
print("BN" in name)       # False（整体匹配，"冰冰"中不包含连续的"BN"）
print("冰冰" not in name) # False（因为"冰冰"在name中）
```

- 强调：成员运算符是基于**整体子串**的匹配，不是逐字符拆分后随意组合。

### 三、索引（下标）

#### 1. 正向索引（从左往右）

- 核心规则：**从 0 开始**编号，不是从 1 开始。
- 演示代码：

```python
name = "冰冰"
print(name[0])  # "冰"（第一个字符）
print(name[1])  # "冰"（第二个字符）
print(name[2])  # "冰"（第三个字符）
print(name[3])  # "冰"（第四个字符）
print(name[4])  # IndexError: string index out of range
```

- 强调易错点：`name[1]` 取的是**第二个字符**，因为索引从 0 开始。如果访问超出范围的索引会抛出 `IndexError`，错误信息为 "String index out of range"。

#### 2. 负向索引（从右往左）

- 核心规则：**从 -1 开始**，-1 表示倒数第一个字符，-2 表示倒数第二个，以此类推。
- 演示代码：

```python
name = "冰冰"
print(name[-1])  # "冰"（倒数第一个）
print(name[-2])  # "冰"（倒数第二个）
print(name[-3])  # "冰"（倒数第三个）
print(name[-4])  # "冰"（倒数第四个）
print(name[-5])  # IndexError（没有倒数第五个）
```

- 再次提醒：超出范围同样会报错。

### 四、切片（Slicing）

#### 1. 切片的基本概念

- 类比"一把锋利的剪刀"，能从字符串中精准截取任意一段内容，返回一个新字符串。
- 语法结构：

```
字符串[起始索引:结束索引:步长]
```

三个参数均为**可选参数**，各有默认值。

#### 2. 参数详解

| 参数 | 名称 | 含义 | 默认值 |
|------|------|------|--------|
| start | 起始索引 | 包含该位置字符 | 0（从头开始） |
| stop | 结束索引 | **不包含**该位置字符（包前不包后） | 字符串末尾 |
| step | 步长 | 每隔多少个字符取一次 | 1（从左往右） |

- **包前不包后原则**：切片结果包含起始索引的字符，但**不包含**结束索引的字符。

#### 3. 省略所有参数

```python
my_string = "Hello world冰冰"
print(my_string[:])  # 完整复制整个字符串
```

- 分析：起始默认 0，结束默认末尾，步长默认 1，所以完整输出原字符串。**注意**：如果三个参数全省略，必须保留冒号（`[]` 不写冒号会被识别为索引操作）。

#### 4. 单独指定起始索引

```python
print(my_string[2:])  # 从下标2开始，一直切到最后
```

- 以 `"Hello world冰冰"` 为例，索引 0 是 H，1 是 e，2 是 l，所以输出 `"llo world冰冰"`。

#### 5. 单独指定结束索引

```python
print(my_string[:2])  # 从头开始，切到下标2之前（不包含2）
```

- 讲解"包前不包后"原则：起始默认 0（包含 H），结束到下标 2（不包含 l），所以输出 `"He"`。

#### 6. 单独指定步长（正数）

```python
print(my_string[::5])  # 从0开始，步长为5，切到末尾
```

- 逐步推导：
  - 起始 0 → 取 H
  - 0 + 5 = 5 → 取空格
  - 5 + 5 = 10 → 取 d
  - 10 + 5 = 15 → 取 l
  - 15 + 5 = 20 → 取 i
  - 20 + 5 = 25 → 取 n
  - 25 + 5 = 30 → 超范围，结束
- 输出：`"H dlin"`

#### 7. 步长为负数（反向切片）

```python
print(my_string[::-8])  # 从右往左切，步长8
```

- 讲解从最后一个字符（下标 26，G）开始，依次 26-8=18（空格），18-8=10（D），10-8=2（L），输出：`"G DL"`。

#### 8. 结合使用三个参数

```python
print(my_string[2:14:2])  # 从2开始，到14之前结束，步长2
```

- 逐步推导：
  - 起始 2 → 取 L
  - 2 + 2 = 4 → 取 O
  - 4 + 2 = 6 → 取 W
  - 6 + 2 = 8 → 取 R
  - 8 + 2 = 10 → 取 D
  - 10 + 2 = 12 → 取空格
  - 12 + 2 = 14 → 到达结束索引，**不包含**，结束
- 输出：`"LOWRD "`

## 本P结论

1. **字符串拼接**（`+`）：将两个或多个字符串连接成一个，要求操作数**均为字符串类型**，否则报 `TypeError`。
2. **字符串重复**（`*`）：让字符串重复指定次数，常用于快速生成分隔线等重复内容。
3. **成员运算符**（`in` / `not in`）：检查子串是否存在于字符串中，返回布尔值 `True`（成立）或 `False`（不成立）。
4. **索引**：每个字符都有专属编号，从左往右从 **0** 开始，从右往左从 **-1** 开始。超出范围会报 `IndexError`。
5. **切片**（`[start:stop:step]`）：精准截取字符串的任意片段，核心原则是**包前不包后**（包含起始索引，不包含结束索引），三个参数均可省略（使用默认值），步长为正数表示从左往右取，为负数表示从右往左取。
