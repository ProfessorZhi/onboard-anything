## 二、Python 内置语法结构

> 以下条目均为 Python 语言规范自带，无需安装任何第三方包。

### 3. 字符串拼接运算符 `+`

| 字段 | 内容 |
|------|------|
| 名称 | 字符串拼接运算符 `+` |
| 类型 | 内置运算符 / 语法结构 |
| 在本P中的作用 | 将多个字符串字面量或字符串变量连接为一个新字符串；演示了合法写法与 `str + int` 引发 `TypeError` 的非法写法 |

### 4. 字符串重复运算符 `*`

| 字段 | 内容 |

| 类型 | 内置运算符 / 语法结构 |

---

### 5. 成员运算符 `in` / `not in`

| 字段 | 内容 |
|------|------|
| 名称 | 成员运算符 `in` / `not in` |
| 类型 | 内置运算符 / 语法结构 |
| 在本P中的作用 | 检查子串是否整体存在于目标字符串中，返回布尔值；演示了连续子串匹配与非连续字符不匹配的对比 |
| 来源 | 字幕直接提到 |
| 建议方向 | 结合 `if` 语句练习条件判断（如"判断用户输入是否包含敏感词"）；注意与 `find()`、`index()` 方法的区别（后者返回位置而非布尔值） |

---

### 6. 字符串索引语法 `str[n]`

| 字段 | 内容 |
|------|------|
| 名称 | 字符串索引（Indexing）`str[n]` |
| 类型 | 内置语法结构 |
| 在本P中的作用 | 按位置访问字符串中的单个字符；演示了正向索引（从 0 开始）与负向索引（从 -1 开始）两套编号规则，以及越界触发 `IndexError` 的场景 |
| 来源 | 字幕直接提到 |
| 建议方向 | 练习用索引取出任意字符串的首字符与末字符（固定写 `[0]` 与 `[-1]`）；刻意写越界索引并阅读 `IndexError` 报错，建立对边界的直觉 |

---

### 7. 字符串切片语法 `str[start:stop:step]`

| 字段 | 内容 |
|------|------|
| 名称 | 字符串切片（Slicing）`str[start:stop:step]` |
| 类型 | 内置语法结构 |
| 在本P中的作用 | 截取字符串任意子段；本P系统演示了省略全部参数（`[:]`）、单独指定起始、单独指定结束、正步长间隔取字符、负步长反向切片、三参数联合使用共六种组合 |
| 来源 | 字幕直接提到 |
| 建议方向 | 练习用 `[::-1]` 将字符串翻转，验证结果；练习用步长 2 提取字符串的奇数位/偶数位字符；在列表上复用同一套切片语法，对比与字符串切片的异同（本P知识框架已提示列表共享该语法） |

## 三、内置异常类型

> 以下两项为 Python 内置异常，本P均通过错误演示直接触发。

### 8. `TypeError`

| 字段 | 内容 |
|------|------|
| 名称 | `TypeError` |
| 类型 | Python 内置异常 |
| 在本P中的作用 | 演示字符串与整数直接拼接（`str + int`）时由解释器抛出，提示类型不匹配 |
| 来源 | 字幕直接提到 |
| 建议方向 | 练习用 `try / except TypeError` 捕获该异常并给出友好提示；搜索官方文档了解 `TypeError` 在哪些场景下会被触发 |

---

### 9. `IndexError`

| 字段 | 内容 |
|------|------|
| 名称 | `IndexError: string index out of range` |
| 类型 | Python 内置异常 |
| 在本P中的作用 | 演示正向/负向索引越界时由解释器抛出，帮助理解字符串长度边界 |
| 来源 | 字幕直接提到（原文报错信息："String index out of range"） |
| 建议方向 | 练习用 `len()` 函数获取字符串长度后再做索引，避免越界；尝试用 `try / except IndexError` 捕获并处理该异常 |
