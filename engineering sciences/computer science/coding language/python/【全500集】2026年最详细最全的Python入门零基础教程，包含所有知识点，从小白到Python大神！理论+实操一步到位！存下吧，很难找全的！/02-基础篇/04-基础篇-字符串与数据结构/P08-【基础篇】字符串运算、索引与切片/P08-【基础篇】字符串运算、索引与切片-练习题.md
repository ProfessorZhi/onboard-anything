# P08 练习题 - 【基础篇】字符串运算、索引与切片
---

## 结构识别

**题目 1.1**
已知字符串 `s = "Python"`，判断以下表达式的返回值类型：

```
s[0]
s[-1]
s[0:3]
s[::2]
```

每个表达式的返回值类型是 `str` 还是 `int`？

---

**题目 1.2**
以下代码会报错吗？如果不报错，输出结果是什么？

```python
"hello" + "world"
"hello" * 3
"a" in "banana"
"x" not in "apple"
```

---

## 机制理解

**题目 2.1**
解释为什么执行 `"hello" + 123` 会报错，而 `"hello" * 3` 不会报错？它们的运算符不同，但操作数类型相似，为什么结果不同？

---

**题目 2.2**
切片语法 `s[start:stop:step]` 中，"包前不包后"是什么意思？请用 `s = "ABCDEF"` 和 `s[1:4]` 为例说明。

---

**题目 2.3**
为什么字符串的负向索引从 `-1` 开始而不是 `-0`？用 `s = "Python"` 验证 `s[-0]` 的实际行为。

---

## 最小实践

**题目 3.1**
编写代码实现：将字符串 `"Hello World"` 逆序输出，期望结果为 `"dlroW olleH"`。要求使用切片完成。

---

**题目 3.2**
给定用户输入的身份证号 `id_card = "110101199001011234"`，提取并拼接：
- 出生年（4位）
- 出生月（2位）
- 出生日（2位）

输出格式：`"1990年01月01日"`

---

**题目 3.3**
写一个表达式，从字符串 `"abcdef"` 中每隔一个字符取一个，结果应为 `"ace"`。

## 扩展思考

**题目 4.1**
切片的三要素 `start`、`stop`、`step` 都可以省略。请思考：省略后的默认值分别是什么？用 `s = "Python"` 验证 `s[:]`、`s[::]`、`s[::2]` 的运行结果。

---

**题目 4.2**
`in` 和 `not in` 是子串匹配，而非逐字符随机组合。请问：
- `"ab" in "abc"` 返回什么？
- `"ac" in "abc"` 返回什么？为什么？

这对判断字符串包含关系时有什么启示？

---

**题目 4.3（延伸）**
本 P 学习的索引和切片语法，后续会被列表、元组等复合数据类型复用。请思考：列表的索引和切片与字符串有何异同？尝试对列表 `lst = [1, 2, 3, 4, 5]` 使用 `lst[::2]` 验证。
