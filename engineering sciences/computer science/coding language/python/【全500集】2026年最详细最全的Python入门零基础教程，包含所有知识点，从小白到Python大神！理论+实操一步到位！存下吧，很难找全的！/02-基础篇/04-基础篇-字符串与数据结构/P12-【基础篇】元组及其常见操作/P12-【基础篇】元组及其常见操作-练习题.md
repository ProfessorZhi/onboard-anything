# P12 练习题 - 【基础篇】元组及其常见操作
## 结构识别

**题目 1.1** 下列代码运行后，`type(t)` 的输出是什么？

```python
t = (1)
t = (1,)
print(type(t))
```

> 考察点：单元素元组必须加尾部逗号的语法规则。

**题目 1.2** 判断正误：元组支持使用 `append()` 方法向末尾添加元素。

> 考察点：元组不可变性这一核心特性。

## 机制理解

**题目 2.1** 代码运行后，变量 `a` 和 `b` 的值分别是多少？

```python
my_tuple = (1, 2, 3, 2, 2)
a = my_tuple.index(2)
b = my_tuple.count(2)
c = my_tuple.index(5)
```

> 考察点：`index()` 与 `count()` 对"不存在元素"的不同处理机制。

> 考察点：不可变性的设计意图与实用价值。

## 最小实践

**题目 3.1** 给定元组 `weekday = ("周一", "周二", "周三", "周四", "周五")`，请完成：
1. 使用负数索引获取 `"周三"`；
2. 使用切片获取 `("周二", "周三", "周四")`；
3. 判断 `"周六"` 是否在元组中。

```python
weekday = ("周一", "周二", "周三", "周四", "周五")
# 你的代码
```

> 考察点：元组索引、切片、成员判断的语法与返回值。

**题目 3.2** 下列代码会触发哪种异常？运行程序后，控制台输出的错误信息是什么？

```python
my_tuple = (1, 2, 3)
my_tuple[0] = 10
```
