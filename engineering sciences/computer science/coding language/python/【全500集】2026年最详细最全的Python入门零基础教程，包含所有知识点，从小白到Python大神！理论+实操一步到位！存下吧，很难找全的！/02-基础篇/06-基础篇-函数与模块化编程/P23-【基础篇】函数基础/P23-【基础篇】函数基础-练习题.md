## 答案

**1. B**  
`def` 是定义函数的关键字，冒号不可省略。

**2. C**  
函数调用必须在名字后加括号 `()`，括号才会触发执行。

**3.**  
- 代码段 A：`greet()` 调用在定义之前 → `NameError`  
- 代码段 B：缺少冒号 `def welcome:` → 语法错误

**4.**  
输入（参数） → 函数体 → 输出（返回值）

**5.**  
函数定义后没有调用语句，`print("I'm here")` 永远不会被执行。

**6.**（答案不唯一，合理即可）  
- 减少重复代码：同一逻辑只需写一遍  
- 提高可读性：函数名即注释，代码结构更清晰  
- 降低维护成本：修改函数体即可全局生效  
- 复杂任务拆分：把大任务分解为多个独立的小函数

**7.**

```python
def triple_hello():
    print("Hello")
    print("Hello")
    print("Hello")

triple_hello()
```

**8.**

```python
print("Good morning")
print("Good morning")
print("Good morning")
```

可见函数的复用价值——当逻辑重复 3 次以上时，函数更优。

```
A
B
A
```

**10.**  
- 不用函数：需要修改 10 次  
- 使用 `report()` 函数：只需修改 1 次（函数定义处）  
- 体现的核心好处：**降低维护成本**

**11.**（答案不唯一，合理即可）  
循环用于重复执行同一段代码（次数已知或循环条件明确），强调的是流程控制；函数用于封装可复用的逻辑单元，强调的是代码组织与调用。循环解决的是「怎么做」，函数解决的是「怎么组织」。两者可结合使用：函数内部可以包含循环，循环内部也可以调用函数。
