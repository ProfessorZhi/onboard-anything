## 知识骨架

```
函数
├── 概念：代码工具箱，封装重复逻辑的独立单元
├── 工作模型：输入（参数）→ 函数体 → 输出（返回值）
├── 核心好处
│   ├── 减少重复代码
│   ├── 代码可读性提升
│   ├── 维护成本降低
│   └── 复杂任务拆分
├── 定义语法
│   └── def 函数名(参数): 函数体 [return]
└── 调用方式
    └── 函数名()
```

## 易错点

1. **先定义后调用**：调用代码写在定义之前 → `NameError`
2. **不调用不执行**：只定义不调用 → 函数体代码永不执行
3. **缩进错误**：函数体缩进不一致 → `IndentationError`
4. **括号遗漏**：写 `greet` 而非 `greet()` → 不执行函数，只引用对象
