# P27 知识框架 - 【基础篇】案例：函数版学员管理系统
## 本P位置

Python入门课程 · 基础篇 · 函数章节的实战延伸
前序：函数定义与调用、列表基础、字典基础
后置：文件读写（数据持久化）、面向对象（类与对象重构）

## 核心问题

如何用函数模块化 + 循环控制 + 数据结构组合，构建一个可交互的增删改查系统。

## 知识骨架

### 一、系统架构模式

```
while True（死循环）
├── 显示菜单 → info_print()
├── 接收输入 → input()
├── 多分支处理 → if-elif-else
│   ├── "1" → add_info()
│   ├── "2" → del_info()
│   ├── "3" → modify_info()
│   ├── "4" → search_info()
│   ├── "5" → print_all()
│   └── "6" → 二次确认 → break
└── break 退出循环
```

### 二、数据结构设计

```
info: List[Dict]  # 全局列表
Dict 键: "id" | "name" | "tel"
示例: [{"id": "1", "name": "冰冰", "tel": "130111222"}]
```

### 三、六功能核心逻辑

| 功能 | 查找方式 | 操作 | 异常处理 |
|------|----------|------|----------|
| 添加 | 遍历姓名去重 | append | 重复→return |
| 删除 | 遍历姓名匹配 | remove | 未找到→提示 |
| 修改 | 遍历姓名匹配 | 键直接赋值 | 未找到→提示 |
| 查询 | 遍历姓名匹配 | 打印字典 | 未找到→提示 |
| 显示 | 遍历全列表 | 格式化打印 | 空列表→提示 |
| 退出 | 无 | break | 二次确认拦截 |

### 四、控制流语法对比

| 语法 | 作用范围 | 本P应用场景 |
|------|----------|-------------|
| return | 结束整个函数 | 添加重复/删除成功/修改成功 → 终止函数 |
| break | 退出当前循环 | 用户确认退出 → 终止while死循环 |
| for+else | 循环完整执行 | 未找到目标 → else分支触发 |

## 关系与依赖

```
函数调用链
main()
├── info_print()  [无依赖，输出型]
├── add_info()    [依赖全局info，写入]
├── del_info()    [依赖全局info，写入]

全局状态

```

## 易错点

### 1. return 与 break 混用

```python
# 错误：在功能函数中用break退出循环，但函数继续执行
for student in info:
    if new_name == student["name"]:
        break  # 仅退出for，函数继续执行后续代码
print("无法添加")  # 这行仍会执行

# 正确：检测到异常条件时用return终止整个函数
for student in info:
    if new_name == student["name"]:
        return  # 终止函数，不再执行后续
# 无需else分支
```

### 2. 全局变量未定义

```python
# 错误：在函数内直接赋值，Python视为创建局部变量
def add_info():
    info.append(...)  # 报错：local variable 'info' referenced before assignment

# 正确方式：确保变量在函数外定义，或在函数内用global声明
info = []  # 模块顶层定义
```

### 3. 退出未二次确认

```python
# 错误：直接退出，误操作无法挽回
if user_num == "6":
    break  # 危险

# 正确：加入确认步骤
if user_num == "6":
    confirm = input("确认退出？（yes/no）")
    if confirm == "yes":
        break
```

### 4. 字典修改误用中转变量

```python
# 不必要：借助临时变量中转
temp = student["tel"]
temp = input("请输入新手机号")
student["tel"] = temp

# 直接赋值：字典键名直接接收input结果
student["tel"] = input("请输入新手机号")
```
