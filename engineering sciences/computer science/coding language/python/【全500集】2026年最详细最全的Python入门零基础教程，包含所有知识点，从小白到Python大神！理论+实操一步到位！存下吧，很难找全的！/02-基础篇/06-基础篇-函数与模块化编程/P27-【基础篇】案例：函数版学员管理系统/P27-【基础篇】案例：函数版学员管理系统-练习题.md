## 参考答案（简要）

**1-2**  
- 手机号：`130111222`
- 获取学号：`info[1]["id"]`
- 添加：`info.append({"id": "3", "name": "浩浩", "tel": "132333444"})`

**2-1**  
- 输入"冰冰"：打印"姓名已存在" → 打印"可以添加"（break只退出for）
- 输入"芳芳"：直接打印"可以添加"（for遍历完执行else，然后执行print）

**2-2**  
- `return`：找到重复立即终止函数，后续"可以添加"不打印
- `break`：仅退出for循环，函数继续执行，后续"可以添加"仍打印

**2-3**  

| 场景 | 控制语句 | 原因 |
|------|----------|------|
| 终止 while 死循环 | break | 退出当前循环结构 |
| 终止当前函数 | return | 结束整个函数执行 |
| 未找到目标 | for + else | else在循环完整执行后触发 |

**3-3 框架示例**

```python
info = []

def info_print():
    print("1-添加 2-删除 3-修改 4-查询 5-显示 6-退出")

def add_info():
    name = input("姓名：")
    for s in info:
        if name == s["name"]:
            print("已存在")
            return
    info.append({"id": "3", "name": name, "tel": input("手机号：")})
    print("添加成功")

def main():
    while True:
        info_print()
        num = input(":")
        if num == "1":
            add_info()
        elif num == "6":
            if input("确认退出？") == "yes":
                break

main()
```
