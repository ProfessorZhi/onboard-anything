**题目 2.2**
输出：`a=1, args=(2,), b=10, kwargs={'a': 3}`

解析：按顺序 a=1，剩余位置参数被 *args 收集，b 使用默认值，a=3 被 **kwargs 收集。

**题目 2.3**
默认参数 `x=5` 位于位置参数 `y` 左侧，Python 语法不允许，因为调用时无法判断参数归属。

def power(base, exponent=2):
    return base ** exponent

def collect_items(*items):

def build_profile(**attributes):

**题目 4.1**
最小调用次数：1 次（只需传 `a`）
最大调用次数：无限制（`*args` 和 `**kwargs` 可接收任意多参数）

**题目 4.2**

```python
def calc(mode, *nums, precision=2):
    if not nums:
        return 0
    if mode == "sum":
        result = sum(nums)
    elif mode == "avg":
        result = sum(nums) / len(nums)
    else:
        raise ValueError("mode 必须是 'sum' 或 'avg'")
    return round(result, precision)
```
