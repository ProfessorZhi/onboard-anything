### 4. Python 内置数据类型：元组（Tuple）与字典（Dict）

- **类型**：内置数据类型
- **在本 P 中的作用**：`*args` 可变参数会将接收到的位置参数自动打包成**元组**，`**kwargs` 关键字可变参数会将接收到的 `key=value` 形参自动打包成**字典**。理解这两种数据类型是掌握可变参数机制的基础。
- **来源**：基于上下文推断（课程虽未专题讲解元组和字典，但 `*args` 和 `**kwargs` 的打包机制必然涉及这两种类型）
- **后续可练习方向**：建议方向 - 在 Python 交互式环境中分别创建一个元组和一个字典，使用 `type()` 验证其数据类型，并尝试用 `for` 循环遍历两者，体会可迭代对象的特性。

## 二、语法结构与关键字

### 5. `*args` 可变参数语法

- **类型**：Python 函数参数语法
- **在本 P 中的作用**：核心教学内容之一。用星号 `*` 加参数名（如 `*args`）声明，可接收任意数量的位置参数，函数内部自动打包成元组。用于解决参数数量不确定的场景（如求和函数）。
- **来源**：字幕直接提到
- **后续可练习方向**：建议方向 - 自行设计一个计算平均值的函数，接收任意数量的数字参数，利用 `*args` 实现。

### 6. `**kwargs` 关键字可变参数语法

- **类型**：Python 函数参数语法
- **在本 P 中的作用**：核心教学内容之一。用双星号 `**` 加参数名（如 `**kwargs`）声明，可接收任意数量的 `key=value` 形式参数，函数内部自动打包成字典。用于解决可选键值对配置的场景（如用户注册时的选填信息）。
- **来源**：字幕直接提到
- **后续可练习方向**：建议方向 - 设计一个配置函数，接收必填的服务器地址参数，以及任意数量的可选配置项（以 `key=value` 形式传入），函数内部遍历并打印所有配置。

## 四、相关概念与前置知识

### 9. 形参与实参

- **类型**：编程概念
- **在本 P 中的作用**：本 P 最基础的两个核心概念。**形参**是函数定义时的占位符变量，`**kwargs` 关键字可变参数语法。**实参**是函数调用时传入的具体值。传参的本质是把实参的值赋给形参。
- **来源**：字幕直接提到
- **后续可练习方向**：建议方向 - 尝试用调试模式或 `print()` 语句跟踪参数传递的过程，加深对"形参=变量名"、"实参=具体值"这一对应关系的理解。

### 10. 位置参数与默认参数

- **类型**：编程概念
- **在本 P 中的作用**：两种基础的参数类型。位置参数要求实参的顺序和个数与形参完全一致；默认参数允许形参指定默认值，调用时可省略或覆盖。
- **来源**：字幕直接提到
- **后续可练习方向**：建议方向 - 设计一个带默认参数的函数，故意将默认参数放在位置参数前面，观察 Python 报出的 `SyntaxError`，体会语法规则背后的合理性。
