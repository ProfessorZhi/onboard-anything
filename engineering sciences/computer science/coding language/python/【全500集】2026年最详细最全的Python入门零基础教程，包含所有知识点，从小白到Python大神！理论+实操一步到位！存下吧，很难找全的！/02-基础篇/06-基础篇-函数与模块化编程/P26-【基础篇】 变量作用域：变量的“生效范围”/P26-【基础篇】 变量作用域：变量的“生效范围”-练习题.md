**题目 7**  
以下代码的输出是什么？请逐行解释变量的可见范围。

```python
name = "Alice"

def greet():
    name = "Bob"
    print(f"函数内: {name}")

def greet_global():
    global name
    name = "Charlie"
    print(f"函数内: {name}")

greet()
greet_global()
print(f"函数外: {name}")
```

## 参考答案概览

| 题目 | 核心答案 |
|------|----------|
| 1 | B（函数内部赋值创建局部变量，全局变量不受影响） |
| 2 | `x` 是 outer 函数内的局部变量 |
| 3 | 解释器将未声明的赋值视为创建新局部变量 |
| 4 | 需要在修改前添加 `global count` |
| 5 | 必须放在函数内部第一行（任何赋值之前） |
| 6 | 写出包含 `global total` 的函数 |
| 7 | Bob / Charlie / Charlie（第三个是 Charlie） |
| 8 | 可维护性：命名冲突、数据流难追踪；调试：副作用难定位 |
| 9 | 局部变量遮蔽全局变量，两者独立存在 |
| 10 | 形参在函数调用时创建，调用结束时销毁，作用域仅限函数内部 |
