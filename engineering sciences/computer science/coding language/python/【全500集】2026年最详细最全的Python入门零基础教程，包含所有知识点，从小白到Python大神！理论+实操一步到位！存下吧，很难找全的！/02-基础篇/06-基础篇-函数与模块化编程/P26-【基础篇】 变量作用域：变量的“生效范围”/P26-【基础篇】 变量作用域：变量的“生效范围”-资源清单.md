## 六、延伸学习资源（建议方向）

| 资源名称 | 类型 | 关联性说明 | 来源 | 后续可练习方向 |
|----------|------|-------------|------|----------------|
| LEGB 规则 | 作用域规则 | Python 变量查找遵循 Local → Enclosing → Global → Built-in 的顺序，理解这一点是掌握作用域的关键 | 基于上下文推断 | 建议方向：编写嵌套函数验证 enclosing 作用域；理解闭包的实现原理 |
| `nonlocal` 关键字 | Python 语法 | 当需要修改嵌套函数外层（非全局）的变量时使用，是 global 的扩展 | 基于上下文推断 | 建议方向：对比 global、nonlocal 的使用场景；编写闭包函数 |
| 闭包（Closure） | 编程概念 | 利用函数内部嵌套和作用域规则实现的特殊函数对象，局部变量在其中可被"记住" | 基于上下文推断 | 建议方向：编写闭包函数；理解闭包与类之间的转换关系 |
| 装饰器（Decorator） | 编程模式 | 大量使用闭包和作用域知识的高级用法，理解变量作用域是学习装饰器的基础 | 基于上下文推断 | 建议方向：阅读装饰器源码；编写自己的装饰器函数 |
| 类属性与实例属性 | OOP 概念 | 类属性类似于全局变量，实例属性类似于局部变量，作用域概念在面向对象编程中的延伸 | 基于上下文推断 | 建议方向：对比类属性与实例属性的访问与修改规则 |
