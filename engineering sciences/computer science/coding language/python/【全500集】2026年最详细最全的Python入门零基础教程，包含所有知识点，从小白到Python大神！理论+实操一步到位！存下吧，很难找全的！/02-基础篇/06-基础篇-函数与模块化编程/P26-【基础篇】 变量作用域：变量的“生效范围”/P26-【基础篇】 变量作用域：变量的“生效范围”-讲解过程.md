# P26 讲解过程 - 【基础篇】 变量作用域：变量的“生效范围”
## 本P定位
本P承接函数定义与参数的内容，聚焦变量作用域这一核心概念。通过对比局部变量与全局变量的定义位置、生效范围、使用限制，帮助学习者理解"变量在哪里定义，就在哪里生效"的基本原则，并补充了`global`关键字在修改全局变量时的作用。

## 讲解过程

### 一、引入概念：变量的作用域
- 从时间戳 `[2.8-11.9]` 开始，直接点题——本节讲解"变量的作用域"，即变量的生效范围
- 明确核心分类：**局部变量**和**全局变量**
- 强调意义：搞清楚这个概念能够避免很多因为变量乱用所导致的错误

### 二、局部变量的定义与特性
- **定义方式**：`在函数内部定义的变量`，包括函数的形式参数（形参）也属于局部变量
- **核心特性**：作用域只在函数内部，函数执行完毕后就失效，外部无法访问
- **图示说明**：`[40.2-46.2]` 提到教材中有一张图示，展示在"文件二"的 `fun2` 函数里定义的局部变量只能在 `fun2` 内部使用，其他函数（如 `fun2` 以外的函数）以及文件外部都无法访问

### 三、局部变量演示：税率计算函数
- **函数设计**：`[70.0-84.8]` 定义 `calculate_tax()` 函数，接收收入作为参数
- **变量定义**：`[94.0-102.0]` 在函数内部定义 `tax_rate = 0.2`（税率），这就是局部变量
- **计算逻辑**：`[122.2-134.2]` 用 `income * tax_rate` 计算应缴税额，通过 `return` 返回结果
- **调用验证**：`[138.2-171.7]` 调用 `calculate_tax(5000)`，接收返回值并打印，得到 1000（验证计算：5000 × 0.2 = 1000）
- **关键验证点**：`[182.3-202.7]` 在函数外部尝试访问 `tax_rate`，触发 `NameError`（名称错误），证明局部变量在函数外部无法使用

### 四、全局变量的定义与特性
- **定义方式**：在函数的**外部**定义的变量
- **核心特性**：作用域覆盖整个程序，函数内部可以访问（除非被局部变量覆盖）
- **图示说明**：`[219.7-232.3]` 提到教材中的图示，展示在"文件一"定义的全局变量可以被文件一中的所有函数使用，也可以被文件二、文件三使用

### 五、全局变量演示：圆面积计算函数
- **变量定义**：`[234.0-247.9]` 在函数外部定义 `π = 3.14`，这就是全局变量
- **函数设计**：`[252.8-266.4]` 定义 `calculate_area()` 函数，接收半径作为参数
- **计算逻辑**：`[268.6-274.2]` 使用公式 `π × r²` 计算圆面积
- **调用验证**：`[281.6-302.0]` 调用 `calculate_area(2)`，打印返回值为 12.56（验证计算：3.14 × 2² = 12.56），证明函数内部可以使用全局变量 `π`
- **外部访问验证**：`[315.3-323.1]` 在函数外部打印 `π`，同样可以正常访问，说明全局变量在整个程序运行过程中都可用

### 六、关键点：函数内部修改全局变量
- **核心规则**：`[332.6-337.2]` 函数内部默认**不能**修改全局变量的值
- **解决方案**：使用 `global` 声明

#### 演示一：不使用 `global` 的情况
- **变量定义**：`[349.5-353.5]` 定义全局变量 `discount = 0.1`（折扣）
- **函数设计**：`[357.4-370.3]` 定义 `update_discount()` 函数，在函数内部直接写 `discount = 0.5`
- **验证过程**：`[372.4-394.3]` 调用函数后，函数内部打印 `discount` 显示为 0.5；但函数外部打印 `discount` 仍然为 0.1
- **原理解释**：`[403.3-411.5]` 函数内部的 `discount = 0.5` 实际上创建了一个**新的局部变量**，覆盖了同名的全局变量。当函数调用结束后，这个局部变量就失效了，所以全局变量 `discount` 的值并未真正被修改

#### 演示二：使用 `global` 的情况
- **关键代码**：`[419.9-427.1]` 在函数内部**第一行**写 `global discount`，作用是将 `discount` 声明为全局变量
- **效果验证**：`[427.1-439.2]` 添加 `global discount` 后，函数内部对 `discount` 的修改就会真正作用于全局变量，函数外部访问时值已变为 0.5

### 七、核心知识点总结
`[472.1-557.3]` 回顾本节及前几节函数相关内容的四大核心知识点：

1. **函数的核心作用**：封装重复逻辑，实现定义一次、多次调用，提高代码复用性和可维护性
2. **函数三大核心要素**：
   - **参数**：位置参数、默认参数、`*args` 可变参数、`**kwargs` 关键字可变参数
   - **返回值**：`return` 指定，可返回单个值、多个值，也可以无返回值（默认返回 `None`）
3. **变量作用域**：
   - 局部变量：在函数内部定义，只在函数内生效
   - 全局变量：全局生效，整个程序运行期间可用
   - 修改全局变量需使用 `global` 声明
4. **关键注意点**：
   - 必须先定义函数，后调用函数
   - 函数不调用则不执行
   - `return` 会终止函数运行，后续代码不再执行

## 本P结论

1. **局部变量**：在函数内部定义（包括形参），作用域仅限函数内部，函数执行完毕后失效，外部无法访问
2. **全局变量**：在函数外部定义，作用域覆盖整个程序，函数内部可以直接访问（但默认不能修改）
3. **`global` 关键字**：在函数内部使用 `global 变量名` 可将变量声明为全局变量，从而在函数内部修改全局变量的值
4. **核心原则**：变量在哪里定义，就在哪里生效——定义在函数内就是局部变量，定义在函数外就是全局变量
