# P25 资源清单 - 【基础篇】函数返回值：工具的“输出结果”
## 概述

本P以 Python 内置语法为核心，所有演示代码均依赖 Python 标准环境，未使用任何第三方资源、在线平台或外部工具。以下资源基于课程内容直接提及及知识点关联推断列出。

---

## 资源列表

**None**

- 类型：Python 内置单例对象
- 在本P中的作用：函数无 return 或 return 空时隐式返回 None；作为函数无返回值时的默认值
- 来源：字幕直接提到（`result = add(1, 2)` 后 result 为 None）
- 建议方向：对比 None 与 0、False、空字符串 "" 的区别，理解 None 的语义用途

---

**tuple（元组）**

- 类型：Python 内置数据结构
- 在本P中的作用：Python 函数返回多个值时，实际返回的是 tuple；用于打包多个数据作为单一返回值
- 来源：字幕直接提到（`return a + b, a * b` 实际返回元组 `(3, 2)`）
- 建议方向：结合 tuple 的不可变特性，理解为何 Python 选择 tuple 而非 list 作为多返回值容器

---

**return 语句**

- 类型：Python 语法关键字 / 控制流语句
- 在本P中的作用：指定函数的返回值；立即终止函数执行
- 来源：字幕直接提到（本P核心语法）
- 建议方向：在复杂函数中练习使用 return 提前退出，掌握 return 与 print 的本质区别

---

**print() 函数**

- 类型：Python 内置函数
- 在本P中的作用：作为 return 的对比项，演示“打印输出”与“返回值”的本质区别；print 仅终端输出，无返回值
- 来源：字幕直接提到（对比 `print(a + b)` 与 `return a + b`）
- 建议方向：完成一个同时包含 print 调试和 return 返回值的函数，体会两者的不同职责

---

**Python 官方文档（The Python Tutorial - Defining Functions）**

- 类型：官方文档 / 在线教程
- 在本P中的作用：课程内容的权威参考来源，return 语句的完整语法说明
- 来源：基于上下文推断（课程以 Python 官方文档体系为基准）
- 建议方向：阅读官方文档中关于函数定义的章节，对照本P知识点查漏补缺
