# P25 讲解过程 - 【基础篇】函数返回值：工具的“输出结果”
## 本P定位

本P承接上一P的参数（输入）概念，系统讲解 Python 函数的返回值（return），即函数的"输出结果"。通过生活比喻引出概念，再通过代码演示对比 print 和 return 的区别，最后讲解返回值的几种情况及接收方式。

## 讲解过程

### 一、引出返回值概念

老师先回顾"参数就是输入"，然后自然过渡到"输出——返回值"。明确指出：返回值是函数处理完数据后反馈给调用者的结果，使用 `return` 关键字来指定。

### 二、生活比喻：猴子买可乐

老师用一个生动的例子解释什么是返回值：

- **参数**：猴子妈妈让小猴子买**三瓶**可乐。这里的"三瓶"就是参数（输入），明确指定了数量，不能多也不能少。
- **返回值**：小猴子买完可乐后，必须把三瓶可乐**带回来**。这带回来的三瓶可乐就是返回值。如果什么都没带回来，就无法证明任务完成了。带回来之后，猴子妈妈会说"非常不错，给你奖励五块钱"，这奖励是基于返回值做出的后续处理。

老师通过这个比喻帮助学生理解：返回值是函数执行后给调用者的"凭证"和"结果"。

### 三、返回值的两大作用

#### 作用一：给调用者返回结果，方便后续处理

老师先演示一个**没有返回值**的函数（只有 print）：

```python
def add(a, b):
    print(a + b)

add(1, 2)  # 打印 3
add(3, 4)  # 打印 7
result = add(1, 2)
print(result + 4)  # 报错！因为 result 是 None
```

老师强调：如果只是 print，函数只是执行了打印操作，并没有返回一个结果可用于后续计算。

然后老师演示**加上 return** 的写法：

```python
def add(a, b):
    return a + b

result = add(1, 2) + add(3, 4)
print(result)  # 7
```

老师逐步拆解执行流程：

1. `add(1, 2)` 执行后得到返回值 3
2. `add(3, 4)` 执行后得到返回值 4
3. 3 + 4 = 7，赋值给 result

老师还演示了另一种写法，直接用变量接收返回值：

```python
result = add(add(1, 2), add(3, 4))
```

#### 作用二：终止函数的运行

老师强调这个作用"很容易被忽视"：

```python
def test():
    print("_______")
    print("123")
    return
    print("456")

test()
```

运行结果显示：

- "_______" 和 "123" 会被打印
- "456" **不会被打印**

原因：`return` 会立即停止函数执行，后面的代码不再运行。老师补充说明：`return` 的本质是"将值返回给函数调用并结束函数执行"。

### 四、返回值的三种情况

#### 情况一：无返回值（隐式返回 None）

老师以之前讲过的 greet 函数为例：

```python
def greet():
    print("你好呀")

result = greet()
print(result)
print(type(result))
```

运行结果：

- 打印"你好呀"
- result 的值是 None
- result 的类型是 NoneType

老师补充：如果 `return` 后面什么都不写，效果一样，也是返回 None。

#### 情况二：返回单个值

```python
def greet():
    return "冰冰"

result = greet()
print(result)
print(type(result))
```

运行结果：

- 输出"你好呀"（函数内部 print 执行）
- result 的值是"冰冰"
- result 的类型是字符串

老师详细拆解执行流程：

1. `return "冰冰"` 将返回值赋给函数调用 `greet()`
2. `greet()` 的值就是"冰冰"
3. 将"冰冰"赋给 result

老师还演示返回数字：

```python
def greet():
    return 123
```

此时 result 就是整数类型。

#### 情况三：返回多个值（实际返回元组）

```python
def calc(a, b):
    return a + b, a * b

result = calc(1, 2)
print(result)  # (3, 2)
```

老师解释：Python 实际上是返回了一个**元组**来接收多个值。元组的特点是"不可变"，适合作为返回值容器。

适用场景：返回坐标（x, y）、返回和与积、返回统计数据等。

### 五、返回值的两种接收方式

#### 方式一：用单个变量接收（得到元组）

```python
result = calc(1, 2)
print(result)  # (3, 2)
```

#### 方式二：解包接收（推荐）

```python
sum_ab, mul_ab = calc(1, 2)
print(sum_ab)  # 3
print(mul_ab)  # 2
```

老师演示：元组有多少个元素，就定义多少个变量来接收，每个变量对应一个元素的值。老师明确推荐这种解包方式，因为"使用起来更加方便"。

本P内容全部基于 Python 内置语法，未使用任何外部资源、程序、网站、仓库或工具。所有演示均为老师现场敲写的代码。

## 本P结论

1. **返回值定义**：函数通过 `return` 关键字将处理结果返回给调用者，是函数的"输出"。

2. **返回值的两大作用**：
   - 给调用者返回结果，用于后续处理（计算、赋值等）
   - 立即终止函数执行，后面的代码不再运行

3. **返回值的三种情况**：
   - 无 return 或 return 空 → 隐式返回 None
   - return 单个值 → 返回该值（类型取决于返回的内容）
   - return 多个值 → 实际返回一个元组

4. **接收方式**：
   - 单变量接收：得到一个元组
   - 解包接收：定义多个变量分别接收，推荐使用
