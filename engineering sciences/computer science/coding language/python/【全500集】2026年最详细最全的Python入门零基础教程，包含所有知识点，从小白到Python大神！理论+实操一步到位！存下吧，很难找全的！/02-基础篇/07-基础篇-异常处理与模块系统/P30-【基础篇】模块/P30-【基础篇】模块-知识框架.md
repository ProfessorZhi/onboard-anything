# P30 知识框架 - 【基础篇】模块
## 本P位置

- **课程阶段**：基础篇 → 模块（Python 语法 → 模块化编程的过渡节点）
- **前置依赖**：P1-P29（变量、数据类型、循环、条件判断、函数等基础语法已掌握）
- **后置衔接**：P31 包（用文件夹组织多个模块）
- **核心角色**：从「写零散代码」升级到「构建可复用项目」的关键转折点

---

## 核心问题

1. 代码量大了怎么组织 → 模块收纳
2. 如何复用自己写的 / 别人写的代码 → 模块导入
3. 同名冲突怎么办 → import 方式 vs from...import * 区别
4. 模块自测时不让测试代码污染导入者 → `if __name__ == "__main__"`

---

## 知识骨架

```
模块
├── 定义：.py 文件（封装变量/函数/类）
├── 类型
│   ├── 内置模块（Python 自带，无需安装）
│   │   └── random、time
│   ├── 第三方模块（pip 安装后才可导入）
│   │   └── requests、numpy、pandas、matplotlib
│   └── 自定义模块（自己写的 .py 文件）
└── 导入方式
    ├── import 模块名 [as 别名]          ← 推荐：无冲突、层级清晰
    ├── from 模块名 import 功能名         ← 简洁：只取部分
    └── from 模块名 import *              ← ⚠️ 慎用：易覆盖
```

---

## 关系与依赖

```
自定义模块 ← 创建（自己写 .py 文件）
         ← 导入（被其他 .py 文件用 import 使用）

第三方模块 ← 安装（pip）
         ← 导入（import）
         ← pip 可配镜像源加速（清华、豆瓣）

内置模块 ← 直接导入（import）
        ← 无需安装

__name__ 变量
         ← 直接运行 → "__main__"
         ← 被导入 → "模块名"
```

---

## 输入 / 输出 / 前置 / 支撑

| 角色 | 内容 |
| --- | --- |
| **输入** | 已有 .py 文件（变量、函数、类）；需要安装的第三方模块名 |
| **输出** | 可被其他文件复用的模块；通过 `模块名.功能` 或 `功能名` 调用 |
| **前置** | Python 基础语法（函数定义、变量、print） |
| **支撑工具** | pip（包管理）、命令行/终端（执行安装和运行） |
| **外部依赖链** | requests → pip install → 可爬取网页；pandas → pip install → 数据分析 |

---

## 易错点

- **模块名冲突**：自定义模块不能与内置模块（random、time）同名，否则覆盖原功能
- **命名规则**：只能字母/数字/下划线，不能以数字开头，不能用关键字
- **`from import *` 覆盖**：当前文件已有同名变量/函数时，后导入的会覆盖先定义的
- **`__name__` 误解**：直接在模块里写测试代码 → 被其他文件导入时也会执行（应包裹在 `if __name__ == "__main__"` 内）
- **安装超时**：pip 从国外源下载慢 → 换用国内镜像（清华：https://pypi.tuna.tsinghua.edu.cn/simple）

## 外部资源与对象

| 对象 | 类型 | 关系 |
| --- | --- | --- |
| **random** | 内置模块 | 无需安装，`random.randint(a, b)` 生成闭区间随机整数 |
| **time** | 内置模块 | 内置，处理时间相关功能 |
| **requests** | 第三方模块 | pip install requests，用于 HTTP 请求（爬虫/调用 API） |
| **numpy** | 第三方模块 | pip install numpy，数值计算 |
| **pandas** | 第三方模块 | pip install pandas，数据分析 |
| **matplotlib** | 第三方模块 | pip install matplotlib，绑定数据可视化 |
| **pip** | 工具 | Python 包管理器，`pip install 模块名` 安装第三方模块 |
| **清华镜像** | 镜像源 | 加速 pip，`pip install xxx -i https://pypi.tuna.tsinghua.edu.cn/simple` |
| **豆瓣镜像** | 镜像源 | 同上，加速 pip 安装 |
| **mytools** | 自定义模块（演示用） | 包含 `model_name` 变量、`add()` 函数、`greet()` 函数 |
| **百度 URL** | 测试目标 | `https://www.baidu.com`，演示 requests 爬取与编码处理 |
