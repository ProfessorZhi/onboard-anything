## 一、结构识别

**题目**：判断以下场景适合使用哪种模块类型，以及最合适的导入方式。

| 场景 | 模块类型 | 推荐导入方式 |
| --- | --- | --- |
| 生成 1-10 之间的随机整数 | ① _____ | ② _____ |
| 发送 HTTP 请求获取网页内容 | ③ _____ | ④ _____ |
| 记录程序运行耗时 | ⑤ _____ | ⑥ _____ |

**选项**：
- A. 内置模块
- B. 第三方模块
- C. 自定义模块
- D. `import 模块名`
- E. `from 模块名 import 功能名`

## 二、机制理解

**题目**：阅读以下代码，判断运行结果。

```python
# utils.py
def say_hello():
    print("Hello from utils!")

print("模块自身被加载")

if __name__ == "__main__":
    print("直接运行本文件")
else:
    print("被其他文件导入")
```

**问题**：分别说明以下两种运行方式，终端会输出哪些内容：
1. 在命令行执行 `python utils.py`
2. 在另一个文件执行 `import utils`

```python
# config.py
api_key = "ABC123"
timeout = 30

# main.py
from config import *

api_key = "USER_KEY"      # 这里会发生什么？
```

**问题**：

## 五、最小替代练习

**背景**：课程使用 `requests` 模块爬取百度网页。为避免安装第三方依赖，以下练习全部使用 Python 内置模块完成。

**任务**：创建文件 `demo.py`，实现以下功能：

1. 导入 `random` 模块，生成 1-100 之间的随机整数
2. 导入 `time` 模块，获取当前时间戳并格式化输出
3. 创建自定义模块 `mymath.py`，包含 `multiply(a, b)` 函数；在 `demo.py` 中导入并调用该函数
4. 在 `mymath.py` 底部添加 `if __name__ == "__main__":` 语句，编写自测代码（当直接运行 `mymath.py` 时输出 "模块自测通过"）

**验收标准**：
- `python demo.py` 能正常运行并输出结果
- `python mymath.py` 只输出自测信息，不执行模块级代码
- 文件命名规范，无模块名冲突
