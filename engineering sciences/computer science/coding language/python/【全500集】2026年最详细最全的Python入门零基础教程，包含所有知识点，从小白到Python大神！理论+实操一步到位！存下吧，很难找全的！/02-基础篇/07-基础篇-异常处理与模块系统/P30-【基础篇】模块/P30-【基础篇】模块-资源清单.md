## 内置模块

**random**

- 类型：内置模块
- 在本 P 中的作用：演示内置模块的用法，通过 `random.randint(a, b)` 生成指定区间内的随机整数（包前包后），用于后续构建猜数字游戏等案例。
- 来源：字幕直接提到
- 后续可练习方向：使用 random 模块实现一个完整的猜数字游戏；结合循环和条件判断设计更复杂的随机点名或抽奖程序。

**time**

- 类型：内置模块
- 在本 P 中的作用：列举内置模块时提到，用于处理时间相关的功能。
- 来源：字幕直接提到
- 后续可练习方向：使用 time 模块的 `sleep()` 函数实现程序延时；结合其他模块练习时间戳转换或格式化输出。

**max**

- 类型：内置模块
- 在本 P 中的作用：列举内置模块时提到，用于数学计算场景。
- 来源：字幕直接提到
- 后续可练习方向：在实际编程中替代手动比较，练习使用内置函数处理集合数据的极值问题。
