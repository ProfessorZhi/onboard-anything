# P03 讲解过程 - 【基础篇】安装PyCharm和AI插件
## 本P定位

本P属于【基础篇】第三个小节，承接Python解释器安装之后，讲解Python专用代码编辑器PyCharm的安装、基础配置及两个实用插件的使用，是后续写代码的实操准备环节。

---

## 讲解过程

### 一、PyCharm是什么，为什么用它

老师先介绍PyCharm的定位：专门为Python开发设计的代码编辑器。用写作文需要Word来类比，写Python代码用PyCharm更方便。列举了PyCharm的三个核心优势：自动补全代码、写错及时提示、一键运行，全程可视化操作，不需要死记复杂命令，对新手友好。

### 二、强调必须先装Python解释器

老师重点解释了PyCharm和Python解释器的关系。用笔和动力源做类比：PyCharm只是写代码的工具，相当于笔；Python解释器是执行代码的引擎，相当于让纸上想法真正动起来的动力源。没有装解释器，PyCharm里写的代码无法运行。提醒两个软件的先后顺序千万不能搞反。

### 三、下载PyCharm安装包

老师给出官网下载地址：WWW点j t brains，点download按钮选择对应电脑版本。由于服务器在国外，访问和下载速度较慢。老师也提到可以使用课程提供的安装包省去查找时间。

### 四、安装步骤演示

老师演示了完整的安装流程：

**选择安装路径**：建议把PyCharm和之前装的Python解释器放在同一个文件夹，比如D盘新建一个PyCharm文件夹，路径设为D盘PyCharm，方便后续找文件和管理软件。

**勾选安装选项**：重点勾选两个选项——创建桌面快捷方式方便双击打开PyCharm，以及更新上下文菜单让双击.py文件时自动用PyCharm打开，不需要手动选打开方式。老师还提到可以勾选更新PATH变量添加到环境变量方便后续调用。

**等待安装完成**：直接点击下一步直到安装，不需要做任何操作，耐心等进度条走完即可。

### 五、新手容易踩的坑

老师总结了两个常见问题：

**坑一：没装解释器直接装PyCharm**。会导致写好的代码根本无法运行，还得回头重新装解释器，耽误时间。

**坑二：PyCharm版本和Python版本不匹配**。比如PyCharm选了最新版而Python选了很老的3.5、3.6，或者反过来PyCharm很老而Python很新，可能出现兼容性问题。

老师再次强调核心要点：先装解释器，再装编辑器，最后做好关联。

### 六、过渡到PyCharm基本使用

老师预告接下来学习PyCharm编辑器的基本使用，指出核心操作围绕三件事展开：建项目、写代码、调设置。新手不用学太多，掌握新建项目、配置解释器、运行代码三个核心技能就够了，再根据需求调整界面和安装插件。

### 七、讲解两个核心概念

**概念一：什么是项目**。把项目理解成一个专属文件夹，专门用来放某一项编程任务里的所有代码文件和相关资料。比如刚开始学Python基础可以建一个叫Python入门的项目，后续练手的代码都放这个项目里，方便找文件和管理代码。

**概念二：为什么要配置解释器**。重复强调PyCharm只是写代码的工具，想让代码跑起来就得让PyCharm知道解释器在哪里。如果没有配置解释器或配置错了路径，代码就无法运行，还会弹出“无有效解释器”的提示。

### 八、新建项目演示

老师演示了第一次使用PyCharm新建项目的步骤：

**点击new project**：在欢迎界面找到new project按钮点击。

**选择项目路径**：比如D盘Python study，后面写上项目名称Python base杠一。

**配置解释器**：在interpreter type选择custom environment，点击select existing，点击文件夹图标找到之前安装的Python解释器，即D盘tools文件夹Python目录下的Python.exe，点击配置好。

**创建Python文件**：项目创建好后，在项目下新建Python代码文件new一个Python file，命名为py01，就可以开始写代码了。

### 九、关闭和打开已有项目

老师演示了如何操作：

**关闭当前项目**：点击顶部菜单栏的file，选择close project，会回到PyCharm欢迎界面，然后可以新建或打开其他项目。

**打开已有项目**：在欢迎界面点击open，找到之前建的项目文件夹，比如D盘Python study里的推箱子游戏或飞机大战项目，点击select folder，然后选择trust project信任项目，就能看到项目里存放的代码文件继续编辑。

### 十、基础设置调整

老师演示了调整字体和主题的方法：

**调整界面字体（菜单、标题等）**：点击顶部file，选择settings，在左侧找到Appearance and Behavior，点击Appearance，勾选use custom font使用自定义字体，在下拉框选择字体如consoles，size选择字号如16或18，根据个人习惯调整，调整好后点击apply应用，再点击OK。

**调整代码字体**：在settings左侧找到Editor，点击Font，选择字体如consoles（编程专用字体，看起来清晰），size选择字号如16或18，Line height设为1.2让文字之间不挤在一起，调整好后点击apply再点击OK，回到编辑区就能看到代码字体的变化。

**切换主题**：在Appearance and Behavior的Appearance里找到theme下拉框，可以选择Light浅色主题看着不刺眼，或者Dark深色主题长时间写代码能保护视力，选好后直接点击OK不需要点apply。

### 十一、配置Python解释器（解决运行报错）

老师演示了当运行代码弹出please select a valid Python interpreter提示时如何解决：

**进入设置界面**：点击file，选择settings。

**找到解释器配置**：在左侧找到project选项，下面点击Python interpreter。

**添加解释器**：点击右上角的加号添加解释器，选择第一个add local interpreter本地解释器，勾选select existing，点击已显示的Python路径或点击文件夹图标找到D盘toolsPython目录下的Python.exe，点击确定。

**确认配置**：依次点击OK完成配置，配置好后再运行代码就能正常运行了。

老师特别说明：如果代码运行报错是因为游戏用到了第三方库如pygame，需要下载安装第三方库才能使用，这不是解释器配置的问题，代码报错和代码根本无法运行是两个概念。

### 十二、安装Translation翻译插件

老师演示了安装步骤：

**进入插件安装界面**：点击file，选择settings，左侧找到Plugins，点击顶部的marketplace，在搜索框搜索translation。

**安装插件**：找到后点击install安装，弹出提示选择accept同意，等待安装完成。

**重启IDE**：安装好后会出现restart IDE提示，点击重启PyCharm让它生效。

**使用方法**：选中代码里不懂的英文单词或注释，右键点击translate选择翻译，就能看到对应的中文意思。也可以用于变量命名，右键选择replace with translation可以把中文替换成英文。

### 十三、安装通义灵码AI插件

老师介绍了通义灵码的功能：由阿里云提供的智能编码辅助工具，可以代码自动生成、智能问答、多文件修改、编程智能体等能力，提升开发效率。新手写代码没思路或遇到报错不懂怎么改时可以求助它。

**安装方式一： marketplace搜索下载**：在settings的Plugins里marketplace搜索通义灵码，点击install安装。要确保网络良好，可能下载较慢。

**安装方式二：官网下载zip包本地导入**：去通义灵码官网下载zip安装包，在settings选择install plugin from disk，从本地导入zip文件这种方式可能更快。

老师演示了官网下载后本地导入的方式，安装好后点击restart IDE重启。

**登录使用**：重启后右侧会出现通义灵码logo图标，点击登录。可以选择中国站账号登录，用支付宝或淘宝扫码，很安全不收费只是获取登录状态。登录后就可以使用插件。

**使用方法一：代码自动续写**：写代码时插件会自动给出提示，按Tab键一键采纳。也可以用自然语言描述功能需求让它生成代码，比如写九九乘法表，它会生成代码按Tab采纳。如果生成很多行代码，按Ctrl加下箭头逐行采纳。

**使用方法二：代码解释**：看不懂某段代码时，可以选中代码让它解释一下，它会用对话方式给出解释。代码运行报错也可以问它原因。

### 十四、常见问题总结

老师最后总结了三个常见问题：

**问题一：项目路径有中文或空格**。会导致部分功能无法正常使用甚至代码运行报错，全程要用英文路径，不要有中文和特殊字符。

**问题二：解释器路径选错**。配置解释器时一定要找到之前安装在Python文件夹里面的Python.exe，路径选错代码就跑不起来。

**问题三：安装插件后不重启PyCharm**。插件无法生效，安装完后尽量重启一下让它加载过来。

老师总结：PyCharm基本使用抓住三个关键操作就够了——创建项目、配置解释器、运行代码。基础设置和插件都是为了用得更舒服效率更高，按需求调整即可，不用一开始就想掌握所有功能。

## 本P结论

1. **安装顺序不能乱**：必须先安装Python解释器，再安装PyCharm编辑器，PyCharm只是写代码的工具，解释器才是执行代码的引擎。

2. **安装时关键选项要勾选**：创建桌面快捷方式、更新上下文菜单关联.py文件、更新PATH变量。

3. **安装路径注意事项**：PyCharm和Python解释器建议放同一文件夹，全程路径不要出现中文、空格等特殊字符。

4. **新建项目核心步骤**：选对路径、配好解释器。配置解释器时要找到Python.exe的路径。

5. **插件安装后必须重启**：Translation插件和通义灵码安装后都要点击restart IDE重启才能生效。

6. **通义灵码使用需要登录**：用支付宝或淘宝扫码登录，用于代码自动续写、自然语言生成代码、代码解释等功能。

7. **新手三个必备技能**：新建项目、配置解释器、运行代码，掌握这三个就能开始写代码，其他设置和插件按需调整即可。
