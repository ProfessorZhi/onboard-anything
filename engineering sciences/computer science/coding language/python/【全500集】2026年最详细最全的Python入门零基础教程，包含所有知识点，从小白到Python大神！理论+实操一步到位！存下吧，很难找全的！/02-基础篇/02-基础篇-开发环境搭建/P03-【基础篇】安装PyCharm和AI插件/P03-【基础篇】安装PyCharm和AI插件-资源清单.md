## 路径与版本注意事项

> 以下为本P中老师反复强调的实操规范，对资源使用有直接影响，单独列出备查。

- **路径规范**：PyCharm 安装目录、项目目录、解释器路径，全程使用纯英文、无空格、无特殊字符。违反此规范可能导致部分功能失效或代码运行报错。
- **版本匹配**：PyCharm 版本与 Python 解释器版本差异过大（如一方过新一方过旧）可能出现兼容问题，建议选择接近的稳定版本组合。
- **安装顺序**：Python 解释器 → PyCharm → 插件，顺序不可颠倒。
- **插件生效**：Translation 和通义灵码安装完成后，必须点击 `Restart IDE` 重启 PyCharm，插件才会加载生效。
