## 知识骨架

### 层级一：Python本质认知

**Python定义**：人与计算机之间的"翻译官"，一种高级编程语言
**核心定位**：新手入门最优解 + AI领域第一语言
**存在价值**：自然语言无法直接驱动计算机，必须通过编程语言转译为二进制指令

### 层级二：Python四大优势

| 优势 | 核心表现 | 关联工具/库 |
|------|----------|-------------|
| 语法简单 | print(123)、type() 等近乎自然语言 | - |
| 功能全面 | Web、数据分析、游戏、AI | PyTorch、TensorFlow、NumPy、pandas、NLTK |
| 资源丰富 | 社区活跃，问题可搜索解决 | Stack Overflow、稀土掘金 |
| 配置简单 | pip install xxx 即装即用 | pip |

### 层级三：编程语言分类

```
人类自然语言
      ↓  [编程语言：充当翻译]
二进制机器语言（0/1）
```

**编译型语言**：整段翻译 → 生成可执行文件 → 执行（C、C++、Java）
- 特点：运行快、跨平台差
**解释型语言**：逐行翻译 → 边翻边执行 → 每次运行需重新翻译（Python、JavaScript）
- 特点：跨平台好、写完即运行

Python = 解释型语言

### 层级四：Python解释器安装流程

**安装包来源**：python.org（官网）或课程提供包
**必勾选项**：
1. Add Python.exe to PATH（环境变量，决定命令行能否调用python）
2. Use admin privileges when installing py.exe（管理员权限）

**组件建议**：全选，默认路径避免踩坑
**路径规范**：无中文、无空格、无特殊符号，推荐D:\tools\Python

**验证命令**：
```
Win+R → cmd → python → 出现版本号+>>> 即成功
```

## 关系与依赖

```
Python语言层
    ↓ 依赖
Python解释器（运行时环境）
    ↓ 依赖
pip（包管理器）→ 第三方库（PyTorch/TensorFlow等）
    ↓ 被使用
IDLE / PyCharm / 其他编辑器（编码工具）
```

**依赖链**：解释器 → pip → 外部库 → 应用场景（AI/数据分析/Web）

## 输入 / 输出 / 前置 / 支撑

| 类型 | 内容 |
|------|------|
| 输入 | .py文件（Python源代码） |
| 输出 | 程序执行结果（通过解释器转译为机器指令后由CPU执行） |
| 前置依赖 | 无（零基础可直接学习） |
| 运行时支撑 | Python解释器（必须安装） |
| 包管理支撑 | pip（随解释器安装） |
| 应用支撑 | PyTorch（训练大模型）、TensorFlow（图像识别）、NumPy/pandas（数据处理）、NLTK（自然语言处理） |

## 易错点

| 错误 | 后果 | 规避 |
|------|------|------|
| 漏勾"Add Python.exe to PATH" | cmd输入python提示"不是内部或外部命令" | 安装时务必勾选 |
| 路径含中文/空格/特殊符号 | 解释器无法启动或组件调用失败 | 路径全英文、无空格 |
| 未以管理员权限安装 | 部分系统文件写入失败，功能残缺 | 右键安装包→以管理员身份运行 |
| 找不到桌面图标 | 误以为安装失败 | 解释器无桌面图标，属正常现象 |
