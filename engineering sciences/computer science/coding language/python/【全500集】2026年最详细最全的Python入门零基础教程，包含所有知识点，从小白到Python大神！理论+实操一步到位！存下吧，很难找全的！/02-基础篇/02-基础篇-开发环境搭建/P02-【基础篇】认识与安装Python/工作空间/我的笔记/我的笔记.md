# 内置函数、标准库、第三方库 区别

## 三层结构

| 类型 | 怎么用 | 存在形式 | 例子 |
|---|---|---|---|
| 内置函数 | 直接用，不 import | C 语言写的，编译进 python.exe | `print()`, `len()`, `input()` |
| 标准库模块 | `import xxx` | Python 写的 .py 文件 | `dis`, `os`, `sys`, `json` |
| 第三方库 | `pip install` 后 `import` | 装在 site-packages | `requests`, `numpy` |

## 只要是库，都得 import 对吗？

**对。**"库"（library）= 别人写好的代码打包起来给你用。

```
内置函数  → python.exe 内核自带，零配置，直接用（不用 import）
标准库    → python 自带，`import` 才能用
第三方库  → pip 安装，`import` 才能用
```


## 怎么知道一个东西是内置还是标准库还是第三方？

```python
import something
print(something.__file__)
```

- 结果是 `None` → 内置函数（C 写的，没文件）
- 结果指向 `python312.zip` → 标准库
- 结果指向 `Lib/site-packages/...` → 第三方库

## 例子

```python
# 内置函数 - 直接用
print("hello")      # 不用 import
len([1,2,3])       # 不用 import
input()             # 不用 import

# 标准库 - 需要 import
import dis
import os
import sys
import json

# 第三方库 - 先 pip install，再 import
import requests      # 需要先 pip install requests
import numpy        # 需要先 pip install numpy
```
