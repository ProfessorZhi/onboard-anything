## 二、运行时环境

### 4. Python 解释器

| 字段 | 内容 |
|------|------|
| 名称 | Python 解释器（CPython，官网默认发行版） |
| 类型 | 运行时环境 / 解释器 |
| 在本P中的作用 | 本P核心安装对象；负责逐行读取 `.py` 文件并将其翻译为机器指令执行；是后续一切 Python 代码能够运行的基础依赖 |
| 来源 | 字幕直接提到 |
| 后续可练习方向 | 安装完成后在 cmd 中执行 `python --version` 确认版本；进入交互模式（`>>>`）输入 `print("hello")`，完成第一行代码；执行 `exit()` 退出，熟悉进出交互模式的操作 |

## 三、命令行工具与命令

### 5. cmd（命令提示符）

| 字段 | 内容 |
|------|------|
| 名称 | cmd（Windows 命令提示符） |
| 类型 | 系统命令行工具 |
| 在本P中的作用 | 用于验证 Python 是否安装成功；通过输入 `python` 命令确认解释器已加入环境变量并可正常调用 |

---

### 6. python 命令

| 字段 | 内容 |
|------|------|
| 名称 | `python`（命令行调用命令） |
| 类型 | 命令 |
| 在本P中的作用 | 在 cmd 中输入以验证解释器安装；进入 Python 交互模式（`>>>`）；是后续在命令行运行脚本的核心调用方式 |
| 来源 | 字幕直接提到 |
| 后续可练习方向 | 练习 `python --version` / `python -V` 查看版本；练习 `python 文件名.py` 运行脚本；了解 `python3` 与 `python` 在不同系统下的区别 |

---

### 7. pip install 命令

| 字段 | 内容 |
|------|------|
| 名称 | `pip install`（示例：`pip install tensorflow`） |
| 类型 | 命令 |
| 在本P中的作用 | 作为"配置简单"这一优势的核心例证；说明安装第三方库只需一行命令，无需手动编译或复杂配置 |
| 来源 | 字幕直接提到 |
| 后续可练习方向 | 练习 `pip install requests`（轻量无依赖，适合新手初次体验）；练习 `pip list` 查看已安装库；练习 `pip uninstall 库名` 卸载；了解 `pip install -i 镜像源 库名` 解决国内下载慢的问题 |

## 四、包管理工具

### 8. pip

| 字段 | 内容 |
|------|------|
| 名称 | pip（Package Installer for Python） |
| 类型 | Python 内置包管理工具 |
| 在本P中的作用 | 随解释器一同安装（Tcl/Tk 以外的独立组件）；是后续安装所有第三方库（PyTorch、TensorFlow、NumPy 等）的唯一管理入口；本P中以 `pip install tensorflow` 为例说明其便捷性 |
| 来源 | 字幕直接提到 |
| 后续可练习方向 | 验证 pip 可用性：`pip --version`；配置国内镜像（如清华源、阿里源）解决下载慢问题；了解 `requirements.txt` 的作用，尝试用 `pip freeze > requirements.txt` 导出环境依赖 |
