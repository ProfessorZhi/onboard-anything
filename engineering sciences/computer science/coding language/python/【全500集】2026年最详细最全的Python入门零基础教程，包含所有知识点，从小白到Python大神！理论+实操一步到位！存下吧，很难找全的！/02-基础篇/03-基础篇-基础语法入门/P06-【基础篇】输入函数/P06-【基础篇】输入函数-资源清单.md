# P06 资源清单 - 【基础篇】输入函数
本清单汇总本P涉及的程序、软件、函数、概念、工具等资源，共8项。

---

## 1. input()

- **名称**：input() 函数
- **类型**：Python 内置函数 / 命令
- **在本P中的作用**：核心教学对象，用于获取用户在控制台输入的数据，实现程序与用户的交互
- **来源**：字幕直接提到（讲解语法结构、基本使用、返回值类型时多次出现）
- **后续可练习方向**：创建交互式问答程序，如用户信息收集、简单调查问卷、多轮对话场景

---

## 2. int()

- **名称**：int() 函数
- **类型**：Python 内置函数 / 类型转换命令
- **在本P中的作用**：将 input 获取的字符串转换为整数类型，使数值运算成为可能
- **来源**：字幕直接提到（类型转换解决方案一节明确讲解）
- **后续可练习方向**：处理整数输入场景，如计算两数之和、判断奇偶数、整数排序等

---

## 3. float()

- **名称**：float() 函数
- **类型**：Python 内置函数 / 类型转换命令
- **在本P中的作用**：将 input 获取的字符串转换为浮点数类型，支持小数运算
- **来源**：字幕直接提到（处理成绩加分等带小数场景时讲解）
- **后续可练习方向**：处理小数输入场景，如成绩计算、身高体重数据、商品价格计算等

---

## 4. print()

- **名称**：print() 函数
- **类型**：Python 内置函数 / 输出命令
- **在本P中的作用**：与 input() 互补，用于向用户输出信息，展示程序处理结果
- **来源**：字幕直接提到（与 input 函数对比说明、组合输出演示时多次出现）
- **后续可练习方向**：结合 input 进行输入输出组合练习，设计格式化输出、动态问候语等

- **名称**：变量赋值语句
- **类型**：Python 语法 / 概念
- **在本P中的作用**：将 input 获取的数据存储到变量中保存，涉及“右先左后”的执行顺序概念
