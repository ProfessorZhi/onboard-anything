# P06 参数绑定流程图

## 这份笔记真正解决什么问题

看到一次函数调用时，不再靠猜，而是按固定流程判断：

- 每个实参最终绑定给谁
- 哪些会进 `*args`
- 哪些会进 `**kwargs`
- 为什么会报错

## 一、最短总流程

把一次调用理解成这 5 步：

1. 先看签名，分清参数区
2. 先处理位置实参
3. 再处理关键字实参
4. 再看多余参数有没有人接
5. 最后检查必填参数有没有缺失

## 二、第一步：先看签名，分清参数区

例如：

```python
def f(a, /, b, c=30, *args, d, e=50, **kwargs):
    pass
```

先不要急着看调用，先拆签名：

- `a`
  - 仅位置参数
- `b`
  - 普通参数
- `c=30`
  - 普通参数，带默认值
- `*args`
  - 收多余位置参数
- `d`
  - 仅关键字参数，必传
- `e=50`
  - 仅关键字参数，带默认值
- `**kwargs`
  - 收多余关键字参数

## 三、第二步：先处理位置实参

位置实参按顺序，从左到右绑定。

例如调用：

```python
f(1, 2, 3, 4, 5, d=6, x=7)
```

先只看前面的纯位置实参：

```python
1, 2, 3, 4, 5
```

按顺序绑定：

- `1 -> a`
- `2 -> b`
- `3 -> c`

这时前面能正常吃位置参数的普通形参已经用完了。

剩下：

- `4`
- `5`

会进入：

```python
args = (4, 5)
```

## 四、第三步：再处理关键字实参

然后再看：

```python
d=6, x=7
```

处理规则：

- 如果关键字名能匹配某个还没绑定的形参，就绑定过去
- 如果匹配不到：
  - 有 `**kwargs` 就收进去
  - 没有 `**kwargs` 就报错

所以这里：

- `d=6 -> d`
- `x=7 -> kwargs`

于是：

```python
kwargs = {'x': 7}
```

## 五、第四步：再看默认值

如果某个参数没传，但有默认值，就用默认值。

上面的例子里：

- `e` 没传
- 但它有默认值 `50`

所以：

```python
e = 50
```

## 六、第五步：最后检查有没有缺失

如果某个参数：

- 没传
- 没默认值
- 也没有别的规则帮它兜住

那就报错。

例如：

```python
def f(a, *, b):
    pass

f(1)
```

这里：

- `a = 1`
- 但 `b` 是必传仅关键字参数
- 没传，所以报错

## 七、把完整流程走一遍

看这个调用：

```python
def f(a, /, b, c=30, *args, d, e=50, **kwargs):
    pass

f(1, 2, 3, 4, 5, d=6, x=7, y=8)
```

### 第一步：拆签名

- `a`：仅位置
- `b`：普通
- `c=30`：普通，默认 30
- `*args`：收多余位置
- `d`：仅关键字，必传
- `e=50`：仅关键字，默认 50
- `**kwargs`：收多余关键字

### 第二步：处理位置实参

位置实参：

```python
1, 2, 3, 4, 5
```

绑定：

- `a = 1`
- `b = 2`
- `c = 3`
- `args = (4, 5)`

### 第三步：处理关键字实参

关键字实参：

```python
d=6, x=7, y=8
```

绑定：

- `d = 6`
- `kwargs = {'x': 7, 'y': 8}`

### 第四步：补默认值

- `e = 50`

### 第五步：检查缺失

没有缺失，调用合法。

### 最终结果

```python
a = 1
b = 2
c = 3
args = (4, 5)
d = 6
e = 50
kwargs = {'x': 7, 'y': 8}
```

## 八、最常见的三种报错来源

### 1. 少给了必填参数

例如：

```python
def f(a, b):
    pass

f(1)
```

### 2. 多给了位置参数，但没有 `*args`

例如：

```python
def f(a, b):
    pass

f(1, 2, 3)
```

### 3. 多给了关键字参数，但没有 `**kwargs`

例如：

```python
def f(a, b):
    pass

f(1, 2, x=3)
```

## 九、最短口诀

可以记成：

**先吃位置，再吃关键字；多余给星号，缺失看默认。**

展开就是：

1. 先看签名分区
2. 先绑定位置实参
3. 再绑定关键字实参
4. 多余位置给 `*args`
5. 多余关键字给 `**kwargs`
6. 没传但有默认值就补默认值
7. 还有必填没绑定就报错
