# P06 输入函数学习引导

## 这节课真正解决什么问题

让程序不只会输出，还能从用户那里拿到数据。

## 最小骨架

1. `input("提示信息")` 用来接收用户输入。
2. 接收到的结果默认都是 `str`。
3. 需要做数字计算时，要先转成 `int` 或 `float`。

```python
name = input("请输入你的名字：")
print("你好，", name)

age = int(input("请输入你的年龄："))
print(age + 1)
```

## 这节最容易错的地方

- 错把 `input()` 的结果当成数字。
- 不写提示信息，导致用户不知道该输入什么。
- 需要小数时还用 `int()`，或者输入内容和目标类型不匹配。

## 现在必须会的判断

- `input()` 返回的默认类型是 `str`
- `18` 这串输入内容，如果来自 `input()`，也先是字符串 `"18"`
- 只有做了类型转换，才能稳定参与数值运算

## 一句话记忆

`input()` 负责拿数据，`int()/float()` 负责把“像数字的字符串”变成真正的数字。
