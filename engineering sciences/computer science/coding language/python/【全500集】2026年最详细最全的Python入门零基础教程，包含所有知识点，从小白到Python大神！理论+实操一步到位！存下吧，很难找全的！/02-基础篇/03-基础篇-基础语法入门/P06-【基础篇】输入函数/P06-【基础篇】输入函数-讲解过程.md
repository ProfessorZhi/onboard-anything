# P06 讲解过程 - 【基础篇】输入函数
## 本P定位

本P是Python基础语法系列的一个独立小节，承接上一P的print函数，引出与之互补的input函数。本P的核心教学目标是让学习者理解程序如何通过input函数实现与用户的交互，并掌握input返回数据类型（字符串）以及如何进行类型转换。

## 讲解过程

### 1. 概念引入：input函数是什么

讲师首先指出之前学习的print函数能够让程序展示内容，而今天要学的input函数刚好和它互补——它是Python中的用户对话工具。有了input函数，程序就能够主动向用户索要数据，比如让用户输入姓名、年龄或成绩。讲师强调，input函数是实现交互式程序的基础，学会它之后程序就能和用户互动起来。

### 2. 形象类比：奶茶店点单

为了帮助理解input函数的工作原理，讲师用奶茶店点单来类比：店员主动问“你要什么口味的奶茶”，这就是给用户的提示信息；顾客回答“珍珠奶茶”，这就是用户的输入；店员把顾客的选择记下来，相当于程序把输入的数据存储起来。input函数就是程序里负责提问和接收答案的“店员”。

### 3. 语法讲解：input函数的基本结构

讲师介绍input函数的语法是`input(--prop=None)`，其中`--prop`参数就是给用户的提示信息，相当于奶茶店店员问“你要什么奶茶”时说的那句话。提示信息会显示在控制台中，告诉用户该输入什么。讲师明确建议提示信息一定要写，虽然它可以省略，但不写会导致用户体验很差。

### 4. 第一次代码演示：输入姓名

讲师打开Python环境，演示第一个input代码：让用户输入姓名。代码是`input("请输入姓名")`，运行时控制台会显示这个提示信息。用户点击后即可输入数据。

### 5. 变量保存与print结合

讲师演示如何保存输入内容：把input赋值给变量`name = input("请输入姓名")`。讲师回顾变量赋值规则——遇到赋值运算符先执行右边表达式，所以程序先执行input接收用户输入，然后保存到左边的变量name中。之后结合print函数打招呼：`print(name, "你好呀")`，逗号隔开的多个值在输出时用空格分隔。讲师演示运行效果：输入“冰冰”，回车后打印“冰冰 你好呀”，说明程序完成了提问、接收、回应这样一个完整的互动过程。

### 6. 提示信息省略的演示

讲师演示不写提示信息的情况：`name = input()`，说明程序不会报错，但控制台没有任何提示，只显示光标，用户根本不知道要输入什么，体验很差。借此强调写代码时不要偷懒。

### 7. 重要坑点：input返回类型

讲师郑重提醒，这是新手最容易栽跟头的地方——input函数默认获取的输入，不管用户输入的是文字、数字还是特殊符号，通通都会被当作字符串类型来处理。讲师用代码演示：用户输入“18”，打印后发现类型是字符串（`class 'str'`）；输入“1.23”同样也是字符串。所有使用input输入的数据都会默认以字符串形式存储。

### 8. 类型不一致导致的报错

讲师演示直接拿输入的数字做计算会报错。用户输入年龄18，想加1岁变成19，但运行时报错。这是因为age变量是字符串类型，字符串不能和整数相加。讲师用形象的比喻解释：这就好像人和猪相加，到底是两个人还是两头猪，都不是，无法进行相加。不同数据类型无法直接进行算术运算。

### 9. 解决方案：类型转换

讲师介绍解决办法——手动给输入数据转换类型。需要整数用`int()`转换，需要小数用`float()`转换。

### 10. 示例一：计算两数之差（int转换）

讲师演示让用户输入两个整数并计算它们的差。首先写代码但没有做类型转换：分别用`input("请输入第一个整数")`和`input("请输入第二个整数")`保存到变量number1和number2，然后`print(number1 - number2)`。运行输入19和2，本应得到17，但报错——字符串不支持相减操作。解决方法是把input的返回值包在`int()`里：`number1 = int(input("请输入第一个整数"))`，`number2 = int(input("请输入第二个整数"))`。再次运行输入19和2，得到正确结果17。

### 11. 示例二：成绩加分（float转换）

讲师演示处理可能有小数的场景。让用户输入数学成绩并计算加5分的结果：先写`score = input("请输入数学成绩")`，然后`print(score + 5)`。运行输入98.5，报错——字符串和整数不能相加。解决方法是把score转换成浮点数：在计算时写`float(score) + 5`或直接修改input那行`score = float(input("请输入数学成绩"))`。运行输入93.5，结果正确显示98.5。

### 12. 类型转换的注意事项

讲师提醒，如果输入的内容无法转换成对应类型，程序会报错。比如用`int("ABC")`尝试把字符串"ABC"转换成整数，程序会报错，因为"ABC"根本没有对应的数值。讲师建议学习者确保输入内容与转换类型匹配，目前阶段注意输入规范即可，后续学了异常处理再处理这类情况。

### 13. 课堂总结

讲师分五点总结本节内容：

- input函数的核心功能是获取用户在控制台的输入，实现程序和用户的交互
- 基本语法是`input(提示信息)`，提示信息可以省略但建议必写
- 关键特性：默认输入的都是字符串类型的数据，哪怕输入的是数字
- 如果需要输入的数字参与运算，使用`int()`或`float()`进行类型转换
- 实用技巧：提示信息要写得清晰，如“请输入姓名”“请输入年龄”“请输入身高（单位：厘米）”

### 14. 课后练习建议

讲师建议学习者课后自己动手练习，比如让用户输入身高和体重计算BMI，或者输入两门课的成绩计算平均分，多练几次就能熟练掌握。

## 本P结论

1. **input函数的作用**：获取用户在控制台输入的数据，实现程序与用户的交互，是实现交互式程序的基础
2. **基本语法**：`input(提示信息)`，提示信息可省略但强烈建议编写
3. **返回值类型**：input函数默认将所有输入（包括数字）转换为字符串类型
4. **类型转换方法**：使用`int(x)`将x转换为整数，使用`float(x)`将x转换为浮点数
5. **注意事项**：确保输入内容与目标转换类型匹配，否则程序会报错
6. **最佳实践**：提示信息应写得清晰具体，明确告知用户需要输入什么以及单位等格式要求
