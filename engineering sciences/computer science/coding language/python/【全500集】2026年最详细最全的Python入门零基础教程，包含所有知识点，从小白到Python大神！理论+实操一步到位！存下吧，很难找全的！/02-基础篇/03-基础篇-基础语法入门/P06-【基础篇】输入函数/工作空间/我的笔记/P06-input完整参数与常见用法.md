# P06 input 完整参数与常见用法

## 先说结论

`input()` 并没有很多参数。

它的真实签名是：

```python
input(prompt='', /)
```

这表示：

- 它只有一个参数：`prompt`
- 这个参数是可选的
- 默认值是空字符串 `''`
- `/` 表示它是**仅位置参数**

## `prompt` 是什么

`prompt` 就是输入前显示给用户看的提示文字。

例如：

```python
name = input("请输入你的名字：")
```

这里：

- `input(...)` 会先显示 `请输入你的名字：`
- 然后等待用户输入
- 最终返回用户输入的文本

## 最完整但不绕弯的理解

### 1. 不传参数

```python
text = input()
```

效果：

- 程序直接等待输入
- 屏幕上没有提示文字

问题：

- 用户不知道该输入什么

所以功能上可以，实际开发里通常不推荐。

### 2. 传入提示信息

```python
name = input("请输入你的名字：")
```

这是最常见、最推荐的写法。

### 3. 接收结果并保存到变量

```python
age_text = input("请输入年龄：")
```

这里 `age_text` 得到的是一个 `str` 对象。

即使输入的是 `18`，得到的也是：

```python
"18"
```

不是：

```python
18
```

### 4. 需要整数时配合 `int()`

```python
age = int(input("请输入年龄："))
```

含义：

1. `input()` 先拿到字符串
2. `int()` 再把字符串转成整数

### 5. 需要小数时配合 `float()`

```python
price = float(input("请输入价格："))
```

### 6. 多次输入组成完整交互

```python
name = input("请输入名字：")
city = input("请输入城市：")
print(name, city)
```

## `/` 到底是什么意思

这个是语法层面的补充。

```python
input(prompt='', /)
```

其中 `/` 的意思是：

`prompt` 只能按位置传，不能按关键字名传。

所以推荐写法是：

```python
input("请输入内容：")
```

而不是：

```python
input(prompt="请输入内容：")
```

入门阶段你只要记住：

`input` 的参数直接写在括号里就行，不需要写参数名。

## `input()` 的返回值特点

永远返回 `str`。

```python
x = input("请输入内容：")
print(type(x))   # <class 'str'>
```

## 一个最小总表

| 写法 | 作用 | 返回类型 |
|------|------|----------|
| `input()` | 无提示地读取输入 | `str` |
| `input("提示")` | 带提示地读取输入 | `str` |
| `int(input("提示"))` | 读取后转成整数 | `int` |
| `float(input("提示"))` | 读取后转成小数 | `float` |

## 常见误区

### 误区 1

“输入 18，拿到的就是整数 18。”

错。

先拿到的是字符串 `"18"`。

### 误区 2

“`input()` 有很多复杂参数。”

错。

它真正的参数只有一个：`prompt`。

### 误区 3

“`input(prompt='xxx')` 是标准写法。”

不推荐。

因为 `prompt` 是仅位置参数，教学和实战里都直接写：

```python
input("xxx")
```

## 一句话记忆

`input()` 的 API 很简单，复杂的不是参数，而是你拿到字符串之后要不要做类型转换。
