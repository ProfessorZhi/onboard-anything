# P04 知识框架 - 【基础篇】注释与输出函数
## 本P位置

第4章【基础篇】注释与输出函数

第1个语法入门P，课程路径：基础语法→注释→print→变量→输入→运算符

为后续所有实战代码奠基：注释提供可读性保障，print提供输出验证能力

## 核心问题

如何给代码写备注（注释）

如何让程序展示结果（print函数）

## 知识骨架

```
注释
├── 单行注释：# 
│   ├── 单独一行 或 行尾（与代码间隔≥2空格）
│   └── PEP8规范：#后加空格
├── 多行注释：""" """ 或 ''' '''
│   ├── 内容跨行
│   └── 类/函数整体说明
└── 快捷键：Ctrl+/ 批量切换

print(...)
├── *args：输出任意数量内容，逗号分隔
│   └── 字符串必须加引号
├── sep：分隔符，默认空格
└── end：结束符，默认换行（未完整讲解）
```

## 关系与依赖

注释 → 增强代码可读性 → 依赖：编辑器/IDE支持

print → 基础输出工具 → 依赖：Python解释器

print输出内容 → 可被注释标注说明 → 两者协同用于调试

## 输入 / 输出 / 前置 / 支撑

| 类型 | 内容 |
|------|------|
| 前置 | 编辑器（IDE） |
| 输入 | 字符串、数字 → print() |
| 输出 | 控制台文本 |
| 支撑 | PEP8规范、IDE语法检测 |

## 易错点

字符串内容不加引号 → 变量未定义报错

注释与代码间距不足2空格 → IDE黄色波浪线

调试后忘记取消注释 → 代码不执行

多行注释用错引号 → 未闭合字符串报错
