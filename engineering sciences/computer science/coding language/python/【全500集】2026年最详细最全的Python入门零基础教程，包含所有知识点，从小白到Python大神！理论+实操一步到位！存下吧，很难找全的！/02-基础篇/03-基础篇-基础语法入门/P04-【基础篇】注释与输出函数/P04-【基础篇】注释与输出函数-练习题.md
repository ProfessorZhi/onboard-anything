# P04 练习题 - 【基础篇】注释与输出函数
## 结构识别

以下代码片段中，哪些行是注释？print函数传入了几个参数（按*args计数）？

```python
# 输出问候语
print("Hello", name, sep=" -> ", end="!\n")
# author: 张三
```

---

## 机制理解

1. print("A", "B", "C") 的输出结果是 `A B C`，而 print("A", "B", "C", sep="") 的输出结果是什么？请解释 sep 参数的作用。

2. 为什么下列代码会报错？

```python
print(Hello World)
```

---

## 最小实践

1. 编写一个完整的 Python 程序，包含：
   - 一个多行注释，说明程序功能
   - 两行单行注释
   - 一条 print 语句，输出你的姓名和年龄（用逗号分隔）

2. 修改下面代码，使其输出格式为：`苹果-香蕉-橘子`（不许改动字符串内容）

```python
print("苹果", "香蕉", "橘子")
```

## 扩展思考

1. 在实际项目中，为什么推荐在代码完成后添加注释，而不是先写注释再写代码？注释过多或过少分别有什么风险？

2. 除了 print，还有哪些方式可以"看到"程序内部的运行状态？请列举至少2种替代方案及其适用场景。

3. Python 的多行注释实际上是什么语法？为什么 class 或 def 的说明文档通常用三个引号而不是 # 号？
