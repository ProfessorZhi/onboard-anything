# P05 练习题 - 【基础篇】变量的定义与赋值
---

## 结构识别

**1. 变量在内存中的角色是什么？以下哪个比喻最准确地描述了变量？**

- A. 变量是一个盒子，里面可以放任何东西
- B. 变量是一个标签，贴在内存中的一块数据上
- C. 变量是一个房间，数据住在里面
- D. 变量是一个计算器，用来处理数据

---

## 机制理解

**2. 以下代码运行后，x 的值是多少？**

```python
x = 10
x = x + 5
```

- A. 10
- B. 15
- C. x + 5
- D. 报错

**3. 下面两行代码有什么区别？哪个会报错？**

```python
a = 100        # 第1行
b = "100"      # 第2行
```

- A. 第1行是数字100，第2行是字符串"100"，都不会报错
- B. 第1行会报错，因为100是数字不能赋值给变量
- C. 第2行会报错，因为引号内的100会被当成未定义的变量
- D. 两者都是数字100，没有任何区别

**4. 以下哪个变量名是非法的？**

- A. `user_name`
- B. `_private`
- C. `2nd_place`
- D. `Class`

## 参考答案

<details>

**1. B**  
变量是标签而非容器，它指向内存中的数据。

**2. B（15）**  
`=` 是赋值运算符，执行顺序从右到左：先计算 `x + 5 = 15`，再赋值给 x。

**3. A**  
不带引号是数字，带引号是字符串，两者都不会报错，但类型不同。

**4. C（`2nd_place`）**  
变量名不能以数字开头。A、B 合法，D 虽然是关键字但实际运行才会报错。

**5.**

```python
message = "Hello"
count = 42
print(type(message))  # <class 'str'>
print(type(count))    # <class 'int'>
```

**6.**

| 行号 | 错误 | 修正 |
|---|---|---|
| 第1行 | 变量名含连字符 `-` | `my_name = "Alice"` |
| 第2行 | `class` 是关键字 | `class_name = "Python"` |
| 第3行 | 变量名含特殊符号 `$` | `price = 99.9` |

**7. 利用了序列赋值特性**  
`a, b = b, a` 会先计算右侧的元组 `(b, a)`，再按位置依次赋值给左侧变量。

**8. 会报错**  
`num` 先是整数 5，后被覆盖为字符串 `"five"`。执行 `num + 3` 时，会尝试将字符串与整数相加，触发 `TypeError`。

**9. 略**（在在线环境中完成即可）

</details>
