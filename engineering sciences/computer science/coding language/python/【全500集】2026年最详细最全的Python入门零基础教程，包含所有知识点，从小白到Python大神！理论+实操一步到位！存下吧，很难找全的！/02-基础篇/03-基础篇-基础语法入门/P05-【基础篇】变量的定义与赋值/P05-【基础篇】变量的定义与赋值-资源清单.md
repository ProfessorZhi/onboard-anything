- **类型**：代码风格指南

## 四、Python 内置函数

### type()

- **类型**：Python 内置函数
- **在本P中的作用**：查看变量或数据的类型，返回 `<class 'int'>`、`<class 'str'>` 等格式。老师用它逐一演示 int、float、bool、str、complex 等五种数据类型的检测。
- **来源**：字幕直接提到
- **建议方向**：对整型、浮点型、布尔型、字符串、复数、列表、元组、字典等多种数据使用 type()，观察返回结果

### print()

- **类型**：Python 内置函数
- **在本P中的作用**：输出变量或数据的值到控制台。老师在演示直接赋值、变量间赋值、序列赋值时都用 print() 验证结果。
- **来源**：字幕直接提到（多次演示但未在资源表格中列出）
- **建议方向**：练习 print() 输出单个变量、多个变量（逗号分隔）、带格式化字符串输出

## 五、Python 数据类型

### int（整型）

- **类型**：Python 内置数据类型
- **在本P中的作用**：存储整数，如 100、-50、0。老师用 `money = 1000001234` 演示整型赋值。
- **来源**：字幕直接提到
- **建议方向**：练习大整数运算、观察整型的表示范围（Python 3 整型无上限限制）

### float（浮点型）

- **类型**：Python 内置数据类型
- **在本P中的作用**：存储小数，如 3.14159。老师演示 `print(type(3.14159))` 输出浮点型。
- **来源**：字幕直接提到
- **建议方向**：练习浮点数运算、注意浮点精度问题（如 0.1 + 0.2 不等于 0.3）

### bool（布尔型）

- **类型**：Python 内置数据类型
- **在本P中的作用**：只有 True 和 False 两个值，用于条件判断。老师强调 True 是关键字，在 PyCharm 中有颜色显示，加引号就变成字符串。
- **来源**：字幕直接提到
- **建议方向**：理解布尔值与整数的关系（True == 1, False == 0）、练习布尔运算（and、or、not）

### complex（复数型）

- **类型**：Python 内置数据类型
- **在本P中的作用**：用于表示复数，如 `num = 23j`。老师演示了复数类型的 type() 检测。
- **来源**：字幕直接提到
- **建议方向**：了解复数的创建方式（1+2j 或 complex(1,2)）、尝试复数的基本运算

### str（字符串类型）

- **类型**：Python 内置数据类型
- **在本P中的作用**：存储文字数据，用单引号、双引号、三引号包裹。老师强调带引号是字符串，不带引号会被当作变量名。
- **来源**：字幕直接提到
- **建议方向**：练习单引号和双引号的区别、三引号多行字符串、字符串的拼接和索引

## 六、命名规范

### snake_case（下划线分割法）

- **类型**：命名规范
- **在本P中的作用**：老师推荐的变量命名方式，格式为 `user_name`、`student_age`，全部小写，单词间用下划线隔开。
- **来源**：字幕直接提到
- **建议方向**：用 snake_case 为自己的变量命名，如 `max_score`、`first_name`、`is_valid`

### PascalCase（大驼峰命名法）

- **类型**：命名规范
- **在本P中的作用**：每个单词首字母大写，如 `UserName`。老师说明适用于类名定义（后续课程会用到）。
- **来源**：字幕直接提到
- **建议方向**：了解何时使用大驼峰（如类名、异常类名），与 snake_case 对比记忆

### camelCase（小驼峰命名法）

- **类型**：命名规范
- **在本P中的作用**：第一个单词小写，从第二个单词开始首字母大写，如 `userName`。老师简要介绍作为对比。
- **来源**：字幕直接提到
- **建议方向**：了解该命名风格在其他语言（如 JavaScript）中的使用场景
