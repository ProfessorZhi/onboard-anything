# P07 练习题 - 【基础篇】运算符
---

## 结构识别

**题目**：判断以下表达式使用了哪类运算符（算术运算符 / 赋值运算符 / 两者均有）

| 表达式 | 运算符类型 |
|--------|-----------|
| `a = 10` | |
| `b = 5 + 3` | |
| `c *= 2` | |
| `d = 8 // 3` | |
| `e = a + b - 1` | |

---

## 机制理解

**题目 1**：计算并说明结果类型

```python
result = 10 / 5
print(result)
```

输出结果是多少？类型是 `int` 还是 `float`？

**题目 2**：判断以下表达式是否会报错，如报错说明原因

```python
a = 10 / 0
b = 5 // 0
c = 7 % 0
```

**题目 3**：不加括号，判断 `2 + 3 * 4 ** 2 / 8 - 1` 的运算顺序，并计算最终结果。

---

## 最小实践

**题目 1**：阅读以下代码，补全注释中的计算结果

```python
x = 10
x //= 3      # x = ___
y = 7
y %= 4       # y = ___
z = 2
z **= 5      # z = ___
print(x, y, z)
```

**题目 2**：仅使用 `+` 和 `-`，编写一段代码，利用复合赋值将变量 `n` 从 1 逐步变化到 10，输出每一步的 `n` 值。

**题目 3**：不使用除法运算符，判断一个整数 `num` 除以 3 的余数是否为 0（提示：取模运算）。

---

## 扩展思考

**题目 1**：场景应用

某商品定价 99.9 元，顾客支付 200 元。请设计计算找零金额的代码，要求：
- 输出应找回多少钱
- 考虑可能存在小数找零的情况（保留两位小数）

**题目 2**：对比 `a = a + 5` 和 `a += 5`，两者在功能上有何区别？在什么场景下复合赋值有明显优势？

## 最小替代练习

**题目**：使用 Python 交互式环境（或任意编辑器），编写脚本实现以下功能

```python
# 1. 定义变量 m = 7
# 2. 使用两种方式让 m 翻倍（算术运算 vs 复合赋值）
# 3. 验证两种方式的结果是否相同
# 4. 尝试用 % 判断 m 是否为偶数
```

---

**参考答案提示**：

- 结构识别：赋值/算术+赋值/赋值/算术/两者均有
- 机制理解 1：输出 `2.0`，`float` 类型
- 机制理解 2：三个表达式均触发 `ZeroDivisionError`
- 机制理解 3：先算 `4 ** 2 = 16`，再 `* 8 / 8` 抵消，最终结果是 `2 + 16 - 1 = 17`
- 最小实践 1：x=3, y=3, z=32
