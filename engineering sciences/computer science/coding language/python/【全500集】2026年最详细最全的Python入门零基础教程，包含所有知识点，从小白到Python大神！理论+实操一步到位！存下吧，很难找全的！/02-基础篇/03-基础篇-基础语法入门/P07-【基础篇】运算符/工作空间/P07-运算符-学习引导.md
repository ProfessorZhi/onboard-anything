# P07 学习引导

## 这一课真正解决什么问题

把“拿到的数据”变成“可计算的结果”，并且学会把计算结果写回变量。

## 你现在所处的位置

- 上一课 `P06`: 学会从用户那里拿输入
- 这一课 `P07`: 学会对数字做计算
- 下一步: 用比较和逻辑运算把“计算结果”变成“判断条件”

## 本课最小骨架

1. 算术运算符产生新值: `+ - * / // % **`
2. 赋值运算符把值交给变量: `=`
3. 复合赋值一边算一边改: `+= -= *= /= //= %= **=`

## 最容易混淆的点

- `/` 是普通除法, 结果通常是 `float`
- `//` 是取整除, 会去掉小数部分
- `%` 是取余, 常用来判断能不能整除
- `=` 不是“等于”, 而是“把右边结果放进左边变量”

## 这一课的目标

看到表达式时, 你能回答三件事:

1. 先算什么
2. 最后得到什么值
3. 哪个变量被改了
