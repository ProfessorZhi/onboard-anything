## 三、语法元素

**算术运算符**
- 类型：运算符（7个）
- 在本P中的作用：用于数值计算。加（+）、减（-）、乘法（*）、除法（/）、取整除（//）、取余（%）、幂运算（**），运算逻辑与数学基本一致。
- 来源：字幕直接提到
- 建议方向：设计综合练习题，如计算圆的周长和面积、三角形面积、复利公式等，巩固各运算符的使用。

**赋值运算符**
- 类型：运算符（两类）
- 在本P中的作用：用于修改变量值。简单赋值（=）装数据，复合赋值（+= -= *= /= //= %= **=）先计算再重新赋值，比 `x = x + y` 写法更简洁、可维护性更高。
- 来源：字幕直接提到
- 建议方向：用复合赋值模拟累计场景，如购物车累加价格、积分递增等。

**运算符优先级规则**
- 类型：语法规则
- 在本P中的作用：确定表达式中各运算符的执行顺序。优先级从高到低为：括号 > 幂 > 乘除取整取余 > 加减，同级从左往右计算。
- 来源：字幕直接提到
- 建议方向：编写复杂表达式验证优先级，或故意不加括号观察结果差异。

**浮点数运算规则**
- 类型：语法规则
- 在本P中的作用：只要运算中有浮点数参与，结果通常也是浮点数。因为浮点数精度范围比整数更广。
- 来源：字幕直接提到
- 建议方向：设计int与float混合运算的练习，观察结果类型的变化规律。

## 四、异常与错误

**ZeroDivisionError**
- 类型：内置异常
- 在本P中的作用：除法、取整除、取余运算中除数为零时触发。强调除数不能为零这一数学规定，是需要处理的边界情况。
- 来源：字幕直接提到
- 建议方向：模拟用户输入除法程序，添加除数为零的防御逻辑。

**NameError**
- 类型：内置异常
- 在本P中的作用：使用复合赋值运算符前变量未定义时触发。强调复合赋值必须先定义变量并赋初值。
- 来源：字幕直接提到
- 建议方向：故意遗漏变量初始化，观察报错信息，加深对变量定义顺序的理解。

## 五、通用概念

**赋值与比较的区别**
- 类型：概念辨析
- 在本P中的作用：澄清 `=` 是赋值（把右边的值装入左边的变量），不是数学中的等于；比较相等应使用 `==`。
- 来源：字幕直接提到
- 建议方向：在练习中区分赋值语句和比较表达式，避免混用。

**取余数的应用：判断奇偶性**
- 类型：应用场景
- 在本P中的作用：`n % 2 == 0` 为偶数，`n % 2 == 1` 为奇数，是取余运算符的典型实战用法。
- 来源：字幕直接提到
- 建议方向：编写一个程序，输入一个数判断奇偶，或遍历1-100输出奇数/偶数。
