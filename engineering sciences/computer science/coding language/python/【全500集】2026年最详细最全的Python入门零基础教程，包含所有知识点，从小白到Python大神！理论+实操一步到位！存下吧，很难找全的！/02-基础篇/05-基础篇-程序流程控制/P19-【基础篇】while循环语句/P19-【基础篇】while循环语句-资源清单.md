# P19 资源清单 - 【基础篇】while循环语句
## 资源列表

### 1. PyCharm

- **名称**：PyCharm
- **类型**：IDE（集成开发环境）
- **在本 P 中的作用**：本P的演示开发环境，用于编写、运行和调试Python代码。提供了代码编辑区、运行按钮、变量监视和debug调试功能，是初学者接触Python编程的主要工具界面。
- **来源**：字幕直接提到（debug调试演示部分明确使用PyCharm作为演示IDE）
- **后续可练习方向**：建议方向：自行安装PyCharm Community Edition（免费版），创建Python项目，新建.py文件练习编写while循环代码，尝试运行不同场景下的循环程序，逐步熟悉IDE的基本操作流程和快捷键。

---

### 2. debug 调试功能

- **名称**：PyCharm Debug（调试器）
- **类型**：调试工具/功能
- **在本 P 中的作用**：帮助学生理解while循环执行流程的关键工具。通过逐行执行和变量监视，学生可以亲眼看到count变量如何从1递增到6，以及条件判断如何最终变为False导致循环结束。在排查死循环成因时，debug功能直观展示了变量值不变化的根本原因。
- **来源**：字幕直接提到（debug调试演示环节专门使用此功能追踪程序执行）
- **后续可练习方向**：建议方向：在PyCharm中为while循环代码设置断点，使用Step Over（F8）和Step Into（F7）逐行观察变量变化，尝试在不同位置设置断点来理解代码执行顺序，特别是区分循环体内代码与循环体外的代码在执行次数上的差异。

---

### 3. while True + break 组合

- **名称**：while True + break 组合模式
- **类型**：语法组合/控制流模式
- **在本 P 中的作用**：作为死循环的合理变体被预告。老师指出虽然`while True`本身会导致无限循环，但配合`break`语句和合适的退出机制（如用户输入特定内容时跳出）可以实现带终止条件的持续运行程序。本P只做铺垫，详细用法留待后续break/continue章节。
- **来源**：基于上下文推断（知识框架中明确列出，讲解过程也预告了后续章节会详讲，但字幕未在本P中展开具体语法）
- **后续可练习方向**：建议方向：等待学习break语句章节后，回顾本P内容，尝试编写带退出机制的小程序（如猜数字游戏、菜单循环选择等），实践while True与break的组合使用，理解如何设计合理的退出条件来控制死循环。
