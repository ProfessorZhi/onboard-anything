## 二、调试工具与功能

**Debug调试器**
- 类型：IDE内置调试工具

- 来源：字幕直接提到
- 后续建议方向：练习设置条件断点、查看调用栈、监视变量变化等进阶调试技巧

**断点（Breakpoint）**
- 类型：调试功能
- 在本P中的作用：在代码行设置暂停标记，配合Debug工具实现单步执行，控制程序执行流程
- 来源：字幕直接提到
- 后续建议方向：练习使用条件断点、临时断点、日志断点等不同类型的断点

## 四、运算符（命令）

**% 取模运算符**
- 类型：算术运算符（命令）
- 在本P中的作用：核心教学点，用于判断整除（余数为0则为偶数），是实现偶数判断的关键表达式
- 来源：字幕直接提到
- 后续建议方向：练习用取模运算判断其他整除规则，如能被3整除、能被5整除等

**== 相等比较运算符**
- 类型：比较运算符（命令）
- 在本P中的作用：用于if语句的条件表达式中，判断两边是否相等，返回布尔值True或False
- 来源：上下文推断
- 后续建议方向：练习在if条件中使用不同比较运算符（==、!=、>、<、>=、<=）

**/ 普通除法运算符**
- 类型：算术运算符（命令）
- 在本P中的作用：作为反面教材演示，说明普通除法产生浮点数，无法用于整除判断
- 来源：字幕直接提到
- 后续建议方向：理解浮点数运算特性，与整除运算进行对比练习

**// 整除运算符**
- 类型：算术运算符（命令）
- 在本P中的作用：作为反面教材演示，说明整除只返回商值，无法判断是否整除
- 来源：字幕直接提到
- 后续建议方向：结合取模运算，理解两者配合使用的方法（如判断商和余数）
