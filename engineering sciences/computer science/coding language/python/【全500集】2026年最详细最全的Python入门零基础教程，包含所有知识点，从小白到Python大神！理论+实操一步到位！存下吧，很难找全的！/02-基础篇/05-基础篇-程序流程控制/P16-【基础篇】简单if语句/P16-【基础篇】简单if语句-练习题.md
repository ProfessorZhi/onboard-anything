**题目 2**：以下代码的执行结果是「没中奖」还是「中奖了」？

```python
number = 7
if number % 2 == 0:
    print("中奖了")
print("没中奖")
```

---

## 机制理解

**题目 3**：判断正误

- [ ] `if` 语句的条件表达式结果必须是 `True` 或 `False`
- [ ] if 代码块内部的语句必须缩进
- [ ] if 语句后面的冒号可以用中文冒号替代
- [ ] 当条件为 `False` 时，if 代码块内的所有语句都会被跳过
