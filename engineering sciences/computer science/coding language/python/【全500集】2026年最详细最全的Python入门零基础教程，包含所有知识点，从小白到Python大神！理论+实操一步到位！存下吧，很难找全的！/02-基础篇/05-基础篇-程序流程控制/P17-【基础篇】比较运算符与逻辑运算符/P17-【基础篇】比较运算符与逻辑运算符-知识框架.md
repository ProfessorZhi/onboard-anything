# P17 知识框架 - 【基础篇】比较运算符与逻辑运算符
## 本P位置
Python基础语法 → 条件判断体系 → 比较与逻辑运算符（条件判断的构建模块）

前序：变量赋值、if语句基础
后续：else/elif分支、三元表达式、while循环条件

## 核心问题
如何让程序做决策？需要什么工具来构建判断条件？

## 知识骨架

### 一、比较运算符（原子条件）
```
==  !=  >  <  >=  <=
```
返回布尔值：`True` / `False`

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| == | 等于 | `1 == 1` | True |
| != | 不等于 | `1 != 1` | False |
| > | 大于 | `2 < 3` | True |
| < | 小于 | `3 < 2` | False |
| >= | 大于或等于 | `3 >= 2` | True |
| <= | 小于或等于 | `3 <= 3` | True |

### 二、逻辑运算符（组合条件）
```
and  or  not
```

| 运算符 | 含义 | 逻辑 |
|--------|------|------|
| and | 并且 | 所有条件为True才返回True |
| or | 或者 | 任一条件为True即返回True |
| not | 非 | 反转结果：True→False，False→True |

### 三、隐式真假值（Python特性）
非零非空 → True
零或空 → False

| 视为True | 视为False |
|----------|-----------|
| 非零数字 | 0 |
| 非空字符串 | '' |
| 非空容器 | [] () {} set() |
| | None |

## 关系与依赖

```
赋值(=) ──区分──> 比较(==)

原子条件 ──组合──> 复合条件
  比较运算符 ──逻辑运算符──> 复合条件
          └──隐式真假──> 简化写法

输入(input) ──返回──> 字符串 ──类型转换──> 数值
                              ↓
                        比较/逻辑运算
                              ↓
                        布尔值(True/False)
                              ↓
                        if条件判断
```

## 输入 / 输出 / 前置 / 支撑

| 类别 | 内容 |
|------|------|
| 输入 | `input()` → 字符串 |
| 输出 | 布尔值True/False、打印结果 |
| 前置 | 变量赋值概念、if语句语法 |
| 支撑 | `int()`类型转换、`print()`输出 |

## 易错点

| 错误类型 | 正确认知 |
|----------|----------|
| `=` 与 `==` 混淆 | `=`赋值，`==`比较 |
| `>=` 误理解为"且" | `>=`是"或"关系：满足大于或等于其一即可 |
| 字符串数字比较 | `'5' == 5` → False（类型不同） |
| 大小类型混合比较 | 字符串与数字不可比大小（报错） |
| 相等性混合比较 | 不同类型相等性比较不报错，返回False |
| 冗余写法 | `if is_student == True` 应简化为 `if is_student` |
| 类型转换遗漏 | `input()`返回字符串，数字比较需`int()`转换 |
