### 5. 逻辑运算符 `and`

| 字段 | 内容 |
|------|------|
| **名称** | `and` |
| **类型** | Python 语言内置关键字 / 逻辑运算符 |
| **在本P中的作用** | 组合多个条件，全部条件为真时整体才为 True；本P以登录验证（用户名且密码均正确）为案例演示其行为 |
| **来源** | 字幕直接提到 |

### 6. 逻辑运算符 `or`

| 字段 | 内容 |
|------|------|
| **名称** | `or` |
| **类型** | Python 语言内置关键字 / 逻辑运算符 |
| **在本P中的作用** | 组合多个条件，任意一个条件为真时整体即为 True；本P以判断输入是否为常见水果（苹果或香蕉或橙子）为案例演示其行为 |
| **来源** | 字幕直接提到 |
| **官方文档** | https://docs.python.org/zh-cn/3/reference/expressions.html#boolean-operations |
| **建议方向** | 同样构造真值表；探索 `or` 的短路求值特性（左侧为 True 时右侧不执行）；对比本P的多条件 `or` 写法与后续章节可能出现的 `in` 运算符写法（如 `fruit in ['苹果', '香蕉', '橙子']`），体会不同写法的可维护性差异 |

---

### 7. 逻辑运算符 `not`

| 字段 | 内容 |
|------|------|
| **名称** | `not` |
| **类型** | Python 语言内置关键字 / 逻辑运算符（一元） |
| **在本P中的作用** | 反转布尔值，True 变 False、False 变 True；本P演示用 `not num == 0` 等价替代 `num != 0`，并结合隐式真假值讲解 `if not num` 的简化写法 |
| **来源** | 字幕直接提到 |
| **官方文档** | https://docs.python.org/zh-cn/3/reference/expressions.html#boolean-operations |
| **建议方向** | 练习 `not` 与隐式真假值的组合用法：`not 0`、`not ''`、`not []`、`not None` 各自返回什么；理解"非零非空即 True"的隐式转换规则，为后续简化条件写法建立肌肉记忆 |
