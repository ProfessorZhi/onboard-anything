## 二、内置工具与功能

### range()
- 类型：Python内置函数
- 在本P中的作用：生成数值序列（range(1,6)），作为for循环的经典可迭代对象，配合吃苹果场景演示break和continue
- 来源：字幕直接提到
- 后续可练习方向：掌握range的start、stop、step参数；结合列表推导式使用；模拟斐波那契数列等

### break 语句
- 类型：Python关键字
- 在本P中的作用：满足条件时直接终止当前循环，配合吃苹果吃饱的场景演示
- 来源：字幕直接提到
- 后续可练习方向：在嵌套循环中使用break只退出内层循环；结合用户登录验证场景

### continue 语句
- 类型：Python关键字
- 在本P中的作用：满足条件时跳过当前次剩余代码，直接开始下一次循环，配合吃到坏苹果跳过的场景演示
- 来源：字幕直接提到
- 后续可练习方向：跳过列表中特定元素；配合条件筛选数据；嵌套循环中的continue行为

### 循环 + else 语法
- 类型：Python语法特性
- 在本P中的作用：else块的代码只在循环正常结束（未执行break）时执行，用于判断循环是否被提前中断
- 来源：字幕直接提到
- 后续可练习方向：搜索场景中判断是否遍历完毕；结合标志位替代else块的写法对比

- 类型：错误类型
- 在本P中的作用：break和continue单独用于if语句或函数时会报此错误，错误信息包含具体位置提示
- 来源：字幕直接提到
- 后续可练习方向：主动触发各类SyntaxError并分析错误信息；理解缩进错误的报错形式

### "break outside loop" 错误
- 类型：错误信息
- 在本P中的作用：break不在循环体内时的报错内容，用于说明break的使用约束
- 来源：字幕直接提到
- 后续可练习方向：对比"break outside loop"和"continue not properly in loop"的区别
