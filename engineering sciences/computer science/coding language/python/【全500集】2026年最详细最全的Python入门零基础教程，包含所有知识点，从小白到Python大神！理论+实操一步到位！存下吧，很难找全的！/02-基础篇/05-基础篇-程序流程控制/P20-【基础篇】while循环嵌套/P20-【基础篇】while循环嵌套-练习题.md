# P20 练习题 - 【基础篇】while循环嵌套
## 一、结构识别

**1.1** 以下代码运行后，print 语句共执行了多少次？

```python
row = 1
while row <= 3:
    col = 1
    while col <= 4:
        print(f"({row},{col})", end=" ")
        col += 1
    row += 1
```

- A. 3 次
- B. 4 次
- C. 7 次
- D. 12 次

---

**1.2** 下面代码的输出是「一行」还是「多行」？内层循环的 `end` 参数设置了什么值？

```python
i = 1
while i <= 2:
    j = 1
    while j <= 3:
        print("*", end="\t")
        j += 1
    i += 1
```

## 二、机制理解

**2.1** 补全执行流程：在第 2 次进入外层循环时，变量 `row` 的值是多少？内层变量 `col` 会被重新初始化为多少？

```python
row = 1
while row <= 3:
    col = 1          # ← 内层变量在这里初始化
    while col <= 2:
        print(f"{row}-{col}")
        col += 1
    row += 1
```

---

**2.2** 以下代码的错误是「内层循环未正确嵌套」。请预测如果把外层和内层的 `while` 写成平级（同一缩进），输出会变成怎样的顺序？

```python
# 错误写法示例
row = 1
while row <= 2:
    print(f"外层第{row}次")
    row += 1

col = 1
while col <= 2:
    print(f"内层第{col}次")
    col += 1
```

## 三、最小实践

**3.1** 补全代码：用 while 循环嵌套打印一个 3 行 5 列的星号矩阵，效果如下：

```
* * * * *
* * * * *
* * * * *
```

```python
row = 1
while row <= 3:
    col = 1
    while col <= 5:
        print("*", end=" ")
        col += 1
    # 在这里加一行代码实现换行
    row += 1
```

---

**3.2** 修改下面代码，使得输出从 1 开始递增（而非从 0 开始），外层循环 2 次，内层 4 次：

```python
row = 0
while row < 2:
    col = 0
    while col < 4:
        print(row * 4 + col + 1, end=" ")
        col += 1
    print()
    row += 1
```
