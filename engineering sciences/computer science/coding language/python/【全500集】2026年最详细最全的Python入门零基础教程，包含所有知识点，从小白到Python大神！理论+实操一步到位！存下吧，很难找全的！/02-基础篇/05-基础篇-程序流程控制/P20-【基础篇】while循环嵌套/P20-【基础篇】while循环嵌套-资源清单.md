## 二、内置函数与参数

### print() 函数
- **类型**：Python 内置函数
- **在本 P 中的作用**：在嵌套循环中负责输出座位信息，通过 `end` 参数控制每列之间的分隔方式和每排结束后的换行行为。
- **来源**：字幕直接提到（多次在讲解代码时调用 `print()`，并演示了 `end` 参数的效果）
- **后续可练习方向**：对比 `print()` 默认行为与显式指定 `end` 参数的区别，尝试用不同的 `end` 值（如空格 ` `、连字符 `-`）实现不同的输出格式。

### end 参数
- **类型**：函数参数（print() 的关键字参数）
- **在本 P 中的作用**：
  - `end="\t"`：在内层循环（列循环）中实现同行对齐，使同一排的多个座位标注横向排列而非自动换行。
  - `end="\n"`（默认值）：在内层循环结束后触发换行，使输出呈现为整齐的多行多列表格。
- **来源**：字幕直接提到（老师通过修改变量值演示了 `\t` 和 `\n` 的区别）
- **后续可练习方向**：尝试将 `end` 设置为其他分隔符（如空格、逗号、箭头），观察输出格式的变化；结合文件写入（`print(..., file=f)`）理解 `end` 参数在文件操作中的作用。

## 三、转义字符

### \t（制表符）
- **类型**：转义字符
- **在本 P 中的作用**：在内层循环的 `print()` 中作为 `end` 参数值，使输出在同一行内以固定宽度（制表宽度）对齐各列，实现三排六列的整齐表格效果。
- **来源**：字幕直接提到（老师将 `end` 参数设为 `\t` 以实现列对齐）
- **后续可练习方向**：在输出表格数据（如九九乘法表、星号图案）时使用制表符对齐；对比 `\t` 与多个空格在排版效果上的差异。

### \n（换行符）
- **类型**：转义字符
- **在本 P 中的作用**：作为 `print()` 的默认 `end` 参数值，在内层循环结束后插入换行，将光标移至下一行开头，从而在视觉上分隔不同的排。
- **来源**：字幕直接提到（讲解 `print()` 默认行为时提及）
- **后续可练习方向**：在字符串中显式使用 `\n` 实现多行文本输出；理解不同操作系统（Windows `\r\n`、Linux `\n`、macOS `\r`）对换行符的处理差异。
