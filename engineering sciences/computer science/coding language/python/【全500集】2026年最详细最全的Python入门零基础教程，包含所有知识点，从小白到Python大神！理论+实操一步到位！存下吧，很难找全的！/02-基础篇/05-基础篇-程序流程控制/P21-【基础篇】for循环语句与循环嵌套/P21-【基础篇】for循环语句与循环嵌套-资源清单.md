## 内置函数 / 内置对象

### range()

| 属性 | 内容 |
|---|---|
| **名称** | range() |
| **类型** | 内置函数 |
| **在本P中的作用** | 生成连续整数序列，是for循环实现精确计次的标配工具；本P详细讲解了三种参数格式：`range(stop)`、`range(start, stop)`、`range(start, stop, step)`，以及「包前不包后」规则和负数步长用法 |
| **来源** | 字幕直接提到 |
| **后续可练习方向** | ① 熟练使用三种range参数组合；② 验证包前不包后边界；③ 用步长实现倒序输出；④ 用range结合循环实现累乘、阶乘、数列生成；⑤ 探索range与列表的互相转换 |

---

### print()

| 属性 | 内容 |
|---|---|
| **名称** | print() |
| **类型** | 内置函数 |
| **在本P中的作用** | 用于输出循环结果；利用`end="\t"`参数实现不换行输出列，用空`print()`实现内层循环结束后的换行 |
| **来源** | 字幕直接提到 |
| **后续可练习方向** | ① 结合end参数控制输出格式；② 实现各种图案对齐输出；③ 格式化字符串在循环中的应用 |

### 可迭代对象

| 属性 | 内容 |
|---|---|
| **名称** | 可迭代对象（Iterable） |
| **类型** | 概念统称 |
| **在本P中的作用** | 统一定义可以用for循环遍历的对象类型，包括列表、字符串、元组、字典、集合；老师强调「能用for循环语法不报错的就是可迭代对象」，并演示整数遍历会报`is not iterable`错误 |
