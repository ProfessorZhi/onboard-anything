**6. 字符串头尾截取翻转拼接**

- 输入：字符串 `s`、整数 `d`
- 逻辑：截取字符串头部 `d` 个字符和尾部 `d` 个字符，分别翻转后拼接
- 返回：拼接结果

```python
# 示例
# flip_concat("HelloWorld", 3) → "olleHWorl"
# 解释：头部"Hel"翻转→"lleH"，尾部"rld"翻转→"dlr"，拼接→"lleHdlr"  （此处需核实原题逻辑，可调整为"Hel"翻转得"lleH"）
```

| 题号 | 核心思路 |
