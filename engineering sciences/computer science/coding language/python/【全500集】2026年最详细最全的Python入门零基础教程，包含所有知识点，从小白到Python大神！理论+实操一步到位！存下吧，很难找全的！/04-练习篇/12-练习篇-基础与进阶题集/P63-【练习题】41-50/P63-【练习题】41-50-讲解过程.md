### 第41题 翻转字符串的头尾部分并拼接

**需求引入**：老师先读题：定义一个函数，接收一个字符串和一个整数作为参数。操作包括：从字符串头部截取指定数量的字符并翻转；从字符串尾部截取相同数量的字符并翻转；分别打印头部和尾部翻转后与剩余部分拼接后的新字符串。

**代码实现**：

1. **定义函数**：老师先考虑函数三大要素：作用是截取翻转拼接，参数为字符串`s`和整数`d`，无返回值要求。函数命名为`rotate_and_concat`。

2. **头部截取翻转**：
   - 使用切片`str[d:]`从头部截取指定数量字符（步长默认为1，包前不包后）
   - 翻转则设定步长为-1，实现逆序输出
   - 保存到变量`head_flipped`
   - 头部剩余部分用`str[d:]`截取，保存为`head_remaining`
   - 拼接：`head_result = head_flipped + head_remaining`

3. **尾部截取翻转**：
   - 使用负索引`str[-d:]`从尾部截取
   - 翻转同样设定步长为-1
   - 保存到变量`tail_flipped`
   - 尾部剩余部分用`str[:-d]`截取，保存为`tail_remaining`
   - 拼接：`tail_result = tail_flipped + tail_remaining`

4. **打印结果**：
   ```python
   print("头部切片翻转后拼接为:", head_result)
   print("尾部切片翻转后拼接为:", tail_result)
   ```

5. **函数调用测试**：
   ```python
   input_string = "round bob"
   num_chess = 2
   rotate_and_concat(input_string, num_chess)
   ```
   老师演示：以"round bob"为例，`d=2`时，头部截取"ro"翻转成"or"拼接剩余"und bob"得到"orund bob"；尾部截取"b"翻转成"b"拼接剩余"round bo"得到"bround bo"。

6. **代码健壮性增强**：老师补充边界检查，判断截取数量是否超出字符串长度，若超出则打印提示并`return`结束函数。

### 第42题 交换字典的键与值

**需求引入**：题目要求编写函数交换指定字典的key和value。例如给定字典`{'A':1, 'B':2, 'C':3}`，调用函数后返回`{1:'A', 2:'B', 3:'C'}`。

**代码实现**：

1. **定义函数**：
   - 函数名`swap_dict_keys_and_values`
   - 参数`dct`（老师强调不用`dict`命名，因为Python已有内置类）
   - 返回交换后的新字典

2. **遍历原字典**：
   - 使用`for key in dct`遍历所有键
   - 通过键获取对应值：`dct[key]`

3. **交换并存入新字典**：
   ```python
   swapped_dict = {}
   for key in dct:
       swapped_dict[dct[key]] = key
   return swapped_dict
   ```

4. **调用测试**：传入`original_dict = {'A':1, 'B':2, 'C':3}`，验证输出`{1:'A', 2:'B', 3:'C'}`。

### 第43题 验证整数输入

**需求引入**：定义函数尝试将用户输入的字符串转换为整数。转换成功返回整数，转换失败返回错误信息。

**代码实现**：

1. **函数设计**：
   - 函数名`string_to_int`
   - 无参函数，在函数内部用`input()`接收用户输入
   - 有两种返回值情况

2. **异常处理结构**：
   ```python
   try:
       num = int(input("请输入一个整数:"))
   except:
       return "输入的不是有效整数"
   else:
       return num
   ```

3. **测试演示**：
   - 输入`123`：正常转换，输出`123`
   - 输入`ABC`：捕获异常，返回"输入的不是有效整数"

4. **讲解要点**：老师强调使用`try-except`处理可能报错也可能不报错的代码，避免程序崩溃，优雅地返回错误信息。

### 第44题 验证年龄

**需求引入**：定义函数接受年龄作为参数，若年龄不在合法范围（0-120岁）则抛出异常。

**代码实现**：

1. **函数设计**：
   - 函数名`verify_age`
   - 参数`age`
   - 正常情况返回年龄，不合法则抛出异常

2. **基本返回**：
   ```python
   return age
   ```
   老师先写基本返回，测试传`19`正常返回，传`190`也返回了但实际不合法。

3. **添加合法性判断与抛出异常**：
   ```python
   if age < 0 or age > 120:
       raise Exception("年龄必须在0~120岁之间")
   ```

4. **异常处理调用**：
   ```python
   try:
       result = verify_age(190)
       print(result)
   except Exception as e:
       print(e)
   print("12323")
   ```
   - 不处理时，`190`触发异常，后面的`print("12323")`不执行
   - 用`except Exception as e`捕获后，异常信息保存在变量`e`中打印出来，后面的代码继续执行

5. **讲解要点**：老师说明`raise`用于抛出自定义异常，`Exception`是万能异常类型，可传入描述信息。异常被捕获后程序不会终止，后面的代码继续运行。

### 第45题 绘制螺旋形圆圈图案

**需求引入**：使用Python的tkinter库创建图形界面，绘制由圆圈组成的螺旋形图案。

**外部库说明**：tkinter是Python内置的GUI库，用于创建图形用户界面、对话框和绘图。

**代码实现**：

1. **导入模块**：
   ```python
   from tkinter import *
   ```

2. **创建画布**：
   ```python
   canvas = Canvas(root, width=500, height=400)
   canvas.pack(expand=True, fill=BOTH)
   ```
   - `Canvas`组件用于绘制图形
   - `width`和`height`设置画布尺寸（像素）
   - `pack()`布局管理器，`expand=True`允许扩展，`fill=BOTH`填充

3. **绘制圆圈循环**：
   ```python
   k = 1
   j = 1
   for i in range(26):
       canvas.create_oval(250-k, 200-k, 250+k, 200+k, width=1)
       k += j
       j += 0.3
   ```
   - `create_oval()`实际绘制椭圆，通过设置相同的半径差形成正圆
   - 参数`(x0, y0, x1, y1)`表示左上角和右下角坐标
   - 圆心固定在`(250, 200)`，通过加减`k`控制半径
   - `k`每次增加`j`，`j`每次增加`0.3`，使圆圈增长差异逐渐变大，形成螺旋效果

4. **进入主事件循环**：
   ```python
   root.mainloop()
   ```
   老师解释：`mainloop()`让窗口保持打开状态，响应用户操作（如关闭窗口）。

5. **运行效果**：展示了完整的螺旋形圆圈图案，圆圈由小到大排列形成螺旋。
