## 三、方法 / 函数

### create_oval()

| 字段 | 内容 |
|------|------|
| **名称** | `Canvas.create_oval()` |
| **类型** | Canvas 实例方法 |
| **在本P中的作用** | 第45题：循环调用，通过递增的 `k` 值控制圆圈半径，绘制由小到大的螺旋形圆圈序列 |
| **来源** | 字幕直接提到 |
| **参数签名** | `create_oval(x0, y0, x1, y1, **options)`，其中 `(x0,y0)` 为边界框左上角，`(x1,y1)` 为右下角；`width` 控制线宽 |
| **易错点** | 参数为边界框坐标，而非圆心+半径；当 `x1-x0 == y1-y0` 时才绘制正圆 |

**建议方向**
- 修改 `j` 的增量（当前为 `0.3`）观察螺旋疏密变化
- 为每个圆圈添加随机颜色（`outline` 参数 + `random` 模块）
- 尝试将圆心坐标也设为变量，绘制偏心螺旋

## 五、语言机制 / 语法结构

### 字符串切片（Slice）

| 字段 | 内容 |
|------|------|
| **名称** | 字符串切片语法 `str[start:end:step]` |
| **类型** | Python 语言内置语法 |
| **在本P中的作用** | 第41题：`str[:d]` 截取头部、`str[d:]` 保留剩余、`str[-d:]` 截取尾部、`str[:-d]` 保留尾部剩余、`str[::-1]` 步长-1翻转字符串 |
| **来源** | 字幕直接提到 |
| **易错点** | `str[d:]` 是"从第d个字符开始到末尾"，而非"截取d个字符"；负索引方向为从右向左 |

**建议方向**
- 系统练习切片三参数的所有组合，尤其是负数步长
- 用切片实现回文字符串检测：`s == s[::-1]`
- 对比列表切片与字符串切片的行为异同

### raise 主动抛出异常

| 字段 | 内容 |
|------|------|
| **名称** | `raise` 语句 |
| **类型** | Python 语言异常机制 |
| **在本P中的作用** | 第44题：当年龄参数不在 0–120 范围时，使用 `raise Exception("年龄必须在0~120岁之间")` 主动抛出异常，由调用方的 `try-except` 捕获 |
| **来源** | 字幕直接提到 |
| **易错点** | `raise` 后的代码不会执行；异常未被捕获时程序终止；`Exception` 是通用异常类，建议场景复杂时自定义异常类 |

**建议方向**
- 尝试自定义异常类（继承 `Exception`），给错误类型命名
- 练习 `raise` 与 `raise ... from ...` 的区别
- 阅读 Python 异常类层级结构（`BaseException` → `Exception` → 具体子类）
