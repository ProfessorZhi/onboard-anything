## 知识骨架

```
字符串切片
├── 正索引切片：str[start:end]（包前不包后）
├── 负索引切片：str[-n:] 从尾部截取n个字符
├── 步长翻转：str[::-1] 步长-1实现逆序
└── 拼接：+ 连接符合并字符串

字典操作
├── 遍历键：for key in dict
├── 获取值：dict[key]
└── 新建键值对：新字典[值] = 键

异常处理
├── try：放可能报错的代码
├── except：捕获异常，处理错误
├── else：try成功执行后执行
└── raise：主动抛出异常

tkinter GUI
├── Canvas：绑定图形的画布组件
├── create_oval(x0,y0,x1,y1)：绘制椭圆/圆
└── mainloop()：进入事件循环，保持窗口
```

## 易错点

- **切片边界**：`str[d:]`截取头部d个字符，`str[:-d]`截取除尾部d个外的部分，容易混淆正负索引的含义
- **异常捕获粒度**：except后不应空写pass，应有具体处理逻辑；try块只放可能报错的代码
- **raise后代码执行**：未捕获异常时，raise之后的代码不会执行；捕获后才会继续
- **tkinter坐标**：create_oval参数是左上角和右下角坐标，而非圆心和半径
- **事件循环**：tkinter程序必须调用mainloop()才能保持窗口，否则绘制完直接退出

## 外部资源与对象

| 外部资源 | 类型 | 用途 | 关键方法/属性 |
|----------|------|------|---------------|
| **tkinter** | Python内置GUI库 | 创建图形界面 | import Tkinter |
| Canvas | tkinter组件类 | 绑定图形的画布 | Canvas(root, width, height) |
| create_oval | Canvas方法 | 绘制椭圆/圆 | create_oval(x0, y0, x1, y1) |
| mainloop | Tk对象方法 | 启动事件循环 | root.mainloop() |

tkinter是Python标准库，无需额外安装。Canvas是tkinter中最常用的绑定图形组件，create_oval通过坐标对定义边界框来绘制椭圆，当宽高相等时为正圆。mainloop是GUI程序的入口点，调用后程序进入事件监听状态。
