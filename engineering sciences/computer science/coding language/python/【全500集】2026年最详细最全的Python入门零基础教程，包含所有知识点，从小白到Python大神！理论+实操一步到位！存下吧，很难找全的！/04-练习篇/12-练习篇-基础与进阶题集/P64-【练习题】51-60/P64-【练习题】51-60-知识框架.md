# P64 知识框架 - 【练习题】51-60
## 本P位置
Python标准库学习路径中「常用模块」板块 → time模块专项练习  
前序：datetime模块、calendar模块  
后续：random模块、os模块基础  
定位：时间处理是编程中的高频需求，time模块是Python时间处理的底层接口

## 核心问题
如何在Python中获取、转换、测量和控制时间？

## 知识骨架

### 一、time模块三大核心类型

| 类型 | 说明 | 典型值示例 |
|------|------|-----------|
| 时间戳 | Epoch以来的秒数（浮点数） | 1717102524.0 |
| struct_time | 结构化时间对象（9元素元组） | time.struct_time(tm_year=2024, ...) |
| 时间字符串 | 人类可读格式 | "2025-04-02 23:40:00" |

### 二、time模块方法速查

**时间戳 ↔ 其他**

```
time.time()           → 获取当前时间戳
time.ctime(ts)        → 时间戳 → 本地时间字符串（默认当前）
time.localtime(ts)    → 时间戳 → 本地struct_time
time.gmtime(ts)       → 时间戳 → UTC struct_time
```

**struct_time ↔ 字符串**

```
time.asctime(st)      → struct_time → 固定格式字符串
time.strftime(fmt, st) → struct_time → 自定义格式字符串
time.strptime(s, fmt)  → 字符串 → struct_time
```

**struct_time → 时间戳**

```
time.mktime(st)       → struct_time → 时间戳
```

**控制**

```
time.sleep(N)         → 程序暂停N秒
time.perf_counter()   → 高精度单调时钟（适合短时测量）
```

### 三、格式占位符

| 占位符 | 含义 | 示例 |
|--------|------|------|
| %Y | 四位年份 | 2025 |
| %m | 月份（01-12） | 04 |
| %d | 日（01-31） | 02 |
| %H | 小时（00-23） | 23 |
| %M | 分钟（00-59） | 40 |
| %S | 秒（00-59） | 00 |

## 关系与依赖

```
时间字符串 ←——strptime—— struct_time ——mktime——→ 时间戳
     ↓                                      ↑
strftime                               localtime/gmtime
     ↓                                      ↑
自定义格式字符串 ←——ctime/asctime——←———————（默认当前）
```

**核心转换链路**：
- 字符串 → struct_time：strptime（需精确匹配格式）
- struct_time → 字符串：strftime
- struct_time → 时间戳：mktime
- 时间戳 → struct_time：localtime（本地）或 gmtime（UTC）
- 时间戳 → 字符串：ctime（固定格式）或 strftime

## 易错点

1. **%m vs %M**：`%m`是月份，`%M`是分钟，大小写不同含义完全不同
2. **%H vs %S**：`%H`是小时（24h制），`%S`是秒
3. **分隔符必须匹配**：strptime格式中的分隔符（-/空格）必须与输入字符串完全一致
4. **range(3000)范围**：产生0~2999，不是0~3000
5. **perf_counter vs time**：短时测量必须用perf_counter，time()受系统时钟调整影响
6. **UTC vs 本地时间**：gmtime返回UTC，localtime返回本地时间（国内UTC+8）
7. **sleep是阻塞**：sleep期间程序完全暂停，无法响应其他事件
8. **\r覆盖打印**：需配合end=""使用，实现行内数字刷新
