## 一、结构识别

**题目 1.1**：时间类型配对

Python的 `time` 模块中，时间数据有三种核心表示形式。请将左侧的类型名称与右侧的特征描述连线：

| 类型名称 | 特征描述 |
|----------|----------|
| A. 时间戳 | ① 9元素元组，含 tm_year、tm_mon 等字段 |
| B. struct_time | ② "2025-06-01 14:30:00" 格式的字符串 |
| C. 时间字符串 | ③ 从 Epoch（1970-01-01 00:00:00 UTC）起算的秒数（浮点数） |

## 二、机制理解

**题目 2.1**：格式占位符辨析

以下代码用于解析字符串 `"2025/12/25 08:30:45"`：

```python

s = "2025/12/25 08:30:45"
result = time.strptime(s, "%Y/%m/%d %H:%M:%S")
```

请判断：若将格式字符串改为 `"%Y-%m-%d %H:%M:%S"`，程序运行结果会怎样？选择正确答案：

- A. 能正常运行，结果相同
- B. 会抛出 ValueError 异常
- C. 能运行但结果中日期部分为默认值
- D. 警告但继续执行
