## 七、格式化占位符速查

> 来源：字幕直接提到，用于 `strptime` / `strftime` 的格式字符串参数

| 占位符 | 含义 | 易混淆点 |
|--------|------|---------|
| `%Y` | 四位年份 | 大写 Y |
| `%m` | 月份（01-12） | 小写 m，与 `%M` 严格区分 |
| `%d` | 日（01-31） | 小写 d |
| `%H` | 小时（00-23，24h 制） | 大写 H |
| `%M` | 分钟（00-59） | 大写 M，与 `%m` 严格区分 |
| `%S` | 秒（00-59） | 大写 S |

**建议方向**：制作一张个人速查卡片，覆盖上述六个占位符及其大小写规则；练习解析至少五种不同格式的真实时间字符串，强化格式匹配的肌肉记忆。
