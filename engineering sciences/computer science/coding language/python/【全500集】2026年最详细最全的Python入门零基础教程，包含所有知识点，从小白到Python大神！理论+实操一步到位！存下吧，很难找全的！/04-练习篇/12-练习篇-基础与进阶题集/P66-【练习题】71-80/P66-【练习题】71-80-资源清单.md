## 资源列表

### PyCharm

- **名称**：PyCharm
- **类型**：IDE（集成开发环境）
- **在本P中的作用**：视频演示练习题时使用的Python代码编写与运行环境，老师在此工具中演示BankAccount类和StudentGradeManager类的完整实现过程
- **来源**：字幕直接提到
- **建议方向**：熟练掌握PyCharm的调试功能，特别是对类方法设置断点调试，理解实例属性和方法的执行流程；利用PyCharm的代码补全功能提升OOP代码编写效率

## 内置功能与语法（本P涉及的Python核心要素）

以下为本P实际使用的Python内置功能与语法要素，虽非外部资源，但属于核心依赖：

| 要素 | 类型 | 在本P中的作用 |
|------|------|--------------|
| class | 语法关键字 | 定义BankAccount、Student、StudentGradeManager三个类 |
| __init__ | 特殊方法 | 初始化实例属性（owner、balance、students、name、grades） |
| self | 关键字 | 在类内部引用当前实例，访问实例属性 |
| dict | 内置类型 | StudentGradeManager中以学生姓名为键、学生对象为值存储数据 |
| list | 内置类型 | Student类中grades属性存储成绩序列 |
| append | 列表方法 | 向grades列表追加单个成绩 |
| len | 内置函数 | 计算成绩列表长度，用于求平均分时的除数 |
| sum | 内置函数 | 累加成绩列表所有元素，用于求平均分 |
| in / not in | 运算符 | 字典键的存在性检查（判断学生是否已存在） |
| and | 逻辑运算符 | 连接取款时的两个校验条件（正数且不超余额） |

## 资源关联说明

本P未引入任何第三方库、框架或外部平台，全部依赖Python标准库内置功能。PyCharm作为IDE提供了代码编辑、运行和调试的环境支撑。练习题的核心练习目标是面向对象编程的类设计与方法实现能力。
