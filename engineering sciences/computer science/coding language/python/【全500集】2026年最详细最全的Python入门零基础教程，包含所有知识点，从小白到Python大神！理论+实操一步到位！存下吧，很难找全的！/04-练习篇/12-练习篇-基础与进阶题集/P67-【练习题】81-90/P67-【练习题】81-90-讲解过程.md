# P67 讲解过程 - 【练习题】81-90
## 本P定位

本P为练习题系列第67集，依次完成第81至90题的讲解。视频内容覆盖四个实际文件处理任务：递归目录搜索找最大文件、课程成绩统计、多文件数据关联、批量文本合并。由于字幕在84题中途截断，以下内容仅还原第81至84题的前半部分讲解过程。

## 讲解过程

### 第81题：递归搜索目录找出最大文本文件

**需求复述**：编写程序搜索指定目录及其子目录下所有 `.txt` 文件，找出其中最大的文件及其大小。

**步骤一：导入OS模块并定义搜索路径**

教师首先指出本题需要使用 `os.walk()` 方法递归搜索目录。随后在代码中执行 `import os`，说明该模块专门用于文件和目录操作。定义变量 `search_dir` 存放待搜索路径，使用 `r` 前缀构造原始字符串以避免反斜杠转义问题。教师提示选择 `C:\` 路径进行演示，因为该路径下既有文件又有文件夹，适合展示递归搜索效果。

**步骤二：使用 os.walk() 遍历目录**

直接调用 `os.walk(search_dir)` 并打印返回值，说明其返回结果是一个生成器对象。教师随后用 `for` 循环遍历，解释该循环返回三个变量：`root` 表示当前遍历到的目录路径、`dirs` 表示当前目录下的子目录列表、`files` 表示当前目录下的文件列表。为了帮助学生直观理解，打印输出这三个变量，让学生看到完整的目录结构和文件信息。

**步骤三：提取所有文件并过滤 .txt 文件**

教师指出根据需求，只需要操作文件列表 `files`。使用 `for file in files` 遍历当前目录下的所有文件，随后调用字符串方法 `file.endswith('.txt')` 判断文件是否以 `.txt` 结尾，以此过滤出目标文件。

**步骤四：构造完整路径并获取文件大小**

教师提醒获取文件大小需要传入完整路径，而非仅文件名。使用 f-string 构造完整路径 `file_path = f"{root}\{file}"`，通过 `os.path.getsize(file_path)` 获取文件大小，默认单位为字节。教师演示将其转换为千字节：除以 1024 并存入变量 `file_size_kb`。

**步骤五：收集结果到列表并排序**

在循环开始前定义空列表 `result_list = []` 用于保存所有找到的文件路径与大小元组。当文件满足 `.txt` 条件时，将其路径和大小以元组形式 `(file_path, file_size_kb)` 添加到列表中。教师解释元组不可变的特性适合保存固定的数据对。

循环结束后，打印 `result_list` 查看结果，发现文件顺序杂乱。随后使用 `sorted()` 函数对该列表进行排序：参数 `key=lambda x: x[1]` 指定按元组下标为 1 的元素（大小）排序，`reverse=True` 设为倒序，实现从大到小排列。排序结果存入变量 `sorted_result`。

**步骤六：输出最大文件**

打印 `sorted_result[0]` 即可得到排序后位于首位的最大文件。教师强调下标为零才是第一个元素，运行后确认输出为 1625KB。

### 第82题：统计课程的最高分、最低分和平均分

**需求复述**：从 `course_student_grade.txt` 文件中读取课程和学生成绩信息，计算并打印每一门课程的最高分、最低分和平均分。

**步骤一：读取文件内容**

教师说明成绩文件位于当前目录下，使用相对路径 `with open('course_student_grade.txt', encoding='utf-8') as f:` 以只读模式打开文件，指定 UTF-8 编码以支持中文。

**步骤二：逐行解析数据**

使用 `for line in f.readlines()` 遍历文件每一行，返回一个列表。教师指出每一行末尾包含换行符，需要使用正向切片 `line[:-1]` 去除。随后观察到数据以逗号分隔，调用 `line.split(',')` 按逗号分割字符串，返回包含四个元素的列表。

**步骤三：解包数据到变量**

教师定义四个变量 `course, sid, name, grade` 接收分割后的四个值，通过解包特性简化代码。打印验证是否正确获取课程名、学号、姓名和分数。

**步骤四：建立课程与成绩的映射字典**

教师分析需求后指出，由于需要按课程分组统计，需要用字典建立课程与成绩列表之间的映射关系。首先定义空字典 `course_grades = {}` 用于存储映射。

随后在遍历循环中加入判断逻辑：`if course not in course_grades:` 判断当前课程是否已存在于字典键中，若不存在则初始化其值为空列表 `course_grades[course] = []`。教师特别强调，添加成绩的代码必须取消缩进放在 if 语句之外，因为无论课程是否已存在，都需要将当前学生成绩追加到对应列表中。

将 `grade` 转换为浮点数（`float(grade)`）后，调用 `course_grades[course].append()` 添加到对应的成绩列表。

**步骤五：遍历字典计算统计值**

循环结束后打印 `course_grades` 字典查看分组结果。随后新增 `for course, grades in course_grades.items():` 遍历键值对，对每个课程的成绩列表调用 `max(grades)` 求最高分、`min(grades)` 求最低分、`sum(grades) / len(grades)` 计算平均分。运行后输出各课程的统计数据：语文最高分94、最低分55、平均分73.2；数学最高分93、最低分53、平均分77.2。

### 第83题：不同文件的数据关联

**需求复述**：从 `course_teacher.txt` 和 `course_student_grade.txt` 两个文本文件中读取信息，打印包含课程名称、教师姓名、学生学号、学生姓名和学生成绩的完整信息。

**步骤一：读取课程-教师对应关系**

教师说明需要关联两个文件。首先打开 `course_teacher.txt`，设置 UTF-8 编码。用 `for line in f.readlines()` 遍历每一行，通过 `line[:-1]` 去除换行符，再用 `line.split(',')` 按逗号分割，得到课程名和教师名两个元素。

**步骤二：建立课程-教师字典**

在读取文件前定义空字典 `course_teacher = {}` 用于保存课程与教师的映射关系。分割后得到 `course` 和 `teacher` 两个变量，通过 `course_teacher[course] = teacher` 将其作为键值对存入字典。循环结束后打印该字典验证结果：语文对应于老师、数学对应树老师、英语对应尹老师。

**步骤三：读取学生成绩文件**

注释掉前一段读取代码后，开始读取 `course_student_grade.txt` 文件。同样设置 UTF-8 编码、用 `readlines()` 遍历每一行、切片去除换行符。分割后定义 `course, sid, name, grade` 四个变量接收值。

**步骤四：通过字典关联教师信息**

教师演示如何将两个文件的信息联动。使用 `course_teacher.get(course)` 方法根据当前遍历到的课程名从字典中获取对应的教师姓名。教师解释 `get()` 方法相比直接用 `course_teacher[course]` 更安全，因为当课程键不存在时前者返回 `None` 而后者会抛出 `KeyError` 导致程序终止。

**步骤五：打印完整信息**

将课程名、教师名、学号、姓名、分数一并打印输出：`print(course, teacher, sid, name, grade)`。运行后验证关联效果，例如输出"语文 于老师 101 小张 94"。

教师最后指出代码存在重复部分，建议可以将读取文件并解析的逻辑封装为函数，实现代码优化。

### 第84题：批量合并文本文件（字幕截断）

**需求复述**：编写程序将指定目录下所有 `.txt` 文件的内容合并到一个新文件中，每个文件内容之间用换行符分隔。

**步骤一：设置目标目录路径**

定义变量 `data_dir` 存放目标目录路径 `"data/_many_texts"`。

**步骤二：获取目录下的文件列表**

调用 `os.listdir(data_dir)` 获取指定目录下的所有文件和文件夹，返回一个列表。用 `for file in ...` 遍历该列表并打印，初步查看包含的文件：`001.txt`、`002.txt`、`003.txt`。

**步骤三：构造完整路径并筛选文件**

教师指出 `listdir()` 仅返回文件名，需要手动构造完整路径。使用 f-string `file_path = f"{data_dir}/{file}"` 构造完整路径，存入 `file_path` 变量后打印查看效果。

**步骤四：判断是否为 .txt 文件**

教师强调需要双重判断：既要确认是普通文件（非目录），又要确认文件扩展名为 `.txt`。通过 `os.path.isfile(file_path)` 判断是否为文件，通过 `file.endswith('.txt')` 判断扩展名。两个条件同时满足时才进行后续操作。

教师随后演示在目录中新建子目录或非 `.txt` 文件（如 `bingbing.ini`），验证代码能够正确过滤，只输出 `.txt` 文件。

## 本P结论

- **第81题结论**：通过 `os.walk()` 递归遍历目录，结合文件过滤、大小获取、列表排序三步操作，可以有效定位指定类型文件中的最值文件。
- **第82题结论**：利用字典存储课程与成绩列表的映射关系，配合 `max()`、`min()`、`sum()/len()` 等内置函数，能够高效完成分组统计任务。数据解析时应注意类型转换（字符串转浮点数）。
- **第83题结论**：多文件关联的核心在于建立统一键的映射字典。使用 `dict.get()` 方法比直接索引更安全，可避免键不存在时的程序崩溃。
- **第84题结论**：`os.listdir()` 返回纯文件名而非路径，需手动拼接完整路径后才能进行文件类型判断和读写操作。结合 `os.path.isfile()` 和字符串 `endswith()` 方法可准确筛选目标文件。
