**题目 2**：分析下面代码的执行结果，`grades` 字典的最终键值对数量是多少？`"语文"` 键对应的值是什么数据类型？

```python
grades = {}
data = [("语文", 85), ("数学", 92), ("语文", 78), ("英语", 88)]
for course, score in data:
    if course not in grades:
        grades[course] = []
    grades[course].append(score)
print(grades)
```
