## 核心问题

1. **递归目录中定位最值文件**：给定目录树，筛选特定类型文件并找最大者
2. **分组统计**：按类别（课程）聚合数据，计算组内统计量
3. **多文件数据关联**：以共同键（课程名）桥接不同数据源

## 易错点

1. **路径转义**：Windows路径用原始字符串 `r"C:\path"` 或双反斜杠
2. **目录 vs 文件**：`os.listdir()` 返回纯文件名，非完整路径，需手动拼接
3. **缩进位置**：字典操作中，append应放在if外部，确保新键初始化后也能追加
4. **类型转换**：文件读取返回字符串，数值统计前需 `float()` 转换
5. **安全索引**：键可能不存在时用 `dict.get()` 而非 `dict[key]`
6. **排序参数**：`key=lambda x: x[1]` 指定按元组第2元素排序
