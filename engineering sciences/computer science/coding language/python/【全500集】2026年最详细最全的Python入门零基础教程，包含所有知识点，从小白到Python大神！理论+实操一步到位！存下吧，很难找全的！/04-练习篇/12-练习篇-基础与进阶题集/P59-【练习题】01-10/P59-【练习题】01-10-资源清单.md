## 一、开发环境与工具

### 1. PyCharm

- **名称**：PyCharm
- **类型**：IDE（集成开发环境）
- **在本 P 中的作用**：全程代码编写、调试与运行的环境。所有练习题的代码演示均在 PyCharm 中完成，包括文件创建、代码编辑、运行验证等操作。
- **来源**：字幕直接提到
- **后续可练习方向**：建议自行安装 PyCharm Community 版，熟悉项目创建（New Project）、运行配置（Run）、调试断点（Debug）、代码格式化（Reformat Code）等基础操作

## 二、内置函数与语法

### 2. print()

- **名称**：print() 函数
- **类型**：内置函数 / 输出语句
- **在本 P 中的作用**：将变量或表达式的值输出到终端。本 P 重点演示了 `end` 参数对输出结束行为的控制：`end="\n"` 实现换行，`end="\t"` 实现制表符对齐（用于九九乘法表的列对齐）。
- **来源**：字幕直接提到
- **后续可练习方向**：尝试 `sep` 参数控制多个输出值之间的分隔符；探索将输出重定向到文件；练习自定义 `end` 实现不换行输出（如进度条效果）

### 3. input()

- **名称**：input() 函数
- **类型**：内置函数 / 输入语句
- **在本 P 中的作用**：接收用户从终端输入的内容，返回值类型始终为字符串（str）。本 P 在统计字符类型个数题目中首次引入，用于获取用户输入的一行字符。
- **来源**：字幕直接提到
- **后续可练习方向**：练习类型转换（int(input())、float(input())）；设计交互式小工具（如计算器、温度单位转换器）；探索 input() 与文件读取的结合

### 4. range()

- **名称**：range() 函数
- **类型**：内置函数 / 序列生成器
- **在本 P 中的作用**：生成连续的整数序列，用于 `for` 循环遍历。讲解了包前不包后的规则，以及通过负数步长实现倒序遍历（如 `range(9, 0, -1)`）。
- **来源**：字幕直接提到
- **后续可练习方向**：结合列表推导式生成复杂序列；使用 range 生成指定步长的等差数列；探索 range 在内存占用上的优势（惰性求值）

### 5. F-string

- **名称**：F-string（格式化字符串字面量）
- **类型**：Python 语法 / 字符串格式化方式
- **在本 P 中的作用**：在字符串前加 `f`，用花括号 `{}` 包裹变量实现字符串插值。本 P 在比较数字大小和统计三位数个数两道题中使用，替代传统的字符串拼接。
- **来源**：字幕直接提到
- **后续可练习方向**：在 F-string 中调用函数和表达式；学习 F-string 的格式说明符（如 `{value:.2f}` 保留两位小数）；对比 `.format()` 方法和 `%` 格式化

## 三、控制结构

### 6. if-elif-else

- **名称**：if-elif-else 多分支结构
- **类型**：控制流语句 / 条件分支
- **在本 P 中的作用**：处理三种及以上互斥条件的判断。本 P 多道题目均使用该结构，包括比较数字大小、筛选数字条件、统计字符类型等场景。强调 `else` 无需写条件，表示“上述均不满足”。
- **来源**：字幕直接提到
- **后续可练习方向**：嵌套 if-elif-else 处理更复杂的条件；结合布尔运算符（and/or/not）实现复合条件判断；设计多级菜单选择程序

### 7. for 循环

- **名称**：for 循环
- **类型**：控制流语句 / 循环结构
- **在本 P 中的作用**：遍历序列中的每个元素。本 P 展示了多种应用：遍历 range 生成的整数序列、遍历字符串的每个字符、三层嵌套循环遍历多维组合、正向遍历与倒序遍历。
- **来源**：字幕直接提到
- **后续可练习方向**：遍历列表、元组、字典等数据结构；结合 enumerate() 获取元素索引；使用 break/continue 控制循环流程；while 循环与 for 循环的场景对比

## 四、运算符

### 8. 赋值运算符（=）

- **名称**：赋值运算符（Assignment Operator）
- **类型**：运算符
- **在本 P 中的作用**：将右侧的值赋给左侧变量。反复强调 `=` 在 Python 中表示“赋值”而非数学中的“等于”，两者含义不同。
- **来源**：字幕直接提到
- **后续可练习方向**：链式赋值（a = b = 5）；增量赋值（+=、-=、*=、/=）；解包赋值（a, b = 1, 2）

### 9. 复合赋值运算符（+=）

- **名称**：复合赋值运算符 - 加法形式（+=）
- **类型**：运算符
- **在本 P 中的作用**：实现累加操作，等价于 `result = result + i`。本 P 在计算1~100整数和的题目中作为累加器使用，先定义 `result = 0`，循环体内执行 `result += i`。
- **来源**：字幕直接提到
- **后续可练习方向**：其他复合赋值形式（-=、*=、/=、//=、%=）；理解复合赋值在性能上的细微优势；练习累乘操作（result *= i）

### 10. 比较运算符

- **名称**：比较运算符（Comparison Operators）
- **类型**：运算符
- **在本 P 中的作用**：在条件判断中比较两个值的大小或相等性。本 P 涉及 `>`（大于）、`==`（等于）、`<`（小于）三种，并补充了 `<=`、`>=`、`!=`（不等于）。
- **来源**：字幕直接提到
- **后续可练习方向**：比较运算符的返回值是布尔值（True/False）；链式比较（a < b < c）；与布尔运算结合构建复杂条件

### 11. 算术运算符（+、**）

- **名称**：算术运算符 - 加法与幂运算
- **类型**：运算符
- **在本 P 中的作用**：`+` 用于变量相加（本 P 第1题）和字符串拼接；`**` 用于幂运算，计算立方和（本 P 水仙花数题目：i**3 + j**3 + k**3）。
- **来源**：字幕直接提到
- **后续可练习方向**：其他算术运算符（-、*、/、//、%）；运算符优先级；数学模块（math）中的三角函数、对数等高级运算

### 12. 逻辑运算符（and、!=）

- **名称**：逻辑运算符 - 与（and）、不等于（!=）
- **类型**：运算符
- **在本 P 中的作用**：`and` 连接多个条件，要求全部满足（如 `i != j and i != k and j != k`）；`!=` 判断两个值是否不相等。本 P 用于筛选互不相同的三位数。
- **来源**：字幕直接提到
- **后续可练习方向**：逻辑运算符的短路求值特性；or（或）、not（非）的使用；德摩根定律的实际应用

### 13. 数位提取运算符（//、%）

- **名称**：整除运算符（//）与取余运算符（%）
- **类型**：运算符
- **在本 P 中的作用**：通过整除和取余运算提取整数的各位数字。百位：`n // 100`；十位：`n // 10 % 10`；个位：`n % 10`。本 P 用于水仙花数的判断。
- **来源**：字幕直接提到
- **后续可练习方向**：处理任意位数数字的提取（如四位数、五位数）；进制转换（如十进制转二进制）；回文数判断（对称性验证）

- **名称**：整数类型（Integer）
- **类型**：数据类型

- **来源**：字幕直接提到
- **后续可练习方向**：字符串的切片与索引；常用方法（split()、strip()、replace()、find()）；字符串拼接的多种方式对比；中文字符串处理

## 六、经典编程问题与数学概念

### 16. 水仙花数

- **名称**：水仙花数（Narcissistic Number / Armstrong Number）
- **类型**：经典编程问题 / 数学概念
- **在本 P 中的作用**：作为数位提取的练习素材。一个三位数，其各位数字的立方和等于该数本身（如 153 = 1³ + 5³ + 3³）。本 P 遍历100~999找出所有水仙花数（153、370、371、407）。
- **来源**：字幕直接提到（作为练习题题材）
- **后续可练习方向**：扩展到任意位数（如四位数153 = 1⁴ + 5⁴ + 3⁴？）；寻找所有水仙花数并统计个数；设计函数封装水仙花数判断逻辑

### 17. 猴子吃桃问题

- **名称**：猴子吃桃问题
- **类型**：经典编程问题 / 数学问题
- **在本 P 中的作用**：体现逆向思维的数学建模问题。从第10天早上只剩1个桃子出发，使用倒序循环往前推算第1天的桃子总数（1534个）。核心公式：`previous = (current + 1) * 2`。
- **来源**：字幕直接提到（作为练习题题材）
- **后续可练习方向**：正向模拟验证结果；参数化问题（如第N天剩M个）；使用 while 循环实现；递归方法求解

### 18. 九九乘法表

- **名称**：九九乘法表
- **类型**：经典输出格式问题 / 循环练习题
- **在本 P 中的作用**：综合练习嵌套循环和 print 的 end 参数控制。核心要点：外层循环控制行数（1~9），内层循环控制列数（1~i），使用 `end="\t"` 实现列对齐，单独 `print()` 实现换行。
- **来源**：字幕直接提到（作为练习题题材）
- **后续可练习方向**：输出下三角形/上三角形变体；添加分隔线；格式化对齐美化；输出为字符串列表而非直接打印

### 19. 统计三位数问题

- **名称**：1~4组成的三位数统计
- **类型**：排列组合问题 / 穷举练习题
- **在本 P 中的作用**：使用三层嵌套循环遍历所有组合（111~444），结合逻辑运算符筛选互不相同且无重复数字的情况，统计并列出所有满足条件的三位数（24个）。
- **来源**：字幕直接提到（作为练习题题材）
- **后续可练习方向**：扩展到0~9十个数字；从组合变为排列（考虑顺序）；使用 itertools 模块简化；统计任意条件下的个数

### 20. 统计字符类型个数

- **名称**：统计字符类型个数
- **类型**：字符分类统计问题
- **在本 P 中的作用**：练习 input() 接收用户输入、for 循环遍历字符串、if-elif-else 多分支判断字符属于英文字母/数字/空格/其他。涉及 str 类型的理解。
- **来源**：字幕直接提到（作为练习题题材）
- **后续可练习方向**：使用字符分类方法（c.isdigit()、c.isalpha()、c.isspace()）；统计更多类型（如大写字母、小写字母）；大小写转换后统计

## 后续练习方向分类

**按技能维度归类建议方向：**

**语法深化方向**：F-string 高级用法（格式说明符）、else/elif 的更多用法（空分支 pass）、循环的 else 子句

**数据类型方向**：列表/元组/字典的基本操作、文件读写（操作文件作为输入输出）、JSON 数据处理

**算法思维方向**：回文数判断、质数判断与筛选、斐波那契数列生成、鸡兔同笼问题、逻辑推理问题

**综合项目方向**：命令行计算器、温度单位转换器、猜数字游戏、数字炸弹游戏、简单学生成绩管理系统
