# P59 练习题 - 【练习题】01-10
> 目标：掌握 Python 基础语法的综合运用，能够独立完成变量、循环、条件三类结构组合的经典问题求解。

---

## 一、结构识别

**1-1** 阅读以下代码，写出程序输出的前三个数字：

```python
for i in range(5, 16, 3):
    print(i)
```

**1-2** 指出下面代码中 `print` 的执行次数（假设 `n = 5`）：

```python
n = 5
for i in range(n):
    if i % 2 == 0:
        print(i)
```

**1-3** 下列哪个选项正确描述了 `range(1, 10)` 生成的元素？  
A. 1~9  
B. 1~10  
C. 0~9  
D. 0~10

---

## 二、机制理解

**2-1** 为什么下面代码的输出是 `True`，而不是 `5`？

```python
x = 5
print(x == 5)
```

**2-2** 用户输入 `123` 后，下面代码的输出是什么？说明原因。

```python
num = input("请输入一个数字：")
print(num + 5)
```

**2-3** 解释 `print("Hello", end=" ")` 和 `print("Hello")` 在输出效果上的区别。

**2-4** 对于三位数 `n = 456`，写出提取百位、十位、个位的表达式。

## 三、最小实践

**3-1** 编写程序：输出 1 到 20 的所有整数，每行一个。

**3-2** 编写程序：接收用户输入的一个整数，判断它是奇数还是偶数。

**3-3** 编写程序：计算并输出 `2¹⁰` 的值。

**3-4** 用 `for` 循环和 `range` 实现倒序输出 10 到 1。

## 四、扩展思考

**4-1**（水仙花数）  
打印所有三位水仙花数（各位数字的立方和等于它本身）。

**4-2**（猴子吃桃）  
猴子每天吃掉当天剩余桃子的一半加一个，第10天只剩1个。求第1天有多少桃子。

**4-3**（九九乘法表）  
用嵌套循环和 `print` 的 `end` 参数，输出九九乘法表（每行如 `1×1=1`）。
