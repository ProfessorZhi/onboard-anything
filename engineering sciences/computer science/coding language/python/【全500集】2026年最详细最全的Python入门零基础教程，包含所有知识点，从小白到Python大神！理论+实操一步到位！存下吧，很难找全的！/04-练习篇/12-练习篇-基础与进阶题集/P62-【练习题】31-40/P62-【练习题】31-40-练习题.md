## 一、结构识别

### 1.1 函数结构识别

以下代码片段缺少注释，请根据函数三要素（作用/参数/返回值）补全：

```python
def process_data(n):
    # ①
    if n <= 0:
        return None
    # ②
    result = 0
    for i in range(1, n + 1):
        result += i
    # ③
    return result
```

**题目**：在①、②、③处分别填写什么注释最合适？

A. ①输入校验 ②累加求和 ③返回结果  
B. ①参数定义 ②循环遍历 ③返回空值  
C. ①数据处理 ②条件判断 ③输出展示

### 1.2 循环步长识别

```python
for i in range(1, 10, 2):
```

A. 1,3,5,7,9 —— 步长为2的递增  
B. 1,3,5,7 —— 步长为2，不含终值

## 二、机制理解

### 2.1 递归终止条件

```python
def fibonacci(n):
    if n == 1 or n == 2:
        return 1
    return fibonacci(n-1) + fibonacci(n-2)
```

**题目**：如果删掉 `if n == 1 or n == 2` 这行，调用 `fibonacci(3)` 会发生什么？

A. 正常返回3  
B. 触发无限递归导致栈溢出  
C. 返回None

### 3.2 步长循环筛选

**要求**：编写函数 `sum_odd(start, end)`，计算从start到end（ inclusive）所有奇数的和。

**验收标准**：
- 使用 `range(start, end+1, 2)` 实现
- start为偶数时，从start+1开始
- 参数为负数或start>end时返回0

---

### 3.3 带校验的递归函数

**要求**：编写函数 `power(base, exp)`，计算base的exp次方，使用递归实现。

**验收标准**：
- exp为0时返回1
- exp小于0时返回None
- base为0且exp大于0时返回0

```python
def sum_scores(scores):
```

### 3.2 步长循环筛选
```python
def sum_odd(start, end):
    if start > end or start >= 0 and end < 0:
        return 0
    if start % 2 == 0:
        start += 1
    return sum(range(start, end + 1, 2))
```

### 3.3 带校验的递归函数
```python
def power(base, exp):
    if exp < 0:
        return None
    if exp == 0:
        return 1
    if base == 0:
        return 0
    return base * power(base, exp - 1)
```

### 4.1 递归改循环
```python
def countdown(n):
    while n > 0:
        print(n)
        n -= 1
```
- 递归优点：代码简洁，逻辑清晰
- 递归缺点：深度大时栈溢出
- 循环优点：无栈开销，适合大数据量
- 循环缺点：某些问题（如树遍历）不如递归直观

### 4.2 切片与月份累加
错误：`days[:month]` 包含第month个月
正确：`days[:month-1]` 前month-1个月
