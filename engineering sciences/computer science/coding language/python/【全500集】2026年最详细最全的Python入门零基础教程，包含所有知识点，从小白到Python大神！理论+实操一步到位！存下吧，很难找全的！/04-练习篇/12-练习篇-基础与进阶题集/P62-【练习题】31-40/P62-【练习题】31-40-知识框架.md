# P62 知识框架 - 【练习题】31-40
## 本P位置

Python入门教程 → 练习题章节（P61-P70系列）→ 综合性技能训练

本P是函数与算法基础的综合练习场，前接函数基础（参数、返回值），后承递归与复杂逻辑。10道题覆盖：字典聚合、循环控制、函数拆分、列表操作、递归思想。

## 核心问题

1. 如何规范定义一个函数（作用/参数/返回值三要素）
2. 如何遍历字典并聚合数据
3. 如何用步长精准控制循环范围
4. 如何用辅助函数拆分复杂逻辑
5. 如何用递归解决自相似问题
6. 如何处理日期相关计算（闰年判断）

## 知识骨架

```
函数定义
  ├─ 三要素：作用、参数、返回值
  ├─ 参数命名：避免与内置类冲突（dict → dict1）
  ├─ return None：提前结束 + 返回空值
  └─ 嵌套调用：主函数调用辅助函数

数据结构操作
  ├─ dict.values()：获取字典所有值
  ├─ 列表存储固定数据：每月天数
  ├─ 列表元素修改：days[1] = 29
  └─ 列表切片 + sum()：前N个月累加

循环控制
  ├─ range(start, stop, step)：步长控制奇偶
  ├─ 累加变量初始化：result = 0
  └─ result += value：聚合求和

递归算法
  ├─ 终止条件：return 1 或 return None
  ├─ 递归调用：return func(n-1) + func(n-2)
  └─ 栈展开：手动展开理解递归过程

逻辑判断
  ├─ 闰年：(n%4==0 and n%100!=0) or (n%400==0)
  ├─ 正整数校验：n <= 0 → return None
  └─ 条件分支外置相同逻辑：return 统一写
```

## 易错点

1. **参数命名冲突**：用`dict`命名参数会覆盖内置类，改用`dict1`或`_dict`
2. **递归无出口**：必须先写`n==1 or n==2`终止条件，否则无限递归
3. **闰年条件漏写**：`year%100!=0`容易漏掉，导致整百年判断错误
4. **切片索引**：`days[:month-1]`是前month-1个月，不是month个月
5. **负数未校验**：未判断`n<=0`会导致循环不执行或结果错误
6. **类型转换缺失**：`input()`返回字符串，必须`int()`转换才能参与运算
7. **列表修改时机**：闰年判断后修改月份列表要放在累加之前
