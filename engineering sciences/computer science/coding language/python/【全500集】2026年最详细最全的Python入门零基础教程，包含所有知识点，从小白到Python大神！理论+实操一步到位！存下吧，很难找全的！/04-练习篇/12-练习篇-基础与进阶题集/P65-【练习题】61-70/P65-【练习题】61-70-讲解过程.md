## 讲解过程

### 第61练：获取当前的日期和时间

**任务目标**：编写程序获取并显示当前的日期和时间。

**教学步骤**：

1. **导入datetime模块**。老师指出之前使用的是`time`模块，今天要介绍新的模块`datetime`，用于处理日期和时间相关的任务。

2. **获取当前日期时间**。使用`datetime.datetime.now()`方法获取本地当前日期时间，该方法返回包含年月日、时分秒和微秒的完整日期时间对象。

```python
import datetime
current_datetime = datetime.datetime.now()
print(current_datetime)
# 输出示例：2025年4月3日 13:33:52.123456
print(type(current_datetime))
# 输出：<class 'datetime.datetime'>
```

3. **格式化日期时间为字符串**。使用`strftime()`方法将日期时间对象转换为指定格式的字符串输出。该方法接收一个`format`参数，即格式化的标准。

老师演示了两种常用格式：
```python
# 格式一：使用冒号分隔时分秒
time_str = current_datetime.strftime("%Y-%m-%d %H:%M:%S")
# 输出：2025年4月3日 13:35:41

# 格式二：自定义分隔符
time_str = current_datetime.strftime("%Y/%m/%d")
```

讲解了常用格式化符号：`%Y`（4位数年份）、`%m`（月份）、`%d`（日期）、`%H`（小时，24小时制）、`%M`（分钟）、`%S`（秒）。

4. **分别获取年月日时分秒**。通过日期时间对象的属性直接访问各个组成部分。

```python
print("年份：", current_datetime.year)   # 2025
print("月份：", current_datetime.month)  # 4
print("日：", current_datetime.day)      # 3
print("小时：", current_datetime.hour)   # 13
print("分钟：", current_datetime.minute) # 37
print("秒：", current_datetime.second)   # 30
```

### 第62练：计算两个日期之间的天数差异

**任务目标**：计算两个日期之间的天数差异。

**教学步骤**：

1. **导入datetime模块**并定义出生日期。老师以“计算自己活了多少天”为例，将出生日期以字符串形式定义。

```python
import datetime
birthday = "1997-12-20"  # 字符串格式
```

2. **获取当前日期时间**。

```python
current_datetime = datetime.datetime.now()
print(current_datetime)
# 输出：2025年4月3日 13:43:54
```

3. **将日期字符串转换为datetime对象**。老师指出此时`birthday`是字符串类型，`current_datetime`是datetime类型，两者格式不匹配，无法直接计算。需要使用`datetime.datetime.strptime()`方法将字符串转换为datetime对象。

```python
birthday_date = datetime.datetime.strptime(birthday, "%Y-%m-%d")
print(birthday_date)
# 输出：1997年12月20日 00:00:00
```

`strptime()`方法有两个参数：第一个是日期时间字符串，第二个是格式。

4. **计算时间差**。直接用两个datetime对象相减，会自动计算时间差，返回一个`timedelta`对象。

```python
time_difference = current_datetime - birthday_date
print(time_difference)
# 输出：9966天, 0:00:00
print(time_difference.days)
# 输出：9966
```

5. **估算年龄**。老师说明由于存在闰年等因素，年龄只能粗略估算，将天数除以365。

```python
estimated_age = time_difference.days / 365
print(estimated_age)
# 输出：27（大约）
```

### 第63练：简单的日期处理

**任务目标**：使用datetime模块完成四项任务：①输出今日日期；②创建并输出宫崎骏的生日日期；③计算并输出宫崎骏生日的下一天；④计算并输出宫崎骏的一岁生日日期。

**教学步骤**：

1. **导入datetime模块**。

2. **输出今日日期**。使用`datetime.datetime.now()`获取当前日期时间，然后调用`strftime()`方法按照指定格式输出年月日。

```python
import datetime
today = datetime.datetime.now().strftime("%Y-%m-%d")
print(today)
# 输出：2025年4月3日
```

3. **创建宫崎骏的生日日期对象**。使用`datetime.date()`方法创建日期对象，该方法接收三个参数：year（年份）、month（月份）、day（日期）。

```python
miyazaki_birthday = datetime.date(1941, 1, 5)
print(miyazaki_birthday)
# 输出：1941-01-05
# 也可以使用strftime自定义格式
print(miyazaki_birthday.strftime("%Y/%m/%d"))
# 输出：1941/01/05
```

4. **计算生日的下一天（日期加法运算）**。使用`timedelta`时间差对象进行日期算术运算。`timedelta`可以传入参数`days`（天数）、`seconds`（秒数）、`minutes`（分钟数）、`weeks`（周数）等。

```python
next_day = miyazaki_birthday + datetime.timedelta(days=1)
print(next_day)
# 输出：1941-01-06
```

5. **计算一岁生日日期（使用replace方法）**。使用日期对象的`replace()`方法可以替换日期中的年份、月份或日期等部分，返回一个新的日期对象。

```python
first_birthday = miyazaki_birthday.replace(year=miyazaki_birthday.year + 1)
print(first_birthday)
# 输出：1942-01-05
```

老师指出`replace()`方法包含三个参数：year（年份）、month（月份）、day（日期），可以根据需要替换其中的部分。

### 第64练：日期减法练习

**任务目标**：定义一个函数，接收两个参数：遵循"YYYY-MM-DD"格式的日期字符串`p_date`以及一个整数`days`，计算从给定日期向前回溯指定天数后的新日期。

**教学步骤**：

1. **导入datetime模块**。

2. **定义函数框架**。老师先不考虑返回值，将函数框架写出来。

```python
import datetime

def calculate_previous_date(p_date, days):
    # 待实现
```

3. **将日期字符串转换为datetime对象**。使用`strptime()`方法，格式为"%Y-%m-%d"。

```python
p_date_obj = datetime.datetime.strptime(p_date, "%Y-%m-%d")
print(p_date_obj)
# 例如输入"2025-05-20"输出：2025-05-20 00:00:00
```

4. **创建timedelta时间差对象**。创建一个`days`参数的时间差对象，用于后续减法运算。

```python
time_gap = datetime.timedelta(days=days)
print(time_gap)
# 例如days=8时输出：8 days, 0:00:00
```

5. **计算新日期（日期减法运算）**。用原日期减去时间差，得到回溯后的新日期。

```python
p_date_result = p_date_obj - time_gap
```

6. **格式化输出并返回**。使用`strftime()`方法按指定格式（"%Y-%m-%d"）输出，并作为函数返回值。

```python
return p_date_result.strftime("%Y-%m-%d")
```

完整函数：
```python
def calculate_previous_date(p_date, days):
    p_date_obj = datetime.datetime.strptime(p_date, "%Y-%m-%d")
    time_gap = datetime.timedelta(days=days)
    p_date_result = p_date_obj - time_gap
    return p_date_result.strftime("%Y-%m-%d")
```

7. **测试函数**。老师演示了多组测试用例，验证函数正确性。

```python
print(calculate_previous_date("2025-05-20", 8))
# 输出：2025-05-12

print(calculate_previous_date("2025-04-01", 1))
# 输出：2025-03-31

print(calculate_previous_date("2025-04-01", 2))
# 输出：2025-03-30

print(calculate_previous_date("2025-04-01", 12))
# 输出：2025-03-20
```

老师强调循环条件中使用`<=`而不是`<`，因为题目要求包括两端日期在内。

### 第65练：生成指定日期范围内的所有日期列表

**任务目标**：定义一个函数，接收两个日期字符串参数`begin_date`和`end_date`，生成包含从起始日期到结束日期（包括两端）的所有日期的列表并返回。

**教学步骤**：

1. **导入datetime模块**。

2. **定义函数框架**，明确函数的三大要素：作用（生成日期范围列表）、参数（起始日期和结束日期两个字符串）、返回值（日期字符串列表）。

```python
def get_date_range(begin_date, end_date):
    # 待实现
```

3. **将日期字符串转换为datetime对象**。使用`strptime()`方法，格式为"%Y-%m-%d"。

```python
begin_date_obj = datetime.datetime.strptime(begin_date, "%Y-%m-%d")
end_date_obj = datetime.datetime.strptime(end_date, "%Y-%m-%d")
```

4. **初始化当前日期变量**。定义一个变量保存当前正在处理的日期，初始值为起始日期。

```python
current_date = begin_date_obj
```

5. **使用while循环生成日期列表**。当当前日期小于等于结束日期时，将当前日期转换为字符串并添加到列表，然后当前日期加一天。

```python
date_list = []
while current_date <= end_date_obj:
    date_str = current_date.strftime("%Y-%m-%d")
    date_list.append(date_str)
    current_date = current_date + datetime.timedelta(days=1)
```

老师强调循环条件中要使用`<=`（包括等于），因为题目要求包括两端日期。

6. **返回列表**。

```python
return date_list
```

完整函数：
```python
def get_date_range(begin_date, end_date):
    begin_date_obj = datetime.datetime.strptime(begin_date, "%Y-%m-%d")
    end_date_obj = datetime.datetime.strptime(end_date, "%Y-%m-%d")
    current_date = begin_date_obj
    date_list = []
    while current_date <= end_date_obj:
        date_str = current_date.strftime("%Y-%m-%d")
        date_list.append(date_str)
        current_date = current_date + datetime.timedelta(days=1)
    return date_list
```
