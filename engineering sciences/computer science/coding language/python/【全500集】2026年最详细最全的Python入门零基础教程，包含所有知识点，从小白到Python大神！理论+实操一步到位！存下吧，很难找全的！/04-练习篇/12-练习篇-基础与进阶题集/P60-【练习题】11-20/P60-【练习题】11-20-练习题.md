**题目 1.2：循环结构与兜底逻辑**

以下代码运行后，输出的结果是什么？

```python
nums = [1, 3, 5]
target = 4

for i, n in enumerate(nums):
    if n == target:
        print(f"找到了，索引是 {i}")
        break
else:
    print("未找到，已遍历全部")

print("程序结束")
```

- A. 找到了，索引是 1
- B. 未找到，已遍历全部\n程序结束
