## 易错点

1. **整除混淆**：`3//2`和`2//2`都是1，无法区分奇偶，必须用`%`取模
2. **字典遍历遗漏**：`for name, age in dict` 只取键，必须用`dict.items()`才能同时拿到键值对
3. **有序插入末尾遗漏**：循环正常结束（未break）时，待插入数可能比所有元素都大，需要`append()`兜底
4. **浮点型陷阱**：第14题若用`int()`转用户输入则无法处理小数，用`float()`更通用
5. **死循环必须配break**：第15、16题的`while True`若无`break`则永不退出
6. **序列赋值原理**：交换值时`num1, num2 = num2, num1`右边先算再同时赋值，不能颠倒顺序
