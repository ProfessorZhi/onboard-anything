## 一、Python标准库与内置对象

**math模块**
- 类型：标准库模块
- 在本P中的作用：提供高精度圆周率 math.pi，用于第23练圆的面积计算，比硬编码3.14精度更高
- 来源：字幕直接提到
- 建议方向：探索 math 模块中的三角函数（sin、cos）、开方（sqrt）、对数（log）等常用函数，以及 math 模块其他数学常量如 math.e

**math.pi**
- 类型：数学常量（math模块成员）
- 在本P中的作用：存储高精度圆周率值（3.141592653589793），在 area_of_circle 函数中作为面积计算基准
- 来源：字幕直接提到
- 建议方向：对比不同精度圆周率对面积计算结果的影响，理解浮点数精度问题

**round()函数**
- 类型：Python内置函数
- 在本P中的作用：将浮点数保留指定小数位数，第23练中用于将圆的面积结果保留两位小数
- 来源：字幕直接提到
- 建议方向：学习其他精度控制方式，如格式化字符串（f"{value:.2f}"）以及 Decimal 模块的高精度计算

**int()函数**
- 类型：Python内置函数
- 在本P中的作用：将字符串或浮点数转换为整数，处理用户通过 input() 输入的数字
- 来源：字幕直接提到
- 建议方向：结合异常处理（try-except）进行输入验证，防止用户输入非数字字符导致程序崩溃

**float()函数**
- 类型：Python内置函数
- 在本P中的作用：将字符串或整数转换为浮点数，支持用户输入半径为小数的情况
- 来源：字幕直接提到
- 建议方向：处理输入中的边界情况，如科学计数法表示的数字

**range()函数**
- 类型：Python内置函数
- 在本P中的作用：生成整数序列，用于循环遍历，第22练阶乘、第24练素数判断、第25练平方和等场景均使用
- 来源：字幕直接提到
- 建议方向：结合列表推导式使用，以及探索 range 的步长参数（range(start, end, step)）

## 二、语法模式与控制结构

**while True + break 循环模式**
- 类型：控制流模式
- 在本P中的作用：第20练实现"持续输入直到满足退出条件"的场景，适用于循环次数不确定但结束条件明确的情况
- 来源：字幕直接提到
- 建议方向：实现菜单系统、多轮问答程序、游戏主循环等需要持续运行的场景

**累加模式 (result = 0, result += x)**
- 类型：算法模式
- 在本P中的作用：第21练连续数字串和、第25练平方和等场景，用于将循环过程中的值累积求和
- 来源：字幕直接提到
- 建议方向：实现累加求平均、统计计数、累加生成列表等扩展应用

**累乘模式 (result = 1, result *= x)**
- 类型：算法模式
- 在本P中的作用：第22练阶乘计算，需要将初始值设为1而非0（否则乘积永远为0），用于累积求积
- 来源：字幕直接提到
- 建议方向：实现组合数计算（C(n,m)）、排列数、指数运算等涉及乘积的场景

**数字串生成公式 (current_term = current_term * 10 + a)**
- 类型：算法公式
- 在本P中的作用：第21练生成连续重复数字串（AA、AAA、AAAA...），通过乘10加A的迭代生成每一项
- 来源：字幕直接提到
- 建议方向：实现数字反转、进制转换、提取数字各位等变体应用

**函数拆分思想**
- 类型：设计模式
- 在本P中的作用：第24练将"素数判断"与"范围打印"拆分为 is_prime 和 print_primes 两个独立函数，提高代码复用性
- 来源：字幕直接提到
- 建议方向：实践更大规模程序的模块化拆分，理解单一职责原则的应用
