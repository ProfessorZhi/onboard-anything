# P61 练习题 - 【练习题】21-30
## 一、结构识别

**题目1：函数定义的三要素是什么？它们分别回答什么问题？**

> 提示：想一想做饭的流程——做什么（作用）、用什么食材（参数）、做完后端上什么（返回值）。

---

**题目2：累加和累乘的初始值分别应该设成什么？为什么？初始值设错会导致什么后果？**

> 提示：如果计算 1×2×3 的积，把初始值设为 0 会怎样？

---

## 二、机制理解

**题目3：什么情况下用 while True + break 比 for + range 更合适？请举一个本P中出现的需求例子。**

> 提示：本P中有一个练需要用户反复输入，直到满足某个条件才停止。

---

**题目4：请用手算验证"AA"型数字串的生成过程：如果 A=7，"77"是如何通过递推产生的？**

```
初始：current_term = 0
第1轮：current_term = 0 × 10 + 7 = ___
第2轮：current_term = ___ × 10 + 7 = ___
```

> 提示：每一步乘10的作用是什么？

---

## 三、最小实践

**题目5：编写函数 is_prime(x)，判断一个数是否为素数，返回布尔值。**

要求：
- x 为正整数
- 素数定义：大于1，且只能被1和自身整除

---

**题目6：设计并实现一个函数，要求：**

| 要素 | 内容 |
|------|------|
| 作用 | 计算圆的面积 |
| 参数 | radius（半径），单位：米 |
| 返回值 | 面积（平方米），保留两位小数 |

> 提示：面积公式 = πr²，圆周率建议用什么？

---

## 四、扩展思考

**题目7：为什么要把"素数判断"和"素数打印"拆成两个独立的函数？这种拆分方式还能应用在什么场景？**

> 提示：考虑复用性、测试便利性，以及"判断逻辑"与"业务逻辑"分离的思想。

## 五、最小替代练习

**题目8：如果不允许使用 math.pi 和 round()，如何完成圆的面积计算并保留两位小数？**

> 提示：可以尝试自定义 π 的近似值，以及用整数运算模拟四舍五入。
