# P51 练习题 - 【进阶篇】生成器
## 结构识别

**1. 括号识错**

以下代码执行后，`g` 是什么类型？

```python
g = (x * 2 for x in range(5))
print(type(g))
```

- A. tuple
- B. list
- C. generator
- D. int

---

**2. 关键词识别**

以下哪个函数是生成器函数？

```python
# 选项 A
def func_a():
    return [1, 2, 3]

# 选项 B
def func_b():
    for i in range(3):
        return i

# 选项 C
def func_c():
    yield 1
    yield 2
    yield 3

# 选项 D
def func_d(items):
    result = []
    for i in items:
        result.append(i)
    return result
```

---

**3. 包含关系**

判断正误：

- 生成器一定是迭代器，迭代器一定是可迭代对象
- 列表是迭代器
- 生成器可以用 `for` 循环遍历

## 机制理解

**4. 调用时机**

```python
def gen():
    print("开始执行")
    yield 1
    print("继续执行")
    yield 2

g = gen()
print("已创建生成器")
result = next(g)
print(f"得到: {result}")
```

程序输出顺序是什么？

---

**5. 耗尽行为**

```python
def gen():
    yield "A"
    yield "B"

g = gen()
print(list(g))  # 输出1
print(list(g))  # 输出2
```

输出2的结果是什么？为什么？

---

**6. yield 与 return 对比**

```python
def gen_with_return():
    yield 1
    return "结束"
    yield 2

g = gen_with_return()
print(next(g))
print(next(g))
```

最后一行会输出什么？异常类型是什么？

**9. next 与默认值**

使用 `next()` 的第二个参数（默认值）重写以下代码，避免 `StopIteration` 异常：

```python
def numbers():
    yield 1
    yield 2

gen = numbers()
print(next(gen))
print(next(gen))
print(next(gen))  # 防止报错
```

## 参考答案要点

**结构识别**

2. C — 只有 `func_c` 包含 `yield` 关键字
