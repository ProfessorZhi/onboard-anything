**题目 3.3** 易错排查

以下迭代器实现有一个 bug，会导致少返回一个元素。请找出并修正：

```text
class WrongIterator:
    def __init__(self, data):
        self.data = data
        self.index = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.index >= len(self.data):
            raise StopIteration
        self.index += 1            # ❌ 先移动指针
        return self.data[self.index - 1]

it = WrongIterator([1, 2, 3])

```
