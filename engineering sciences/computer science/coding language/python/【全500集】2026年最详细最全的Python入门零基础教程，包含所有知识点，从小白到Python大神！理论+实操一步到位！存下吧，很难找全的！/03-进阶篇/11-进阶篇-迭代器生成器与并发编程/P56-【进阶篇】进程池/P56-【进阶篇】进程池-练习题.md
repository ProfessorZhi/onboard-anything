**Q4｜易错点诊断**

下面的代码片段各有一个潜在问题，请指出问题所在并给出修正方案：

**片段 A**
```python
pool = Pool(4)
result = pool.apply_async(task, args=(1))
pool.join()
pool.close()
```

**片段 B**
```python
from multiprocessing import Pool

def task(x):
    return x * x

pool = Pool(4)
for i in range(8):
    pool.apply_async(task, args=(i,))
pool.close()
pool.join()
```

> 片段 B 的任务确实被提交并执行了，但代码存在一个设计上的遗漏，是什么？
