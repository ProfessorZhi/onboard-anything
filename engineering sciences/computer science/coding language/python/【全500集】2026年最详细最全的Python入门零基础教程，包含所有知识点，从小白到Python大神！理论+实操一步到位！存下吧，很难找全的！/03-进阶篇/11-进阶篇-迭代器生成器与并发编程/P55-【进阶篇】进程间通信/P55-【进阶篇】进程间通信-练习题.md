**Q6：join() 的作用边界**

有同学写了如下调用顺序，认为"先让生产者跑完再让消费者跑"：

```python
p1.start()
p1.join()   # 等生产者结束
p2.start()
p2.join()   # 等消费者结束
```

与下面这种写法相比：

```python
p1.start()
p2.start()
p1.join()
p2.join()
```

请回答：
1. 两种写法的行为差异是什么？
