# P49 练习题 - 【进阶篇】魔术方法
## 结构识别

**1. 下列哪些是合法的魔术属性或魔术方法？**

```
A. __name__
B. __init__
C. __call__
D. __module__
E. __main__
F. __doc__
```

> 参考答案：A、B、C、D、F。`__main__`是模块名，不是魔术方法（虽然也是双下划线格式）。

---

**2. 判断正误：所有以双下划线开头和结尾的名称都是魔术方法或魔术属性。**

> 参考答案：错误。`__name__`、`__main__`等是标识符，不属于类专属的魔术成员。魔术方法/属性特指Python为类预留的、解释器会特殊处理的成员。

---

## 机制理解

**3. 填空：对象销毁时自动触发的方法是 _______，必须返回字符串的方法是 _______，使实例可以像函数一样被调用的方法是 _______。**

> 参考答案：`__del__`、`__str__`、`__call__`

---

**4. 简答：在主程序文件中执行以下代码，输出的`__module__`值是什么？为什么？**

```python
class Demo:
    pass

print(Demo.__module__)
```

> 参考答案：输出`__main__`。当模块作为主程序直接执行时，该模块的`__module__`属性为`__main__`，而非实际的文件模块名。

---

**5. 单选：`callable(obj)`返回`True`时，可以确定的是：**

```
A. obj是一个函数
B. obj是一个类
C. obj是实例且实现了__call__方法
D. obj一定是可调用的
```

> 参考答案：D。`callable()`检查对象是否可以带有()语法调用。函数、类、实现了`__call__`的实例都返回`True`，但不能区分具体类型。

```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        # 补全这里
        pass

p = Point(3, 4)
print(p)  # 期望输出: Point(x=3, y=4)
```

> 参考答案：

## 扩展思考

**9. 场景讨论：什么情况下你会让一个类的实例变得可调用（实现`__call__`）？请举出一个实际应用场景。**

> 参考参考答案：
> - **状态携带的函数**：需要在多次调用间保持状态，例如计数器、缓存参数
> - **替代闭包**：类实例比嵌套函数更灵活，可以定义多个方法
> - **装饰器类**：需要保存配置参数的装饰器可用`__call__`实现
> - **API封装**：让对象实例直接可调用，简化调用语法

---

**10. 逆向思考：`__del__`方法有哪些局限性？为什么Python官方文档建议不要在`__del__`中执行关键清理操作？**

> 参考参考答案：
> - 触发时机不确定，依赖垃圾回收器何时运行
> - 循环引用可能导致对象永远不被销毁，`__del__`不执行
> - 全局变量销毁顺序不确定，可能访问已删除对象
> - 更好的做法：使用`contextmanager`或显式`close()`方法

## 最小替代练习

**11. 实践任务：不用`callable()`函数，判断一个对象是否可调用。**

```python
def is_callable(obj):
    # 不使用callable()，通过其他方式判断
    pass

class Test:
    def __call__(self):
        return "called"

print(is_callable(Test()))  # 期望: True
print(is_callable("hello")) # 期望: False
print(is_callable(is_callable))  # 期望: True
```

> 参考参考答案：
> ```python
> def is_callable(obj):
>     return hasattr(obj, '__call__')
> ```
> 说明：`callable()`本质上就是检查对象是否具有`__call__`属性。这道题帮助理解`callable()`的底层实现。

**12. 综合练习：实现一个简单的「可调用计数器」类。**

要求：
- 初始化时设置初始值`start=0`
- 每次调用返回当前计数值，然后递增
- 打印实例时显示格式`Counter(value=N)`

```python
class Counter:
    # 你的实现

c = Counter()
print(c)  # Counter(value=0)
print(c()) # 0
print(c()) # 1
print(c()) # 2
print(c)  # Counter(value=3)
```

> 参考参考答案：
> ```python
> class Counter:
>     def __init__(self, start=0):
>         self.value = start
    
>     def __str__(self):
>         return f"Counter(value={self.value})"
    
>     def __call__(self):
>         current = self.value
>         self.value += 1
>         return current
> ```
