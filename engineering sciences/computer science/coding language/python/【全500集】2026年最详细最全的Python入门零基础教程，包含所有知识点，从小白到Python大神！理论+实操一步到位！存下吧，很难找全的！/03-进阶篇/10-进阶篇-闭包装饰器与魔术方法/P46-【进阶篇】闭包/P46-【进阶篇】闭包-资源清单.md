## 四、代码示例

### 5. 基础版闭包代码

- **名称**：基础版闭包代码（奶茶甜度示例）
- **类型**：代码示例
- **在本 P 中的作用**：演示闭包的最基本形式，包含闭包形成的三个必要条件：函数嵌套、引用外层局部变量、返回内层函数引用
- **来源**：字幕直接提到
- **后续可练习方向**：自己默写这段代码，尝试修改外层函数的局部变量值（如把"sugar"改为"五分糖"），观察闭包的"记忆"效果

```python
def outer():
    sugar = "三分糖"
    def inner():
        print(f"制作一杯{sugar}的奶茶")
    return inner

make_milk_tea = outer()
make_milk_tea()
```

### 6. 升级版闭包代码

- **名称**：升级版闭包代码（带参数版本）
- **类型**：代码示例
- **在本 P 中的作用**：演示如何通过外层函数参数实现动态记忆，展示闭包的灵活性——多个闭包各自独立，互不影响
- **来源**：字幕直接提到
- **后续可练习方向**：基于这个模式创建自己的闭包，如记忆用户配置、缓存计算结果等实际应用场景

```python
def outer(base_num):
    def inner(add_num):
        print(f"{base_num} + {add_num} = {base_num + add_num}")
    return inner

add = outer(10)
add(3)    # 输出 10 + 3 = 13
add15 = outer(15)
add15(3)  # 输出 15 + 3 = 18
```

### 7. 嵌套函数代码示例

- **名称**：嵌套函数代码示例
- **类型**：代码示例
- **在本 P 中的作用**：演示闭包的前置知识——嵌套函数的定义和调用规则，通过演示内层函数无法在外层外部调用来强调闭包的必要性
- **来源**：字幕直接提到
- **后续可练习方向**：编写更多嵌套函数示例，理解内层函数的作用域限制，为理解闭包的空间类比（大房间套小房间）打下基础

```python
def outer():
    print("我是外层函数")
    def inner():

    inner()
outer()
```
