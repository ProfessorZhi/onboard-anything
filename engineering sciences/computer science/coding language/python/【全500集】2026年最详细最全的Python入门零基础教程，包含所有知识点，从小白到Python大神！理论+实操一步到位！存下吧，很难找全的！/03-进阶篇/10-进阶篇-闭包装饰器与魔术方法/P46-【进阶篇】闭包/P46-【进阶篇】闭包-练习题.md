# P46 练习题 - 【进阶篇】闭包
## 结构识别

以下代码是否满足闭包的三条件？请逐一说明。

```python
def outer():
    x = 10
    def inner():
        return x
    return inner
```

## 最小实践（参考答案）

```python
def multiplier(n):
    def calc(x):
        return x * n
    return calc
```

关键点：内层函数 `calc` 引用了外层参数 `n`，满足闭包三条件。`multiplier(2)` 返回的是函数引用而非函数执行结果。
