### functools 模块

- **类型**：Python 内置标准库
- **在本P中的作用**：提供了 wraps 装饰器等工具函数，用于解决装饰器带来的函数元信息丢失问题（__name__、__doc__ 等）
- **来源**：基于上下文推断（装饰器标准用法中常需配合 functools.wraps）
- **后续可练习方向**：在自定义装饰器中导入并使用 @functools.wraps(func)，对比使用前后的 __name__、__doc__ 属性变化

### functools.wraps 装饰器

- **类型**：Python 内置装饰器
- **在本P中的作用**：将原函数的元信息（函数名、文档字符串、参数签名等）复制到 wrapper 函数上，避免调试和 introspection 时丢失关键信息
- **来源**：基于上下文推断（课程易错点中提到装饰器会丢失元信息，wraps 是标准解决方案）
- **后续可练习方向**：为 P47 中的 login_test 装饰器添加 @functools.wraps(func)，验证 inspect.signature() 等工具能否正确读取原函数签名

## 学习辅助工具

### inspect 模块

- **类型**：Python 内置标准库
- **在本P中的作用**：用于查看函数的签名、源代码、参数信息等，配合装饰器可验证 wrapper 是否正确保留了原函数元信息
- **来源**：基于上下文推断（装饰器元信息保留问题的自然延伸工具）
- **后续可练习方向**：使用 inspect.signature()、inspect.getsource() 查看被装饰函数的详细信息，验证 functools.wraps 的效果

### Python Tutor（在线执行可视化）

- **类型**：在线可视化工具
- **在本P中的作用**：逐步展示装饰器的调用流程、闭包变量的捕获过程，帮助理解装饰器从调用到返回的完整执行链路
- **来源**：基于上下文推断（装饰器执行流程较抽象，可视化工具有助于理解）
- **后续可练习方向**：在 Python Tutor 中逐步执行装饰器代码，观察 wrapper 闭包如何持有原函数引用
