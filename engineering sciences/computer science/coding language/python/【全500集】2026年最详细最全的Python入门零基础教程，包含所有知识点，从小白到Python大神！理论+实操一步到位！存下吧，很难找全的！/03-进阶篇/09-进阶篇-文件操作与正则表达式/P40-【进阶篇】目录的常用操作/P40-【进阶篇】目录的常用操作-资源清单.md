# P40 资源清单 - 【进阶篇】目录的常用操作
## Python 内置模块 / 方法

### os 模块
- **类型**：Python 标准库模块（内置）
- **在本 P 中的作用**：提供文件和目录管理的所有核心方法，是本 P 所有操作的基础。需 `import os` 后才能使用。
- **来源**：字幕直接提到
- **后续建议方向**：结合 shutil 模块学习更高级的文件复制、移动操作；探索 pathlib 模块作为路径操作的现代化替代方案

### os.rename(原路径, 目标路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：核心文件操作方法，支持重命名文件或同磁盘内移动文件。本质是通过修改路径实现移动。
- **来源**：字幕直接提到
- **后续建议方向**：批量重命名一个文件夹下的所有文件（如添加前缀、时间戳）；结合 os.listdir 遍历实现文件自动归类

### os.remove(文件路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：删除指定路径的文件，只能删除文件而非目录。删除后不进入回收站，不可逆。
- **来源**：字幕直接提到
- **后续建议方向**：结合 os.listdir 过滤特定类型文件后批量清理；实现日志文件自动清理脚本（按日期删除过期日志）

### os.mkdir(目录路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：创建单个空目录，若父目录不存在会报错，不能创建已存在的目录。
- **来源**：字幕直接提到
- **后续建议方向**：根据项目需求自动创建标准目录结构；批量创建多个互不相关的文件夹

### os.rmdir(目录路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：删除指定的空目录，目录内有文件或子目录时会报错。删除不可逆。
- **来源**：字幕直接提到
- **后续建议方向**：实现临时文件夹的自动清理功能；结合 os.listdir 检查目录是否为空后再决定是否删除

### os.getcwd()
- **类型**：函数 / 方法
- **在本 P 中的作用**：返回 Python 脚本当前运行目录的绝对路径，用于确认相对路径的参考点，避免路径相关错误。
- **来源**：字幕直接提到
- **后续建议方向**：在项目开头打印当前工作目录辅助调试；结合 os.chdir() 实现工作目录切换，方便管理文件操作上下文

### os.listdir(目录路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：返回指定目录下所有文件和子目录名称的列表，不传参默认查看当前目录。
- **来源**：字幕直接提到
- **后续建议方向**：遍历目录内容并按文件类型分类统计；结合正则表达式筛选特定格式文件进行批量处理

### os.path.exists(路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：路径检验方法，判断指定路径（文件或目录）是否存在，返回布尔值。支持相对路径和绝对路径。
- **来源**：字幕直接提到
- **后续建议方向**：在执行任何增删改操作前作为安全守卫；批量处理前先检查所有目标路径是否就绪

### os.path.isfile(路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：路径检验方法，判断指定路径是否存在且是文件，两个条件都满足才返回 True。
- **来源**：字幕直接提到
- **后续建议方向**：配合文件处理函数确保只处理真实文件；过滤列表结果中的目录项

### os.path.isdir(路径)
- **类型**：函数 / 方法
- **在本 P 中的作用**：路径检验方法，判断指定路径是否存在且是目录，两个条件都满足才返回 True。
- **来源**：字幕直接提到
- **后续建议方向**：批量处理前验证目标是否为目录；结合 os.listdir 过滤出纯目录列表

## 系统环境 / 工具

### Windows 回收站
- **类型**：操作系统功能
- **在本 P 中的作用**：老师演示打开回收站确认 os.remove 删除的文件不会进入回收站，直接彻底删除，以此强调删除操作不可逆。
- **来源**：字幕直接提到
- **后续建议方向**：重要文件操作前养成手动备份习惯；学习 Windows 文件历史记录或影子拷贝等备份机制

### D:\tools 目录
- **类型**：示例路径 / 测试目录
- **在本 P 中的作用**：作为绝对路径示例，用于演示文件移动和 os.listdir 查看目录内容。帮助理解相对路径与绝对路径的区别。
- **来源**：字幕直接提到
- **后续建议方向**：在本地创建类似结构的测试目录，练习各种路径操作；用真实的项目文件夹替代示例目录进行实战

### E:\A\B 目录结构
- **类型**：说明性路径结构
- **在本 P 中的作用**：演示 os.mkdir 的限制——必须先有父目录 E:\A 才能在下面创建子目录 B，用于说明父目录必须存在这一知识点。
- **来源**：字幕直接提到
- **后续建议方向**：理解后再学习 os.makedirs 方法，该方法可自动创建多级目录结构

## 综合练习建议方向

### 批量文件处理方向
- 设计脚本批量重命名目录下所有 .txt 文件为"文档一.txt"、"文档二.txt"等格式
- 实现按文件类型自动归类（移动图片到 image 目录、文档到 doc 目录等）
- 批量清理过期文件（如删除 7 天前的临时文件）

### 目录结构自动化方向
- 自动创建包含 data、logs、config、backup 等子目录的项目文件夹结构
- 用 os.makedirs 实现多级目录一次性创建
- 生成目录树结构报告并保存

### 安全操作实践方向
- 所有删除操作前必须结合路径检验方法
- 实现"回收站"功能：先移动到临时目录而非直接删除，定期清理
- 添加日志记录每次文件操作，便于追踪和回滚
