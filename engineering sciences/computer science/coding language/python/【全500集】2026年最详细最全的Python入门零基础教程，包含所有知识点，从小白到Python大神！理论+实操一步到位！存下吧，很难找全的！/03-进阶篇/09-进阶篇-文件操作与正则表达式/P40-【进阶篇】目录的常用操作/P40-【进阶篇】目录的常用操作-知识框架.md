# P40 知识框架 - 【进阶篇】目录的常用操作
## 本P位置

- 进阶篇第40P
- 前置：文件读写 + 路径知识
- 后续：批量操作与项目结构自动化

## 核心问题

- 如何用 Python 自动化管理文件和目录？
- 如何在增删改前安全检验路径，避免报错？

## 知识骨架

OS 模块 → 两大类操作（文件 / 目录）+ 路径检验

文件操作：
- os.rename(原路径, 目标路径) → 重命名 or 同磁盘移动
- os.remove(文件路径) → 删除文件

目录操作：
- os.mkdir(目录路径) → 创建空目录
- os.rmdir(目录路径) → 删除空目录
- os.getcwd() → 返回当前工作目录绝对路径
- os.listdir(目录路径) → 返回该目录下所有文件/子目录名列表

路径检验（返回 bool）：
- os.path.exists(路径)
- os.path.isfile(路径)
- os.path.isdir(路径)

## 关系与依赖

所有操作依赖 `import os`，OS 是 Python 内置模块无需安装

文件操作 ↔ 目录操作：互斥，文件方法不能删目录，目录方法不能删文件

路径检验 → 任何增删改操作之前执行，作为安全守卫

os.getcwd() → 为相对路径提供基准参考点

## 输入 / 输出 / 前置 / 支撑

| 方法 | 输入 | 输出 | 前置条件 |
|------|------|------|----------|
| os.rename | 原路径, 目标路径（str） | 无 | 原路径存在，同磁盘 |
| os.remove | 文件路径（str） | 无 | 文件存在且是文件 |
| os.mkdir | 目录路径（str） | 无 | 父目录必须存在 |
| os.rmdir | 目录路径（str） | 无 | 目录存在且为空 |
| os.getcwd | 无 | 绝对路径字符串 | 无 |
| os.listdir | 目录路径（可省略） | 名称列表 | 无 |
| os.path.exists | 路径（str） | bool | 无 |
| os.path.isfile | 路径（str） | bool | 无 |
| os.path.isdir | 路径（str） | bool | 无 |

## 易错点

- os.rename 不支持跨盘移动
- os.remove/os.rmdir 删除后不进入回收站，不可逆
- os.remove 不能删目录，os.rmdir 不能删非空目录
- os.mkdir 父目录不存在会报错；已存在目录再创建报 FileExistsError
- 目标路径已存在同名文件会报错而非覆盖
- 路径检验是防报错的唯一手段

## 外部资源与对象

- D:\tools → 绝对路径示例目录
- E:\A\B → 父目录必须存在的说明性路径
- Python 回收站 → 证明 os.remove 删除文件不经过回收站
- os.makedirs → 进阶方法，支持创建多级目录（课后自学方向）
