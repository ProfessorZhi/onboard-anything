```python
def safe_mkdir(path):

    os.makedirs(path)
    return True
```

**题目7**

思路：
1. 使用 `os.listdir("D:/downloads")` 获取所有文件
2. 用 `endswith(".txt")` 筛选 `.txt` 文件
3. 用 `os.path.isfile()` 确认是文件而非目录
4. 用 `os.makedirs()` 确保目标目录存在
5. 用 `os.rename()` 或字符串拼接构造新路径，完成移动
6. 可选：用 `os.path.join()` 代替手动拼接路径

**题目8**

1. `os.makedirs()` 可以一次性创建多级目录（递归创建），父目录不存在时自动创建；`os.mkdir()` 只能创建一级目录，父目录不存在会报错。

2. 因为不同操作系统路径分隔符不同（Windows 用 `\`，Linux/Mac 用 `/`），`os.path.join()` 会根据操作系统自动选择正确的分隔符，使代码具有跨平台兼容性。
