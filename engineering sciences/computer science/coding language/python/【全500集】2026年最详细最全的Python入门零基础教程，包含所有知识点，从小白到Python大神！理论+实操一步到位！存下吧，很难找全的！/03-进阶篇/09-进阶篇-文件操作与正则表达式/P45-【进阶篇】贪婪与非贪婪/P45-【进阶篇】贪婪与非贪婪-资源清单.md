# P45 资源清单 - 【进阶篇】贪婪与非贪婪
## 资源列表

### re 模块

- 类型：Python 标准库
- 在本 P 中的作用：正则表达式操作的核心工具，所有演示代码均通过 `import re` 引入，作为本 P 所有正则匹配功能的技术底座
- 来源：字幕直接提到（多次出现 `import re` 语句）
- 后续可练习方向：结合实际文本处理场景，综合运用本 P 所学的贪婪/非贪婪模式选择策略处理复杂匹配需求

---

### re.match()

- 类型：re 模块函数
- 在本 P 中的作用：从字符串开头进行匹配，用于演示各量词（`*`、`+`、`{m,n}`）在贪婪与非贪婪模式下的基本行为差异
- 来源：字幕直接提到
- 后续可练习方向：用不同的待匹配字符串和量词组合反复验证匹配结果，加深对两种模式的理解

---

### re.search()

- 类型：re 模块函数
- 在本 P 中的作用：在字符串任意位置搜索匹配，特别用于演示 `A.*B` 类模式在贪婪/非贪婪模式下的差异（贪心模式吞掉整个字符串，非贪心模式精准定位最近目标）
- 来源：字幕直接提到
- 后续可练习方向：模拟实际场景如日志解析、字符串片段提取等，使用 search() 配合不同模式选择策略完成精准匹配

---

本 P 所涉及的外部资源均为 Python 内置标准库，无第三方库依赖，亦无外部网站、文档、论文等其他类型资源。课程全程通过 `re` 模块的两个函数 `re.match()` 与 `re.search()` 完成所有概念演示与实战案例。
