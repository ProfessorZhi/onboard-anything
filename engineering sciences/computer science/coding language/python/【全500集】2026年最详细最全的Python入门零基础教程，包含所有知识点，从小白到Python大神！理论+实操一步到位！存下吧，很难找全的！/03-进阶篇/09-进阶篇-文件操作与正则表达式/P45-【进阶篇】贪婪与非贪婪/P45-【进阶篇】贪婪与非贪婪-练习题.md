## 一、结构识别

**题目 1**  
以下哪些正则量词受贪婪/非贪婪模式的影响？（多选）

- A. `*`
- B. `+`
- C. `?`
- D. `{2,5}`
- E. `.`
- F. `\d`

---

**题目 2**  
将以下描述与其对应的模式连线：

| 描述 | 对应模式 |
|------|----------|
| 能多拿就多拿，直到无法继续 | A. 贪婪模式 |
| 量词后需要加 `?` 来触发 | B. 非贪婪模式 |
| 匹配尽可能长的内容 | C. 两者皆可 |
| 适用于提取单个或最近的片段 | D. 无法确定 |

## 四、扩展思考

**题目 7**  
某同学想用正则 `"<div>内容1</div><div>内容2</div>"` 提取第一个 `<div>` 标签的内容，代码如下：

```python
import re
html = "<div>内容1</div><div>内容2</div>"
result = re.search(r"<div>.*</div>", html)
print(result.group())
```

1. 请问这段代码会输出什么？
2. 如果想只匹配到第一个 `</div>` 结束，应该如何修改正则表达式？
3. 这种场景下，贪婪和非贪婪哪个更合适？为什么？

## 参考答案

<details>
<summary>点击展开答案</summary>

**题目 1**：A、B、C、D（`*、+、?、{2,5}` 受模式影响；`.` 是单字符规则本身不受影响；`\d` 也不受模式影响）

**题目 2**：

- 能多拿就多拿 → A. 贪婪模式
- 量词后需要加 `?` 来触发 → B. 非贪婪模式
- 匹配尽可能长的内容 → A. 贪婪模式
- 适用于提取单个或最近的片段 → B. 非贪婪模式

**题目 3**：

- 贪婪 `a.*b`：匹配 `"aaab"`（从第一个 `a` 一直吞到最后一个 `b`）
- 非贪婪 `a.*?b`：匹配 `"aab"`（满足最低要求就停止，只吞到最近的 `b`）

**题目 4**：

1. 错（默认是贪婪）
2. 对
3. 不受（`.` 是单字符规则，不受量词模式影响）

**题目 5**：
```python
result = re.search(r"Python(.*?)Python", text)
```

**题目 6**：
```python
result2 = re.search(r"1.*?\d", text)
```

**题目 7**：

1. 输出整个字符串的 `<div>内容1</div><div>内容2</div>`（贪婪把两个标签都吞掉了）
2. 将 `.*` 改为 `.*?`：
   ```python
   result = re.search(r"<div>.*?</div>", html)
   ```
3. 这种场景下**非贪婪更合适**，因为只需要提取第一个标签的内容，非贪婪会在遇到第一个 `</div>` 时就停止匹配。
