### 1.1 函数返回值速查

以下代码运行后，变量 `a`、`b`、`c` 的数据类型分别是什么？

```python
import re

text = "手机号: 13812345678，备用: 13987654321"

a = re.search(r"\d+", text)
b = re.findall(r"\d+", text)
c = re.sub(r"\d+", "****", text)

# 请判断 a、b、c 的类型
```

> 考察要点：区分 match对象、list、str 三种返回值类型

## 机制理解

### 2.1 match() vs search()

```python
import re

text = "我的邮箱是 example@python.com"

a = re.match(r"\w+@\w+\.\w+", text)
b = re.search(r"\w+@\w+\.\w+", text)
```

请问 `a` 和 `b` 的值分别是 `None` 还是 match 对象？为什么？

---

### 2.2 findall() 的安全边界

```python
import re

text = "2026年没有匹配的内容"
result = re.findall(r"\d{4}", text)
```

`result` 的值是什么？这种"失败"行为与 `search()` 相比，有什么优点？

### 2.3 sub() 的 count 参数

```python
import re

text = "a1a2a3a4"

r1 = re.sub(r"a", "b", text, count=1)
r2 = re.sub(r"a", "b", text, count=2)
r3 = re.sub(r"a", "b", text)  # 默认
```

请写出 `r1`、`r2`、`r3` 的值。

## 最小替代练习

本 P 使用 Python 标准库 `re` 模块，是内置模块，无需安装外部依赖。

**练习内容**：用 `findall()` + `sub()` 完成一个简单的文本清理器：

```python
import re

raw_text = """
    邮箱列表:
    user1@email.com
    USER2@EMAIL.COM
    user3@Email.com
"""

# 需求：
# 1. 用 findall 提取所有邮箱
# 2. 用 sub 把邮箱统一替换为 [已隐藏]
# 3. 体会 re.I 标志的作用（忽略大小写）
```

## 参考答案（思路）

| 题号 | 核心答案 |
|------|----------|
| 1.1 | a: `None` 或 match对象，b: `list`，c: `str` |
| 1.2 | 邮箱 → `findall`，验证开头 → `match` |
| 2.1 | a 为 `None`（开头不是邮箱），b 为 match对象 |
| 2.2 | `[]`，无需 None 判断，直接遍历不会报错 |
| 2.3 | `b1a2a3a4`、`b1b2a3a4`、`b1b2b3b4` |
| 3.1 | 3个日期，`len(result)` |
| 3.2 | `re.search(r":\s*(.+)$", text).group(1)` |
| 3.3 | `re.sub(r"(\d{3})\d{4}(\d{4})", r"\1****\2", text)` |
| 4.1 | `[]` 可直接遍历，`None` 需先判断，避免 AttributeError |
| 4.2 | `match_obj[1]` 会报错，match对象不是列表，不能用索引 |
| 练习 | `re.findall(r"[\w]+@[\w.]+", text, flags=re.I)` |
