## 参考答案

**结构识别**

- 1-1：B（正则表达式是一套描述字符串模式的规则）
- 1-2：定义规则（正则字符串）→ 执行匹配（调用 re 函数）→ 提取结果（调用 .group()）

**机制理解**

- 2-1：返回 `None`。`re.match()` 从字符串开头匹配，`'hello world'` 开头是 `'hello'`，不是 `'world'`，所以匹配失败
- 2-2：因为 `'\d+'` 匹配数字，字符串 `'abc123'` 开头是字母，匹配失败返回 `None`，对 None 调用 `.group()` 会触发 `AttributeError`。应先判断 `if result:` 再提取
- 2-3：原始字符串 `r''` 阻止反斜杠被 Python 解释为转义符。正则表达式中反斜杠频繁出现（如 `\d`、`\w`），使用 `r''` 可以避免双重转义的困扰和警告

**最小实践**

- 3-1：

```python
import re
text = 'Python3'
result = re.match(r'[A-Za-z]', text)
if result:
    print(result.group())
```

- 3-2：

```python
if result:
    print(result.group())
else:
    print('未匹配到数字')
```

**扩展思考**

- 4-1：应使用 `re.search()`。核心区别是 `re.match()` 只匹配字符串开头，`re.search()` 扫描整个字符串寻找第一个匹配
- 4-2：

```python
import re

def validate_phone(number):
    result = re.match(r'1\d{10}$', number)
    return bool(result)
```
