## 最小实践

### 3.1 基础多选一

补全代码，使用分组多选一匹配 "red"、"green" 或 "blue"：

```python
import re

text = "My favorite color is blue."
pattern = r'____________'
result = re.match(pattern, text)
print(result.group())  # 应输出: blue
```

---

### 3.2 分组引用匹配标签

补全代码，匹配成对出现的 HTML 标签：

```python
import re

text1 = "<span>hello</span>"
text2 = "<div>world</div>"
pattern = r'____________'

for text in [text1, text2]:
    result = re.match(pattern, text)
    if result:
        print(f"匹配: {result.group()}, 标签名: {result.group(1)}")
```

预期输出：
```
匹配: <span>hello</span>, 标签名: span
匹配: <div>world</div>, 标签名: div
```
