## 一、代码语法与规则

### 1. 分组语法 `(规则)`

- **类型**：语法 / 正则表达式构造器
- **在本P中的作用**：用括号将正则的某部分括起来，创建独立的分组单元，是实现多选一和引用分组结果的基础
- **来源**：字幕直接提到
- **建议练习方向**：在简单正则中尝试添加分组，逐步理解分组编号规则（左括号从左到右递增）

### 2. 多选一 `|`

- **类型**：语法 / 正则表达式运算符
- **在本P中的作用**：连接多个候选规则，实现"或"的匹配逻辑，如 `(cat|dog|pig)` 表示匹配 cat、dog、pig 任一选项
- **来源**：字幕直接提到
- **建议练习方向**：设计多选项匹配场景，如验证星期几、月份、常见关键词等

### 3. 引用分组 `\1`（语法层面）

- **类型**：语法 / 正则表达式反向引用
- **在本P中的作用**：在正则中引用第 N 个分组已匹配到的内容，常用于确保首尾一致的场景，如匹配成对标签 `<(\w+)><.*</\1>`
- **来源**：字幕直接提到
- **建议练习方向**：尝试匹配成对出现的符号或重复内容，如引号、引号内嵌套等场景

## 二、Python 内置模块与对象

### 4. `re` 模块

- **类型**：Python 标准库
- **在本P中的作用**：Python 处理正则表达式的唯一内置模块，提供 match()、search()、findall() 等核心函数
- **来源**：字幕直接提到
- **建议练习方向**：系统了解 re 模块其他函数，如 re.search()、re.findall()、re.sub()、re.split() 等，对比各函数的适用场景

### 5. `re.match()` 函数

- **类型**：函数 / re 模块核心方法
- **在本P中的作用**：从字符串开头尝试匹配正则规则，返回 Match 对象或 None
- **来源**：字幕直接提到
- **建议练习方向**：对比 re.match() 与 re.search() 的差异，理解何时用哪个

### 6. `Match` 对象与 `group()` 方法

- **类型**：对象 / 方法
- **在本P中的作用**：re.match() 成功后返回 Match 对象，调用 group() 可提取匹配的完整内容，调用 group(N) 可提取第 N 个分组内容
- **来源**：字幕直接提到
- **建议练习方向**：探索 Match 对象的更多方法，如 .start()、.end()、.span()、.groups()

### 7. Raw String `r''` 字面量前缀

- **类型**：语法 / Python 字符串字面量
- **在本P中的作用**：在字符串前加 r 前缀，使反斜杠不再解释为转义字符，如 `r'<(\w+)><.*</\1>'`，确保正则中的 `\d`、`\w`、`\1` 等按字面意义处理
- **来源**：字幕直接提到
- **建议练习方向**：对比普通字符串与 raw string 对 `\1` 的不同解释，理解为何含反斜杠的正则必须使用 raw string
