**Q5：with 语句的关闭机制**

下面两段代码在功能上等价，但推荐使用哪种？请说明 with 语句在哪种情况下比手动 close() 更可靠。

```python
# 写法 A
f = open("test.txt", "r", encoding="utf-8")
content = f.read()
f.close()

# 写法 B
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
```

提示：考虑 f.read() 执行过程中如果发生异常，两种写法的结果有何不同。
