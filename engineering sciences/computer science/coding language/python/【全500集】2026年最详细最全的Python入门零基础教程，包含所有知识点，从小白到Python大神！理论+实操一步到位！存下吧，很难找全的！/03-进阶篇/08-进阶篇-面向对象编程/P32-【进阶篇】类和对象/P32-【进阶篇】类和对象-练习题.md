# P32 练习题 - 【进阶篇】类和对象
## 一、结构识别

**题目 1-1**  
以下代码定义了 `Person` 类，请标注出各部分名称：

```python
class Person:
    name = "未命名"  # A: __________

    def __init__(self, name):
        self.name = name  # B: __________

    def say_hello(self):  # C: __________
        print(f"你好，我是{self.name}")
```

填空：
- A 处属于类的哪一层？（属性 / 方法）
- B 处 `self.name` 是什么属性？
- C 处定义在类中的函数叫什么？

## 二、机制理解

**题目 2-1**  
阅读代码，说出打印结果的顺序和值：

```python
class Car:
    brand = "未知品牌"  # 类属性

    def __init__(self, brand):
        self.brand = brand  # 实例属性

    def show(self):
        print(self.brand)

car1 = Car("比亚迪")
car2 = Car("特斯拉")

print(Car.brand)      # 结果1
print(car1.brand)     # 结果2
print(car2.brand)     # 结果3
```

**题目 2-2**  
`self` 在以下代码中扮演什么角色？Python 解释器是如何传递它的？

```python
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print(f"{self.name} 在叫")
```

## 三、最小实践

**题目 3-1**  
定义一个 `Phone` 类，满足以下条件：

| 要求 | 说明 |
|------|------|
| 类名规范 | 大驼峰命名 |
| 实例属性 | brand（品牌）、price（价格），在 `__init__` 中初始化 |
| 实例方法 | `show_info()` 打印手机信息 |
| 实例化 | 创建一台 iPhone（品牌="苹果"，价格=7999），调用 `show_info()` |

预期输出格式：
```
品牌: 苹果, 价格: 7999元
```

## 参考答案

**题目 1-1 参考答案**

**题目 2-1 参考答案**

**题目 2-2 参考答案**

`self` 是调用该方法的对象的引用，Python 解释器在调用 `dog.bark()` 时自动将 `dog` 这个对象传递给 `bark` 方法的第一个参数。

**题目 3-1 参考答案**

```python
class Phone:
    def __init__(self, brand, price):
        self.brand = brand
        self.price = price

    def show_info(self):
        print(f"品牌: {self.brand}, 价格: {self.price}元")

iphone = Phone("苹果", 7999)
iphone.show_info()
```
