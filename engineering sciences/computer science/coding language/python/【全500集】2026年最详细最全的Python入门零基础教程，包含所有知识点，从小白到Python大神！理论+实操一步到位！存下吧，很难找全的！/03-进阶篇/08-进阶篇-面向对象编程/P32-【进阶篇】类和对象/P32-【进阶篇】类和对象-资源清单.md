- 在本 P 中的作用：编写和调试类与对象代码，支持代码补全、语法高亮、调试功能，可直观查看对象内存地址

- 在本 P 中的作用：轻量级代码编辑工具，可用于编写类和对象代码，通过插件支持代码补全和调试
