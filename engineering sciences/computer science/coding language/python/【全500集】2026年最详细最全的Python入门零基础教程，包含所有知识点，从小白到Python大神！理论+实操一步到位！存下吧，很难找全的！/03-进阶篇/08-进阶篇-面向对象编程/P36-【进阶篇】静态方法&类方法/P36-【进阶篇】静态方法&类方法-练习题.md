**题目 7**：

| 功能 | 推荐方法 | 理由 |
|------|----------|------|
| 功能A | 类方法 | 可能需要修改类属性（如连接池配置） |
| 功能B | 实例方法 | 需要操作当前连接对象的状态 |
| 功能C | 类方法 | 访问类属性来获取状态 |

**题目 8**（开放性答案，仅供参考）：

1. 代码组织性：将与类逻辑紧密相关的工具函数放在类内部，保持命名空间整洁
2. 继承支持：子类可以覆盖/扩展类方法，模块函数无法做到
3. 多态性：类方法可以被子类继承后改变行为
4. 语义表达：代码即文档，明确表达这是类的一部分而非独立工具
