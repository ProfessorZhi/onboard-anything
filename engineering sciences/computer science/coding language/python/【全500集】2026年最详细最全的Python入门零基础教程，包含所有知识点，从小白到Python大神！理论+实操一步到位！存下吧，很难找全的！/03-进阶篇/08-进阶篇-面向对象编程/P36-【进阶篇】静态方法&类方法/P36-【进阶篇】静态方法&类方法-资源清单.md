## 二、语法元素

**@staticmethod 装饰器**
- 类型：语法元素 / 内置装饰器
- 作用：标记静态方法的关键字，告知Python该方法不依赖类或实例状态，调用时可省略对象创建
- 来源：字幕直接提到（核心教学点）
- 建议方向：对比`@property`、`@classmethod`等装饰器；阅读Python官方文档中关于函数装饰器的章节

**@classmethod 装饰器**
- 类型：语法元素 / 内置装饰器
- 作用：标记类方法的关键字，自动将类本身作为第一个参数传入，用于操作类属性或实现工厂方法
- 来源：字幕直接提到（核心教学点）
- 建议方向：与staticmethod对比记忆；尝试在不同类中定义类方法操作类属性

**cls 参数**
- 类型：语法元素 / 约定参数名
- 作用：类方法的第一个隐式参数，Python自动将类本身绑定到此参数，通过`cls.属性名`访问或修改类属性
- 来源：字幕直接提到
- 建议方向：验证`cls`与`self`的区别；尝试在类方法内创建类的新实例

**self 参数**
- 类型：语法元素 / 约定参数名
- 作用：实例方法的第一个隐式参数，代表调用该方法的具体对象
- 来源：字幕直接提到（前置知识P35）
- 建议方向：对比self与cls的访问权限；验证实例方法中访问类属性的行为
