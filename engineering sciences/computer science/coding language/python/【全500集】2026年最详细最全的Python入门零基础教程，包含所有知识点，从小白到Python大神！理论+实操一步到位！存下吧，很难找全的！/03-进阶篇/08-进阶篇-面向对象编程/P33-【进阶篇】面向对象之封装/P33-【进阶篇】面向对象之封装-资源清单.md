## 资源列表

### 1. Python 官方文档（Classes 章节）

- 类型：官方文档
- 在本P中的作用：本P所有语法的权威参考来源，包括类定义、私有属性`__`的双下划线前缀语法、Name Mangling机制说明
- 来源：字幕直接提到 / 基于上下文推断
- 建议方向：作为语法查阅手册，课程中学到的每个语法点都可在官方文档中验证细节，如私有属性的访问限制范围、方法解析顺序（mro）等进阶话题

### 2. PEP 8 – Style Guide for Python Code

- 类型：编码规范文档
- 在本P中的作用：老师强调“封装是一种编程规范，要遵守规范”，PEP 8是Python社区约定俗成的代码风格标准，其中对类命名、方法命名、私有变量命名有明确建议
- 来源：基于上下文推断
- 建议方向：学习如何规范命名类、方法和属性，理解“公开接口”与“私有实现”在命名层面的体现

### 3. Python 官方教程 – 面向对象编程

- 类型：官方教程章节
- 在本P中的作用：课程的前置知识补充，官方教程对类、实例、self、__init__等基础概念有系统讲解，与本P封装的讲解形成上下承接
- 来源：基于上下文推断
- 建议方向：作为前置知识的复习资料，或在完成封装学习后，阅读官方教程对继承、多态的讲解，完成三大特性的系统学习

### 4. BankAccount 银行账户类（课程示例代码）

- 类型：代码示例 / 练习模板
- 在本P中的作用：本P的核心代码载体，完整演示了如何用私有属性 + 公开接口实现封装，是封装思想的落地实现
- 来源：字幕直接提到
- 建议方向：在本地环境复现并运行该代码，尝试修改验证逻辑（如增加每日取款限额、记录操作日志），加深对封装意义的理解
