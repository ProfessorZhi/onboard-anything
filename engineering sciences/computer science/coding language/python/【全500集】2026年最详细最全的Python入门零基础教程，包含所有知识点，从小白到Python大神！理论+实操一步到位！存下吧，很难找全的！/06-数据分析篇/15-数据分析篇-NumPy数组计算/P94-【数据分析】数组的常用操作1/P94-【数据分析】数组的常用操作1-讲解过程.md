### 三、排序的概念引入与冒泡排序代码演示

**时间区间：3:13 - 4:00**

老师介绍排序的基本概念：给定一组无序数据（如1、2、4、5、3），通过排序可以变成升序或降序排列。回顾在Python基础阶段学习排序时需要编写冒泡排序算法，需要双层for循环嵌套。

**时间区间：5:33 - 6:10**

老师开始演示冒泡排序代码：

```python
import numpy as np
arr = np.array([3, 2, 6, 4, 5, 1])  # 创建一个无序数组
```

**时间区间：6:16 - 9:50**

老师编写冒泡排序的外层循环和内层循环：

```python
for i in range(len(arr)):
    for j in range(i + 1, len(arr)):
        if arr[i] > arr[j]:
            temp = arr[j]
            arr[j] = arr[i]
            arr[i] = temp
```

讲解思路：外层循环控制行数，内层循环从当前位置的下一个开始比较。如果前面的元素比后面大（降序情况下），则交换位置。使用临时变量temp保存要交换的值。

### 四、冒泡排序Bug排查

**时间区间：11:32 - 13:20**

运行代码后发现结果错误。老师主动设置思考环节，让学生观察问题所在，但随后自己揭晓答案。

**Bug分析：**

交换顺序写反了。原代码先执行了 `arr[j] = arr[i]`，导致 `arr[i]` 的原始值被覆盖丢失，然后再用temp赋值时已经没有意义了，最终两个位置的值都变成了原 `arr[i]` 的值。

正确的交换顺序应该是先保存 `arr[i]` 到temp，然后：
1. 先把后面那个赋给前面：`arr[i] = arr[j]`
2. 再把temp赋给后面：`arr[j] = temp`

老师通过图示讲解了数据变化过程，强调临时变量保存的时机至关重要。

## 本P结论

1. **矢量化优势**：数组操作（矢量化）可以替代循环，代码量少且执行速度快一到两个数量级。

2. **排序**：
   - NumPy内置`np.sort()`可直接排序，无需编写冒泡排序
   - 排序默认**升序**，按**行**方向（axis=1）进行
   - 交换元素时注意临时变量的正确使用顺序

3. **转置**：使用`.T`属性可将二维数组的行列互换，即"行变列，列变行"。

4. **数组添加**：
   - `append()`：将数据追加到数组末尾
   - `insert()`：可在指定索引位置插入数据，支持轴参数控制插入方向

5. **轴（axis）的核心概念**：
   - axis=0：表示**垂直**方向（按列操作）
   - axis=1：表示**水平**方向（按行操作）
   - 这是理解后续所有多维数组操作的关键基础
