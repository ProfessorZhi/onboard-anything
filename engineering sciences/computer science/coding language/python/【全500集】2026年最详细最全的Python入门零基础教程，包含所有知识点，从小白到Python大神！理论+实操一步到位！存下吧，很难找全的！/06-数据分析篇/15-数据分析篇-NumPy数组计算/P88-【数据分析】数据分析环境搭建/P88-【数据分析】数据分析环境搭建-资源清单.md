# P88 资源清单 - 【数据分析】数据分析环境搭建
## 一、开发环境核心组件

### 1. Python 解释器

| 属性 | 内容 |
|------|------|
| **名称** | Python 解释器（Python Interpreter） |
| **类型** | 程序 / 运行时环境 |
| **在本P中的作用** | 整个数据分析环境的底层执行引擎，负责运行所有Python代码。解释器本身只是一个"内核"，需要配合第三方库才能完成数据分析任务 |
| **来源** | 字幕直接提到：老师演示从 python.org 下载64位Windows安装包 |
| **建议方向** | 后续编写和运行数据分析脚本前，必须确保解释器路径与PyCharm项目中的设置一致 |

### 2. PyCharm IDE

| 属性 | 内容 |
|------|------|
| **名称** | PyCharm |
| **类型** | 集成开发环境（IDE） |
| **在本P中的作用** | Python代码编辑器，提供代码补全、调试、项目管理等功能。老师推荐安装Professional专业版，功能比社区版更强大 |

### 3. pip

|------|------|
| **名称** | pip（Package Installer for Python） |
| **类型** | 包管理工具 / 命令行工具 |

## 二、数据分析核心库

### 4. NumPy

| 属性 | 内容 |
|------|------|
| **名称** | NumPy（Numerical Python） |
| **类型** | 库 / 数值计算基础库 |
| **在本P中的作用** | 提供高性能的多维数组对象（ndarray）及数组处理工具，是Pandas等高级库的基础。老师明确指出这是第一个需要学习的库，预计3~5次课可掌握 |
| **来源** | 字幕直接提到：列举为数据分析四大核心库之一；安装命令 `pip install numpy` |
| **建议方向** | 理解NumPy数组与Python列表的区别；学习数组的基本操作（索引、切片、运算） |

---

### 5. Pandas

| 属性 | 内容 |
|------|------|
| **名称** | Pandas |
| **类型** | 库 / 数据处理核心库 |
| **在本P中的作用** | 本阶段最重要、最强大的库，被誉为"数据处理的瑞士军刀"。提供切片、索引、分组、聚合、数据清洗、排序统计、与数据库连接、大数据处理、绘图可视化等功能。老师明确这是四库中难度最高、最值得重点讲解的库 |
| **来源** | 字幕直接提到：强调是本阶段最核心的库 |
| **建议方向** | 系统学习Pandas的DataFrame和Series数据结构；掌握数据清洗和统计分析的基本方法 |

---

### 6. Matplotlib

| 属性 | 内容 |
|------|------|
| **名称** | Matplotlib |
| **类型** | 库 / 数据可视化基础库 |
| **在本P中的作用** | Python最基础的绑图库，用于将数据以图表形式（折线图、柱状图、散点图等）可视化展现。在机器学习领域使用广泛，用于结果展示 |
| **来源** | 字幕直接提到：列举为数据分析四大核心库之一 |
| **建议方向** | 学习基础绑图命令；制作折线图、柱状图等常见图表类型 |

---

### 7. Pyecharts

| 属性 | 内容 |
|------|------|
| **名称** | Pyecharts |
| **类型** | 库 / 交互图表库 |
| **在本P中的作用** | 基于Echarts的Python绑图库，生成的图表比Matplotlib更美观、内容更丰富，可实现交互效果。老师评价"画得算比较好的"，是Matplotlib的更高级替代选择 |
| **来源** | 字幕直接提到：与Matplotlib对比时列举；老师展示自己用Pyecharts绘制的图表 |
| **建议方向** | 学习Pyecharts的基本语法；尝试制作可交互的数据可视化图表 |
