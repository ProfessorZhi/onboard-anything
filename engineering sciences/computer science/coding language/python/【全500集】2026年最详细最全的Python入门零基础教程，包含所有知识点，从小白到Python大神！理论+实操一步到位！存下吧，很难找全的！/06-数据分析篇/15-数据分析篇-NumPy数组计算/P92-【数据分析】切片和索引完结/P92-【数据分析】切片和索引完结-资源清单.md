## 核心函数与命令

**enumerate() 函数**

- 类型：Python 内置函数
- 在本 P 中的作用：同时获取数组的索引和元素值，用于优化二维数组遍历写法，避免使用 range(len())
- 来源：字幕直接提到
- 建议方向：在处理需要同时访问索引和值的场景时，练习使用 enumerate() 替代传统的 range(len()) 写法

---

**range() 函数**

- 类型：Python 内置函数
- 在本 P 中的作用：生成整数序列，用于创建循环索引。与切片语法（起始:结束:步长）在形式上进行类比教学
- 来源：字幕直接提到
- 建议方向：对比 range(start, stop, step) 与数组切片 array[start:stop:step] 的参数含义和左闭右开原则

---

**np.arange() 函数**

- 类型：NumPy 库函数
- 在本 P 中的作用：生成一维数组，等同于 Python 内置的 range() 但返回 NumPy 数组，便于后续切片操作
- 来源：字幕直接提到
- 建议方向：练习使用 np.arange() 创建不同范围的数组，如 arange(0, 10)、arange(1, 11)、arange(0, 10, 2)

---

**np.reshape() 函数**

- 类型：NumPy 库函数
- 在本 P 中的作用：将一维数组重新排列为多维数组，本 P 用于创建 4×5 的二维数组进行切片演示
- 来源：字幕直接提到
- 建议方向：练习将一维数组 reshape 为不同维度，如 reshape(2, 10)、reshape(5, 4)、reshape(-1, 2)，理解 -1 参数的自动计算逻辑

---

## 语法要素

**切片语法 array[起始:结束:步长]**

- 类型：NumPy 数组操作语法
- 在本 P 中的作用：实现批量选取数组中连续或不连续元素的核心语法，支持一维和二维数组
- 来源：字幕直接提到
- 建议方向：在 Jupyter Notebook 或 Python 脚本中练习各种切片组合，包括省略参数、负数索引、负数步长等场景

---

**布尔索引语法 array[条件表达式]**

- 类型：NumPy 数组操作语法
- 在本 P 中的作用：使用条件判断自动生成选取掩码，实现条件筛选元素的功能，支持单一条件和复合条件
- 来源：字幕直接提到
- 建议方向：练习使用不同的比较运算符（>、<、>=、<=、==、!=）以及逻辑运算符（&、|）构建复合条件

---

**复合条件表达式 (cond1) & (cond2) / (cond1) | (cond2)**

- 类型：NumPy 布尔运算语法
- 在本 P 中的作用：连接多个条件判断，实现多条件筛选，每个条件需用括号包裹以确保运算优先级正确
- 来源：字幕直接提到
- 建议方向：练习编写包含三个及以上条件的复合表达式，并注意括号的使用位置和数量
