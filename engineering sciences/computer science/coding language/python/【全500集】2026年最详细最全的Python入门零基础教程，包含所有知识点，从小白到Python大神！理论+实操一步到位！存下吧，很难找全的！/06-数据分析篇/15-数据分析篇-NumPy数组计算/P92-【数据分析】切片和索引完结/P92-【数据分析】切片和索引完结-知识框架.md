# P92 知识框架 - 【数据分析】切片和索引完结
## 本P位置

- **课程定位**：NumPy数据分析进阶
- **前置依赖**：P91 索引（取值）
- **后续铺垫**：花式索引 → 矩阵操作基础

## 核心问题

1. 如何批量选取数组中的多个连续元素？
2. 如何用条件判断自动生成选取掩码？
3. 二维数组遍历时索引与元素的区别是什么？

---

## 知识骨架

**切片（Slicing）**

- 一维：`array[起始:结束:步长]`，默认 `0:末尾:1`
- 二维：`array[行切片, 列切片]`，逗号分隔行列
- 负索引：`-1` 为末尾，`-n` 为倒数第 n 个
- 负步长：反向选取，`::-1` 实现倒序
- 原则：左闭右开

**布尔索引（Boolean Indexing）**

- 生成布尔数组：`array > 5` → `[False False True ...]`
- 用布尔数组选元素：`array[array > 5]`
- 复合条件：`&`（且）、`|`（或），条件需括号包裹
- 二维结果自动展平为一维

---

## 关系与依赖

```
索引（P91）
  │
  ├── 二维遍历易错点 → enumerate / range(len())
  │
  ▼
切片（Slicing）
  │
  ├── 一维切片 → [start:stop:step]
  │
  ├── 二维切片 → [行, 列]，逗号分隔
  │
  └── 负数索引 → 倒数第N个，反向选取
  │
  ▼
布尔索引（Boolean Indexing）
  │
  ├── 单一条件 → array[array > 5]
  │
  └── 复合条件 → array[(cond1) & (cond2)]
```

---

## 输入 / 输出 / 前置 / 支撑

- **输入**：NumPy 多维数组
- **输出**：子数组（切片）、展平数组（布尔索引降维后）
- **前置**：索引基础（`arr[0]`、`arr[2,3]`）
- **支撑**：`np.arange()` 创建数组、`np.reshape()` 改变维度

---

## 易错点

| 场景 | 错误 | 正确 |
|------|------|------|
| 二维遍历 | `for item in arr: item[0]` 取子数组第0位 | `for i, item in enumerate(arr): item[0]` 或 `range(len(arr))` |
| 切片范围 | `[2:8]` 误以为包含8 | 左闭右开，8 不包含 |
| 二维切片 | `arr[:2:2]` 只写一个冒号 | 二维需 `arr[:2, :2]`，逗号分隔行列 |
| 复合条件 | `arr > 5 & arr < 10` 无括号 | `arr[(arr > 5) & (arr < 10)]` |
| 布尔索引降维 | 期望保持二维 | 二维布尔索引结果自动展平为一维 |
