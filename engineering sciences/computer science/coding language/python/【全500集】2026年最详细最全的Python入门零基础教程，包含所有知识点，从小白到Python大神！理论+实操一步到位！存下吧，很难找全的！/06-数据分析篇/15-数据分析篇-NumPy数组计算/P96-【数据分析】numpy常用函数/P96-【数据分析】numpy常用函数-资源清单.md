## 核心库

**NumPy**
- 类型：Python 科学计算库
- 在本 P 中的作用：所有统计函数的承载主体，提供了 sum、mean、std、var、min、max、argmin、argmax、cumsum、cumprod、median 等函数的底层实现
- 来源：字幕直接提到
- 后续可练习方向：结合 Pandas 进行数据清洗后的统计分析，使用 Matplotlib 可视化统计结果

## 函数命令

**np.sum**
- 类型：NumPy 函数（聚合命令）
- 在本 P 中的作用：演示累加求和功能，支持 axis 参数控制聚合方向
- 来源：字幕直接提到
- 后续可练习方向：对销售数据进行总量汇总，计算课程总学时

**np.mean**
- 类型：NumPy 函数（聚合命令）
- 在本 P 中的作用：演示算术平均数计算，可用于统计成绩、工资、工时等场景
- 来源：字幕直接提到
- 后续可练习方向：计算班级平均成绩、分析月均消费

**np.std**
- 类型：NumPy 函数（聚合命令）
- 在本 P 中的作用：演示标准差计算，用于量化数据离散程度
- 来源：字幕直接提到
- 后续可练习方向：分析考试成绩稳定性，衡量投资收益率波动

**np.var**
- 类型：NumPy 函数（聚合命令）
- 在本 P 中的作用：演示方差计算，标准差的平方，反映数据分散程度
- 来源：字幕直接提到
- 后续可练习方向：评估数据波动范围，配合 std 使用

**np.min / np.max**
- 类型：NumPy 函数（极值命令）
- 在本 P 中的作用：分别找出数组中的最小值和最大值
- 来源：字幕直接提到
- 后续可练习方向：分析温度数据的最高最低值，找出一组价格中的极值

**np.argmin / np.argmax**
- 类型：NumPy 函数（索引命令）
- 在本 P 中的作用：返回最小值/最大值所在的位置索引，便于后续定位操作
- 来源：字幕直接提到
- 后续可练习方向：在10万行数据中快速定位最高分学生索引，找出股价最高日期

**np.cumsum**
- 类型：NumPy 函数（累积命令）
- 在本 P 中的作用：计算累积和，记录每个位置的累加过程
- 来源：字幕直接提到
- 后续可练习方向：计算资金累积流水，学习进度累积记录

**np.cumprod**
- 类型：NumPy 函数（累积命令）
- 在本 P 中的作用：计算累积积，记录每个位置的累乘过程
- 来源：字幕直接提到
- 后续可练习方向：复利计算，增长率累积效果展示

**np.median**
- 类型：NumPy 函数（聚合命令）
- 在本 P 中的作用：计算中位数，不受极端值影响，适合描述真实中间水平
- 来源：字幕直接提到
- 后续可练习方向：分析收入中位数，比较平均数和中位数差异判断数据分布

**np.arange**
- 类型：NumPy 函数（数组创建命令）
- 在本 P 中的作用：创建连续数值数组，是演示统计函数的输入数据来源
- 来源：字幕直接提到
- 后续可练习方向：快速生成测试数据集，创建时间序列数据

**np.reshape**
- 类型：NumPy 函数（数组变换命令）
- 在本 P 中的作用：将一维数组重塑为多维结构，便于演示 axis 参数的列/行操作差异
- 来源：字幕直接提到
- 后续可练习方向：将扁平数据转换为矩阵形式，准备图像处理输入
