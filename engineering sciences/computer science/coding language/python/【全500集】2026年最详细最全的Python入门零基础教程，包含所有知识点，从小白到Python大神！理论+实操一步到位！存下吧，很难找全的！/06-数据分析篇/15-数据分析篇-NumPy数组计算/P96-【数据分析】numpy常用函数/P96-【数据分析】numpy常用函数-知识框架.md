## 本P位置

Python数据分析 → NumPy → 统计函数模块（最后一节）

本P是NumPy章节的收官课，前置已学：NumPy数组创建、索引切片、axis概念。本P承担将零散函数整合为**统计计算体系**的任务，是后续Pandas/Matplotlib的数据预处理基础。

---

## 核心问题

如何用NumPy快速完成一组数字的**聚合统计**？

具体拆解：
- 总和/均值怎么算
- 离散程度（标准差/方差）怎么量化
- 最值及其位置怎么定位
- 累积过程怎么记录
- 中位数怎么求

---

## 知识骨架

```
NumPy统计函数体系
│
├── 基础聚合（单个值输出）
│   ├── np.sum      → 总和
│   ├── np.mean     → 算术平均数
│   ├── np.median   → 中位数
│   ├── np.std      → 标准差（离散程度）
│   └── np.var      → 方差（离散程度平方）
│
├── 极值类（定位）
│   ├── np.min / np.max     → 找值
│   └── np.argmin / np.argmax → 找索引
│
└── 累积类（过程记录）
    ├── np.cumsum   → 累积和
    └── np.cumprod  → 累积积
```

**axis参数贯穿全部**：axis=0按列，axis=1按行，不指定则展平处理。

---

## 关系与依赖

```
数组对象（np.array）
    │
    ├── 输入：1维或2维数组
    │
    ▼
函数调用：np.函数名(arr, axis=?)
    │
    ├── axis=0：垂直方向（列操作）
    ├── axis=1：水平方向（行操作）
    └── 无axis：标量输出
    │
    ▼
输出：标量 / 1维数组 / 2维数组
```

**依赖链**：arr.shape决定axis的行为 → axis决定聚合方向 → 输出形状由axis决定。

| 类型 | 内容 |

| **输入** | np.array对象（1维或2维） |
| **输出** | 标量（总和/均值）、1维数组（按行/列聚合）、2维数组（保留结构） |

---

## 易错点

1. **axis方向混淆**：axis=0是列方向（向下），axis=1是行方向（向右），容易搞反
2. **二维数组默认行为**：不指定axis时，sum/mean等会把整个数组当一维求和
3. **std/var数学原理**：公式复杂（样本方差vs母体方差），但调用只需知道功能，不需要推导
4. **argmin/argmax返回索引**：返回的是位置索引，不是值，容易误用
5. **cumsum/cumprod输出长度**：与输入数组长度一致，每个位置是累积结果
