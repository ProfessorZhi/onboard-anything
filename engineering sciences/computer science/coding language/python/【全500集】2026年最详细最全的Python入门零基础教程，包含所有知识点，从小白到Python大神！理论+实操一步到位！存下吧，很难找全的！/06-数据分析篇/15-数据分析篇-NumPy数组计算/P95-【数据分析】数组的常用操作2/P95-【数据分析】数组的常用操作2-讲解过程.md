### 二、数组的删除操作（12:00-41:00）

**一维数组的删除**

老师创建一维数组演示删除操作：

```python
ar5 = np.array(np.arange(1, 21))
```

打印显示数组形状后，调用delete方法删除元素。按住Ctrl键点击进入源码查看参数说明，参数object可以传入int整数或列表形式。将索引3传入delete函数：

```python
np.delete(ar5, 3)
```

运行结果显示数组变为[1 2 3 5 6 ...]，原本索引3位置的值4被删除。

**二维数组的删除**

将数组形状改为4行5列：

```python
ar5.reshape(4, 5)
```

调用二维数组的删除：

```python
np.delete(ar5, 2, axis=1)
```

传入索引2和axis=1参数。运行后发现数组形状从4×5变为4×4，宽度变窄。老师对比前后效果，说明确实删除了索引2对应的列。

**轴axis的取值含义**

老师回顾昨天讲过的轴变换概念，强调二维数组删除时axis的含义与直觉相反：

- axis=1：删除列（横向操作，从上往下删除列）
- axis=0：删除行（纵向操作）

通过运行axis=0的删除验证，数组从4行变为3行，确认axis=0表示删除行。老师总结理解概念即可，不用死记硬背，通过写例子能快速回忆起来。

### 三、去除重复数据（41:00-97:00）

**一维数组去重**

创建包含重复数据的数组：

```python
ar6 = np.random.randint(1, 10, 20)
```

打印显示原始数组，其中有7、2等重复出现的值。老师介绍使用np.unique()函数去除重复值：

```python
np.unique(ar6)
```

运行结果显示数组长度从20缩短为8左右，所有值都是唯一的。老师指出unique函数不仅去重，还会自动进行排序，如果需要倒序可以配合之前讲过的排序方法实现。

**二维数组去重尝试**

老师布置思考题：二维数组能否去除重复数据？分析指出每列可能存在重复值（如7在多列出现），也可能某列没有重复值。如果某些列有重复被去除，某些列没有重复保留，会导致列长度不对等，破坏二维数组的矩形结构。

创建二维数组测试：

```python
ar6.reshape(4, 5)
```

分别尝试axis=0和axis=1两种情况运行unique函数，发现结果都只是打乱了顺序，并没有实际去除重复值，也没有变成一维数组。老师总结unique去重主要用于一维数组。

### 四、数组的拼接（合并）（97:00-197:00）

**创建演示数据**

创建两个二维数组用于拼接演示：

```python
arr1 = np.array(np.arange(1, 21), dtype=int).reshape(2, 10)
arr2 = np.array(np.arange(21, 41), dtype=int).reshape(2, 10)
```

打印两个数组观察数据。

**拼接方法介绍**

老师介绍np.concatenate()函数，可以传入两个数组和一个轴参数。点进源码查看，函数名带s表示可以处理多个数组。

**一维数组拼接**

注释掉二维数组代码，改用一维数组演示：

```python
arr3 = np.concatenate((arr1, arr2))
```

一维数组拼接不需要指定axis参数，直接将两个数组合并成长数组。

**二维数组纵向拼接（axis=0）**

默认axis=0，演示纵向拼接效果。将arr1和arr2传入函数：

```python
arr3 = np.concatenate((arr1, arr2), axis=0)
```

结果显示arr1在上、arr2在下的纵向拼接，形状从2×10变为4×10。

**二维数组横向拼接（axis=1）**

将axis改为1：

```python
arr3 = np.concatenate((arr1, arr2), axis=1)
```

结果显示横向拼接，形状从2×10变为2×20，数组变宽了。

**拼接形状匹配规则**

老师演示形状不匹配时的报错情况。创建一个2行10列的数组，尝试与1行11列的数组进行横向拼接（axis=1），结果报错提示形状不匹配。

改用2行5列的数组进行横向拼接验证成功，说明横向拼接时行数必须相同。

然后演示纵向拼接（axis=0）时的情况，创建2行5列和3行5列的数组，纵向拼接成功，说明纵向拼接时列数必须相同。

老师总结数组拼接的规则：合并后的形状必须符合数组要求——可以有任意行数，但每行的列数必须相同。用微信翻译查看错误信息显示为ValueError。

## 本P结论

1. **数组删除**：一维数组直接用np.delete(arr, index)；二维数组需指定axis参数，axis=1删除列，axis=0删除行。

2. **去重排序**：np.unique()函数用于一维数组去重，同时自动排序。二维数组去重实际效果有限，形状结构限制了其应用。

3. **数组拼接**：np.concatenate()函数实现数组合并，axis=0纵向拼接（行追加），axis=1横向拼接（列追加）。拼接时形状必须匹配：纵向拼接要求列数相同，横向拼接要求行数相同。

4. **轴axis理解**：二维数组操作中axis=0表示沿行方向操作（从上到下），axis=1表示沿列方向操作（从左到右）。概念遗忘时可通过写简单例子快速回忆。

5. **pandas重要性**：作为数据分析核心技术，后续数据清洗、分析、绑图都依赖pandas，建议提前预习并多做练习题。
