## 结构识别

**Q1：函数签名对应**
将下列函数与其核心用途连线，并写出各自最关键的一个参数名称：

| 函数 | 用途 |
|---|---|
| np.delete | A. 合并多个数组 |
| np.unique | B. 删除指定元素 |
| np.concatenate | C. 去除重复值 |

> 提示：不要死记，思考每个函数名的英文含义。

|---|---|
| np.delete(arr, 1, axis=0) | ? |
| np.delete(arr, 1, axis=1) | ? |
| np.concatenate((arr, arr), axis=0) | ? |
| np.concatenate((arr, arr), axis=1) | ? |

## 机制理解

**Q3：axis=None 的隐含行为**
对如下二维数组执行 `np.delete(arr, 2, axis=None)`：

```
arr = [[1, 2, 3],
       [4, 5, 6]]
```

1. axis=None 时，NumPy 对数组做了什么预处理？
2. 索引 2 对应的是哪个元素？
3. 最终输出结果是什么形状？

---

**Q4：np.unique 的排序副作用**

1. 输出结果是什么？
2. 原始顺序是否被保留？这在哪些场景下可能引发问题？
3. 如果需要同时知道每个唯一值出现的次数，应传入哪个额外参数？
