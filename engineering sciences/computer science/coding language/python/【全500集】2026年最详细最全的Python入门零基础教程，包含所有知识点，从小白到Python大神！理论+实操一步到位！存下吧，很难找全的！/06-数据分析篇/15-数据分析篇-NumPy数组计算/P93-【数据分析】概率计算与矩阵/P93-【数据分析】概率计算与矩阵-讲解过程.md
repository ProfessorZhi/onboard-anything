# P93 讲解过程 - 【数据分析】概率计算与矩阵
## 本P定位

本P是《Python入门零基础教程》系列中**概率计算与NumPy矩阵操作**的入门章节。前半部分以**正态分布检测异常值**为案例，讲解概率中的均值、标准差概念及NumPy的mean()和std()函数；后半部分转向**NumPy矩阵的创建、加减乘运算**，重点演示矩阵乘法的维度要求和计算规则。承上启下，为后续NumPy统计函数专题和矩阵运算专题打基础。

## 讲解过程

### 一、开场引入：什么是概率

**[0:14-1:02]** 介绍本节三大内容：概率的和事件、正态分布、矩阵。

**[1:02-1:22]** 给出概率的官方定义——研究随机现象背后的客观规律。用通俗例子类比：就像过年回家玩斗牛、炸金花时，估算自己这把赢的概率有多大。

**[1:22-1:54]** 强调概率的本质不是"随机"，而是通过大量随机事件总结出来的**数字模型**。

### 二、掷骰子实验：理解随机事件

**[1:54-2:24]** 以掷骰子为例：每个骰子有六个面，掷N次，每次结果在1到6之间，是非唯一的。骰子落地之前不知道结果，但可以凭感觉猜测大概多少点。

**[2:24-2:42]** 定义一次事件（一次随机试验的结果）本质上就是一个集合，骰子的结果集就是{1,2,3,4,5,6}（7是不可能的）。

### 三、概率的通俗理解

**[2:42-3:10]** 用转盘举例：转盘上面积大的区域中奖概率高，能中奖的都是小概率事件。

**[3:10-3:30]** 给出通俗定义：概率是某事发生的可能性，概率越高越容易发生。

**[3:30-3:42]** 举生活例子：出门看天气感觉会下雨；去两家超市买肉，A超市两次都满意，B超市只有一次满意，再去选大概率也会挨坑。

### 四、正态分布引入：身高举例

**[3:42-3:59]** 提出正态分布，用**身高**作为具体案例。

**[3:59-5:42]** 讲解中国成年男性身高大约在1.7米到1.9米之间，篮球运动员如姚明身高2.23米，远超这个范围，称为**异常值**。从人群中看，姚明这类特别高的人会显得很显眼。异常的极端情况还包括"小人国"——身高只有一米左右的人群。

### 五、代码演示：生成身高数据

**[5:42-6:44]** 开始写代码，首先准备身高数据容器：

```
height = []  # 创建空列表
for i in range(100):  # 取100个人
    height.append(np.random.uniform(1.7, 1.9))  # 用uniform生成1.7-1.9之间的随机身高
```

**[6:44-7:03]** 运行代码查看结果，打印出height列表，包含1.76、1.89、1.85、1.71、1.74、1.75、1.79、1.86、1.89等值，说明分布比较平均。

### 六、添加异常值：姚明与小人国

**[7:03-7:49]** 手动添加两个异常值：

```
height.append(2.23)  # 姚明身高2.23米
height.append(1.0)   # 小人国身高1米
```

### 七、计算均值

**[7:49-8:28]** 引入均值（μ）概念：把所有数加起来除以个数。用简单例子[1,2,3]说明：1+2+3=6，6÷3=2，平均值就是2。

**[8:28-8:58]** 用NumPy计算平均身高：

```
mean_height = np.mean(height)
```

计算结果约为1.7377。

### 八、引入标准差

**[8:58-9:35]** 引出标准差（σ）的概念：比如[1,2,3,4,5]这组数，每个数之间的差都是1，这就是标准差。并预告后面有专门一节课详细讲解标准差函数。

**[9:35-9:58]** 用NumPy计算标准差：

```
std_height = np.std(height)
```

### 九、异常值检测：三倍标准差准则

**[9:58-10:24]** 介绍**3σ准则（三倍标准差准则）**：异常值 = |x - μ| > 3 × σ，即偏离均值超过三倍标准差的点视为异常值。展示公式：`|x - μ| > 3 * σ`

**[10:24-11:03]** 开始写异常值检测代码：

```
result = (x - μ) > 3 * σ
```

其中x是每个身高值，μ是均值，σ是标准差。

**[11:03-11:39]** 完整条件表达式需要加**绝对值**，因为偏离可能比均值高也可能比均值低：

```
result = np.abs(x - mean_height) > std_height * 3
```

这里使用`np.abs()`计算绝对值，因为均值可能是小数，减去后结果也带小数，用绝对值可以取正值进行比较。

**[11:39-12:02]** 打印result查看结果，输出True/False的布尔数组，其中最后两个值为True，对应之前手动添加的2.23米和1米——它们满足异常值条件。

### 十、错误排查：列表无法使用布尔索引

**[12:02-13:32]** 尝试用布尔索引直接取出异常值对应的身高：`height[result]`，运行后报错：**"只有整数标量才可以转换为数组索引"**。

**[13:32-13:44]** 分析错误原因：height是普通Python列表，不支持NumPy的布尔索引功能。

**[13:44-13:57]** 解决方案：将列表转换为NumPy数组：

```
height = np.array(height)
```

**[13:57-14:15]** 再次运行布尔索引，成功取出异常值：2.23和1.0。强调**数据类型转换**的重要性。

### 十一、应用场景扩展

**[14:15-15:30]** 拓展应用场景：

- **班级分数**：通常在60-85分之间，用布尔索引可以找出异常的高分（100分）或低分
- **工资计算**：普通工资3000-5000元，冒出18000元的异常高薪或只有1000元的异常低薪，可以用同样的方法检测出来
- **工厂计件工资**：同理可检测异常值

### 十二、矩阵概念引入

**[15:30-15:52]** 过渡到矩阵部分，声明大学学过高等数学的同学应该了解矩阵——本质上就是**二维数组**。

### 十三、矩阵加法与减法

**[16:00-16:41]** 演示矩阵加法。创建两个二维数组（两行三列），直接用`+`号相加，对应位置的元素相加：1+5=6，2+6=8。

**[16:41-17:11]** 演示矩阵减法，同样是对应位置相减。

**[17:11-17:34]** **重要规则**：矩阵加减法的**行列数必须完全一致**，否则报错`ValueError: operands could not be broadcast together with shapes`。用实际操作验证不同形状的矩阵无法直接相加相减。

### 十四、矩阵乘法：dot函数

**[17:34-17:54]** 矩阵乘法的关键是**NumPy提供了专门的`np.dot()`方法**。两个矩阵直接用`*`相乘或直接`+`都是不行的（报错ValueError）。

**[17:54-18:15]** 正确语法：

```
np.dot(arr1, arr2)
```

### 十五、代码演示：创建两个矩阵

**[18:15-18:54]** 创建矩阵A（2行2列）和矩阵B（5行4列），但后来调整了维度：

```
# 矩阵A: 2行2列，值为1-4
arr1 = np.arange(1, 5).reshape(2, 2)

# 矩阵B: 2行4列，值为5-13
arr2 = np.arange(5, 13).reshape(2, 4)
```

### 十六、错误尝试与修正

**[18:54-19:08]** 先尝试加法，arr1 + arr2，报错维度不一致（2x2 vs 2x4），说明加减法要求形状完全相同。

**[19:08-19:16]** 尝试直接乘法，arr1 * arr2，同样报错。

**[19:16-19:25]** 正确写法：`np.dot(arr1, arr2)`，注意需要**两个参数用逗号分隔**。

### 十七、图解矩阵乘法规则

**[19:25-21:14]** 用画图方式详细讲解矩阵乘法的计算规则。以结果矩阵的第一行第一列元素为例：

- 位置[0,0] = 矩阵A的第一行 × 矩阵B的第一列
- 计算过程：1×5 + 2×9 = 5 + 18 = **23**

再以位置[0,1]为例验证：
- 位置[0,1] = 矩阵A的第一行 × 矩阵B的第二列
- 计算过程：1×6 + 2×10 = 6 + 20 = **26**

**核心规律**：矩阵A的**行**乘以矩阵B的**列**，对应元素相乘再求和。

### 十八、矩阵乘法的维度匹配规则

**[21:14-22:09]** 通过多次调整矩阵维度来验证规则：

- 矩阵A: 2×2
- 矩阵B: 2×4 → 结果: 2×4
- 矩阵B: 1×8 → 维度不匹配
- 矩阵B: 4×2 → 结果: 2×2
- 矩阵B: 3×3 → 结果: 2×3（从1到9）
- 矩阵B: 3×3（从1到9）→ 结果: 2×3，计算验证38 = 1×1 + 2×5 + 3×9 = 1 + 10 + 27 = 38

**[22:09-22:20]** **最终结论**：矩阵乘法的维度规则是——**第一个矩阵的列数必须等于第二个矩阵的行数**。

## 本P结论

1. **概率的本质**：概率是研究随机现象背后的客观规律，通过大量随机事件总结出的数字模型，不是纯粹的"随机"。

2. **3σ准则（异常值检测）**：在正态分布中，偏离均值超过三倍标准差的数据点被视为异常值。公式为：`|x - μ| > 3σ`。

3. **NumPy统计函数**：
   - `np.mean(数组)` 计算均值
   - `np.std(数组)` 计算标准差
   - `np.abs(数组)` 计算绝对值
   - 使用布尔索引`数组[条件]`可以快速筛选出满足条件的数据

4. **数据类型转换**：布尔索引是NumPy数组的特性，普通的Python列表不支持。若要使用布尔索引，需先将列表转换为NumPy数组：`np.array(列表)`

5. **矩阵运算规则**：
   - **加减法**：要求两个矩阵的行列数完全一致
   - **乘法**：使用`np.dot(A, B)`，要求A的列数等于B的行数，结果矩阵的形状为(A的行数, B的列数)
   - **乘法计算规则**：结果矩阵中每个元素 = 第一个矩阵对应行 × 第二个矩阵对应列（对应元素相乘后求和）

6. **应用场景**：均值、标准差、异常值检测可应用于班级成绩分析、工资统计、产品质量检测等日常和工程场景。
