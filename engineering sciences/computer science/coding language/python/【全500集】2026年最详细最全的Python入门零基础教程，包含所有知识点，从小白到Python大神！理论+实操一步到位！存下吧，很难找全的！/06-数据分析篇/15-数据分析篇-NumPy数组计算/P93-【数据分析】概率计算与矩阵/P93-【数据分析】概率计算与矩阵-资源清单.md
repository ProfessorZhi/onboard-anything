## 二、NumPy 函数与方法

### np.mean()

- **类型**：NumPy 统计函数
- **在本P中的作用**：计算数据集的均值（μ），用于描述数据中心趋势；本P中用于计算平均身高
- **来源**：字幕直接提到
- **后续可练习方向**：对不同分布的数据集计算均值，对比 np.average() 与 np.mean() 的差异

### np.std()

- **类型**：NumPy 统计函数
- **在本P中的作用**：计算数据集的标准差（σ），用于描述数据离散程度；与均值配合用于 3σ 异常值检测
- **来源**：字幕直接提到
- **后续可练习方向**：深入理解标准差的数学推导，对比总体标准差与样本标准差（ddof 参数）的区别

### np.abs()

- **类型**：NumPy 数学函数
- **在本P中的作用**：计算绝对值，消除正负方向差异；本P中用于处理均值与数据点的偏差，确保偏差值为正以便与 3σ 比较
- **来源**：字幕直接提到
- **后续可练习方向**：对包含负数的数据集进行绝对值处理，处理复数求模等场景

### np.dot()

- **类型**：NumPy 矩阵运算函数
- **在本P中的作用**：专门用于矩阵乘法运算，实现「行×列」的点积计算；本P中演示了矩阵乘法的正确语法
- **来源**：字幕直接提到
- **后续可练习方向**：对比 np.dot()、@ 运算符、np.matmul() 的异同，处理高维矩阵乘法

### np.arange().reshape()

- **类型**：NumPy 组合方法（数组创建 + 形状重塑）
- **在本P中的作用**：arange() 生成连续数值序列，reshape() 指定二维矩阵形状；本P中用于快速创建指定维度的矩阵
- **来源**：字幕直接提到
- **后续可练习方向**：创建多维数组（3维及以上）、使用 reshape(-1, ...) 自动推断维度、处理 reshape 后数据展平（flatten）

### np.array()

- **类型**：NumPy 类型转换函数
- **在本P中的作用**：将 Python 普通列表转换为 NumPy 数组，启用布尔索引等 NumPy 特有功能；本P中解决列表无法布尔索引的报错
- **来源**：字幕直接提到
- **后续可练习方向**：对比列表与数组的性能差异、探索 np.asarray() 与 np.array() 的区别

## 三、概念与方法

### 布尔索引

- **类型**：NumPy 数据筛选技术
- **在本P中的作用**：用 True/False 布尔数组作为索引，筛选满足条件的数据；本P中用于提取异常值
- **来源**：字幕直接提到
- **后续可练习方向**：多条件组合索引（&、| 运算符）、使用 np.where() 返回索引而非布尔数组、处理多维数组的条件筛选

### 3σ准则（三倍标准差准则）

- **类型**：统计学异常值检测原理
- **在本P中的作用**：判定正态分布中偏离均值超过三倍标准差的点为异常值的经验法则；公式 |x - μ| > 3σ
- **来源**：字幕直接提到（作为正态分布异常值检测的核心方法）
- **后续可练习方向**：用真实数据集验证 3σ 准则的适用性、对比不同阈值（2σ、1.5σ）的检测效果

### 正态分布

- **类型**：概率论核心概念
- **在本P中的作用**：自然界最常见的连续概率分布，本P以此为背景引入异常值检测；强调大多数数据集中在均值附近
- **来源**：字幕直接提到
- **后续可练习方向**：绘制正态分布曲线（配合 matplotlib）、使用 scipy.stats 验证数据是否符合正态分布

## 四、语法与报错信息

### np.abs(x - mean_height) > std_height * 3

- **类型**：NumPy 表达式（条件判断）
- **在本P中的作用**：3σ 异常值检测的完整条件表达式，返回布尔数组标识异常值位置
- **来源**：字幕直接提到
- **后续可练习方向**：封装为函数以便复用、使用其他异常值检测算法（IQR 四分位距）

### ValueError: operands could not be broadcast together with shapes

- **类型**：NumPy 运行时错误
- **在本P中的作用**：矩阵加减法形状不一致时报错，提示两个矩阵的行列数必须完全相同；本P用于验证矩阵运算的形状规则
- **来源**：字幕直接提到
- **后续可练习方向**：阅读官方广播（broadcasting）文档理解规则、排查其他广播场景的报错

### "只有整数标量才可以转换为数组索引"

- **类型**：Python 错误信息
- **在本P中的作用**：列表直接使用布尔索引时报错，提示列表不支持 NumPy 风格的布尔索引；触发解决方案：使用 np.array() 转换
- **来源**：字幕直接提到
- **后续可练习方向**：总结数据类型与索引方式的对应关系、建立错误信息与解决方案的对照表
