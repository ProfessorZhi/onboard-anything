# P93 知识框架 - 【数据分析】概率计算与矩阵
## 本P位置

Python入门零基础教程 → 数据分析专题 → 概率与NumPy矩阵入门。承接统计学基础概念（均值、标准差），为后续NumPy统计函数专题和矩阵运算专题打基础。

## 核心问题

- 如何用概率思维量化"异常"？
- 布尔索引为何要求数组而非列表？
- 矩阵乘法的维度规则是什么？

## 知识骨架

```
概率基础（均值μ、标准差σ）
    ↓
3σ准则：|x - μ| > 3σ → 异常值
    ↓
NumPy统计函数（mean/std/abs）
    ↓
布尔索引（数据筛选）
    ↓
矩阵概念（二维数组）
    ↓
矩阵运算（加减：同形；乘法：np.dot，行×列）
```

## 关系与依赖

| 上层 | 依赖 | 下层 |
|------|------|------|
| 概率概念 | 定义输入 | 均值μ、标准差σ |
| 均值+标准差 | 共同决定 | 3σ准则 |
| 3σ准则 | 提供条件 | 布尔索引 |
| np.array() | 类型转换 | 布尔索引可用 |
| 矩阵乘法 | 规则约束 | np.dot() |

## 输入 / 输出 / 前置 / 支撑

- **输入**：Python列表（height数据）、两个NumPy数组（矩阵A、B）
- **输出**：布尔数组（result）、异常值数组、运算后的矩阵
- **前置**：Python列表基础、NumPy数组创建（np.arange/reshape）
- **支撑**：NumPy库（所有计算的核心）

## 易错点

1. **布尔索引失效**：列表直接用布尔索引报错 → 必须先`np.array(列表)`转换
2. **矩阵加减形状要求**：两个矩阵行列数必须完全相同，否则ValueError
3. **矩阵乘≠逐元素乘**：`arr1 * arr2`是逐元素乘法，矩阵乘必须用`np.dot(arr1, arr2)`
4. **维度不匹配**：`np.dot(A,B)`要求A的列数=B的行数

| 对象 | 类型 | 作用 |
