## 讲解过程

### 1. 回顾昨天的错误，引出正确的创建方式

老师回顾上一集末尾遇到的问题：直接传入列表给pd.DataFrame会报错。强调这种创建方式是不行的，需要先掌握列表外套一层list的方式。后面的课程会再讲其他创建方法。

### 2. 新建pandas包，重启内核

老师新建一个pandas包，将代码重置。然后导入pandas并简写为pd，为后续演示做准备。

### 3. 讲解DataFrame的六种创建方法

#### 方法一：创建空的DataFrame

```python
df = pd.DataFrame()
print(df)
```

输出结果：空DataFrame，列名和索引都为空（NaN）。

#### 方法二：使用NumPy数组创建DataFrame

```python
arr = np.arange(1, 11)
df = pd.DataFrame(arr)
```

输出结果：创建了一个单列DataFrame，行索引自动从0开始排列（0-9），列名也是默认的数值索引。老师用箭头标注：左边是行索引（也叫行号），右边是列名。强调因为没有指定索引，所以自动生成从零开始的索引。

#### 方法三：创建指定索引的DataFrame

```python
df = pd.DataFrame(arr, index=['1', '2', '3', '4', '5', '6', '7', '8', '9'])
```

老师用了键盘最上排的数字键作为索引演示：'1'、'2'、'3'...到'9'共九个。强调这是创建指定索引的DataFrame。

#### 方法四：使用字典创建DataFrame

```python
dict = {'name': ['张三', '李四', '王五'], 'age': [28, 29, 34]}
df = pd.DataFrame(dict)
```

老师构建了一个包含姓名和年龄的字典结构。强调这里对应了行号（0、1、2），并指出这就是昨天去掉外套list后报错的原因——因为字典本身就是键值对结构，去掉list后传字典给DataFrame会报TypeError，需要用list包起来。

#### 方法五：使用zip函数创建DataFrame

老师先构建三个列表：

```python
list1 = ['张三', '李四', '王五', '赵六', '老八', '麻子', '老刀']
list2 = [28, 29, 34, 35, 36, 37, 38]
list3 = ['男', '男', '男', '男', '男', '男', '男']
```

然后用zip组合：

```python
data = list(zip(list1, list2, list3))
df = pd.DataFrame(data)
```

老师解释zip的用法：zip将多个列表打包成元组的列表，每个元组对应一行数据。老师演示了两种写法，第一种直接用zip打包成list：

```python
data = [list1, list2, list3]  # 这种是列表的列表
df = pd.DataFrame(data)
```

老师运行后确认效果相同，强调这就是二维数组结构，外层包列表，里面每个元素是一行数据。老师还演示了用for循环的方式处理zip结果（`list([item for item in data])`），说明zip返回的是元组，外层套list就能转成列表。最后强调：无论数据长度是否一致，都不会报错，缺失的部分会显示为空值（NaN），这一点比NumPy处理得好。

#### 方法六：字典的列表

老师指出这个方法本质上和方法四类似，只是字典作为列表的元素传入。老师强调这六种方法中，常见的就是这几种，重点不在于记住所有创建方式（因为通常是从文件读取数据后直接得到DataFrame对象）。

### 4. DataFrame的常用属性

老师新建一个单元格，开始讲解常用属性。

#### shape属性

```python
df.shape
```

输出：(3, 3)，表示三行三列。

#### dtypes属性

```python
df.dtypes
```

输出各列的数据类型。老师解释object类型等价于string，可以理解为所有元素的父类（在Java中object是所有类的父类）。演示了修改dtype的方式（`df['name'].astype('string')`），修改后数据类型变成StringDtype。老师建议给列命名，修改列名为：姓名、年龄、性别。

#### values属性

```python
df.values
```

返回二维数组形式的所有数据，是一个列表嵌套的结构。

#### index属性

```python
df.index
```

返回索引对象（RangeIndex），可以用`list(df.index)`转成列表。老师演示了两种方式：属性方式（df.index）和方法方式（df.index()），都可以返回索引列表。

#### ndim属性

```python
df.ndim
```

返回维度数。老师回顾DataFrame是二维数据结构，强调之前学的NumPy可以帮助理解这一点。

### 5. head()和tail()方法

老师先尝试运行head()和tail()，但由于数据只有3行，不足以显示效果。于是手动给数据添加更多行：

```python
names = ['张三', '李四', '王五', '赵六', '老八', '麻子', '老刀', '张亮']
ages = [28, 29, 34, 35, 36, 37, 38, 39]
genders = ['男', '男', '男', '男', '男', '男', '男', '男']
data = list(zip(names, ages, genders))
df = pd.DataFrame(data, columns=['姓名', '年龄', '性别'])
```

老师演示了head()默认返回前5行，tail()返回后5行。进一步演示可以传入参数指定行数：head(6)返回6行，tail(1)返回最后1行。老师强调即使传入的参数超过数据行数，也不会报错，会以最大索引显示所有数据。老师提到后续学会读取Excel文件后就不需要手动输入数据了，老师会提供测试数据。

### 6. loc和iloc索引切片

老师新建一个单元格，新建一个df2，开始讲解loc和iloc的区别。

老师引用PPT内容说明：
- loc：通过标签索引获取行数据
- iloc：通过位置获取行的数据

#### loc用法演示

```python
df2.loc[0]  # 获取第一行
df2.loc[1]  # 获取第二行
df2.loc[1:4]  # 切片获取第2到第4行（左闭右开：1, 2, 3）
```

老师强调这里的切片是左闭右开的区间，从1到4实际取到的是索引1、2、3对应的行。

## 本P结论

1. **DataFrame六种创建方法**：空DataFrame、NumPy数组创建、指定索引创建、字典创建、zip函数创建、字典列表创建。其中字典创建需注意外套list，zip创建返回元组列表。

2. **常用属性**：
   - `shape` 返回行数列数元组
   - `dtypes` 返回各列数据类型，object等价于string
   - `values` 返回所有数据的二维数组
   - `index` 返回索引对象
   - `ndim` 返回维度数（DataFrame为二维）

3. **head()和tail()方法**：用于查看数据的前n行和后n行，默认5行。即使参数超出范围也不会报错。

4. **loc和iloc区别**：loc基于标签（索引值）进行索引，支持切片左闭右开；iloc基于位置（数字下标）进行索引，后续会详细展开。

5. **学习建议**：编程要多代入思考，多问为什么，善用搜索引擎和AI辅助，大量练习才能看到效果。
