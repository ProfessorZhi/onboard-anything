本清单整理本P中出现的所有外部资源，涵盖开发工具、依赖库、数据格式与编程规范五大类。

## 开发工具

### PyCharm

- **类型**：IDE（集成开发环境）
- **作用**：本课代码演示的载体，老师全程在PyCharm中新建Python文件、编写Series操作代码、运行调试并观察输出结构
- **来源**：字幕直接提到
- **建议方向**：熟练使用PyCharm进行Python项目管理和调试，掌握其代码补全、错误提示功能

## 核心依赖库

### pandas 1.3.2

- **类型**：Python第三方数据分析包
- **作用**：本课核心学习对象，指定1.3.2版本可确保API调用和参数一致性，避免版本差异导致的排错成本
- **来源**：字幕直接提到
- **建议方向**：通过pip或conda进行版本指定安装 `pip install pandas==1.3.2`，掌握在不同项目中管理包版本的方法

### numpy

- **类型**：Python第三方科学计算包（pandas底层依赖）
- **作用**：pandas建立在numpy之上，Series的底层数据是numpy数组，可以调用.max()、.min()、.mean()、.sum()等统计函数
- **来源**：字幕直接提到（老师强调pandas与numpy的关系）
- **建议方向**：复习numpy数组的基本操作，理解pandas高性能计算的底层原理

---

## 数据格式

### CSV

- **类型**：数据文件格式（Comma-Separated Values）
- **作用**：爬虫常用数据存储格式，pandas擅长处理这类表格型数据，能够同时管理行标签和列标签
- **来源**：字幕直接提到
- **建议方向**：使用pandas进行CSV文件的读写操作 `pd.read_csv()` 和 `.to_csv()`，掌握不同参数（如分隔符、编码）的用法

### Excel文件（xlrd/xlwt）

- **类型**：数据文件格式 + Python库
- **作用**：展示pandas的表格处理能力，xlrd用于读取Excel，xlwt用于写入Excel
- **来源**：基于上下文推断（老师讲解pandas的Excel处理能力时提及）
- **建议方向**：了解xlrd/xlwt的基本用法，学习pandas操作Excel的完整流程（注意：xlrd已停止维护，新项目建议使用openpyxl）

- **类型**：编码规范/命名约定
