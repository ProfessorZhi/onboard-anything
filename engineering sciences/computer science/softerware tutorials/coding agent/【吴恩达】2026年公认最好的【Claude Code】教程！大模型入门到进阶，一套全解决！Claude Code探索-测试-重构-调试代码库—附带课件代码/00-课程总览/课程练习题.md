# Claude Code 教程课程练习题

## 说明

本练习题按知识层次递进组织，分为三类：
- **结构识别题**：检验对概念、术语、组件的认知准确性
- **机制理解题**：考察对工作原理、流程逻辑、关系依赖的理解深度
- **最小实现题**：要求动手完成具体操作，验证实操能力

## 参考答案与评分说明

### 评分标准

| 题型 | 分值 | 评分标准 |
|------|------|----------|
| 结构识别题（1.x） | 每题2分 | 选择题选对得分，概念题按要点给分 |
| 机制理解题（2.x） | 每题3-5分 | 根据理解深度分档给分 |
| 最小实现题（3.x） | 每题5-10分 | 完整性50%，准确性30%，条理性20% |

### 学习建议

- **结构识别题**：建议复习课程中关于基础概念的讲解，确保对术语的准确理解
- **机制理解题**：建议观看具体的实操演示，理解各机制的工作原理和适用场景
- **最小实现题**：建议在本地环境实际动手操作，将理解转化为实践能力
