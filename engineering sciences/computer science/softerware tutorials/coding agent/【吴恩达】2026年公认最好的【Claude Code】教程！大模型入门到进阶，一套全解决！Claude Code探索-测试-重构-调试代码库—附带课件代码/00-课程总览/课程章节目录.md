# Claude Code 教程课程章节目录

本目录按知识依赖关系重新编排，从基础认知到进阶应用，帮助学习者建立系统性理解。

---

## 第一章：入门与基础认知

本章要解决什么问题：理解 Claude Code 是什么、如何安装配置，以及如何快速上手使用它阅读和理解现有代码库。

- P1-Introduction介绍
- P2-什么是Claude Code？
- P3-设置和代码库理解

---

## 第二章：核心开发能力——添加功能

本章要解决什么问题：掌握使用 Claude Code 进行实际功能开发的标准工作流，包括需求分析、代码引用、计划模式、视觉优化等实操技巧。

- P4-添加功能

---

## 第三章：测试驱动开发与代码重构

本章要解决什么问题：建立系统化的错误调试方法，通过编写测试来定位和验证问题，而非盲目复制错误信息；同时学习代码重构技巧。

- P5-测试、错误调试和代码重构

---

## 第四章：并行开发与自定义命令

本章要解决什么问题：解决多个功能同时开发时的文件覆盖问题，掌握自定义命令的创建方法，以及使用 Git 工作树实现多 Claude Code 实例并行工作。

- P6-同时添加多个功能

---

## 第五章：平台集成与自动化

本章要解决什么问题：将 Claude Code 的能力从终端扩展到 GitHub 平台，实现自动代码审查、Issue 驱动修复，以及工具执行前后的自定义钩子逻辑。

- P7-GitHub集成与Hooks探索

---

## 第六章：多场景应用——数据处理

本章要解决什么问题：从 Web 应用开发转向数据科学场景，学习如何使用 Claude Code 重构混乱的 Jupyter 笔记本，实现职责分离和可视化改进，最终生成可部署的 Streamlit 仪表盘。

- P8-重构Jupyter笔记本并创建仪表盘

---

## 第七章：多场景应用——设计工具集成

本章要解决什么问题：掌握使用 MCP（Model Context Protocol）服务器连接外部工具的能力，实现从 Figma 设计原型到可运行 Web 应用的无缝转化。

- P9-基于Figma原型创建Web应用

---

## 第八章：课程总结

本章要解决什么问题：回顾课程核心要点，梳理 Claude Code 的完整能力地图，为后续深入学习和实际应用提供方向指引。

- P10-Conclusion结论

---

## 学习路径建议

1. **新手入门**：按第一章 → 第二章 → 第三章顺序学习，打牢基础
2. **效率提升**：第四章和第五章帮助你提升并行工作能力和团队协作效率
3. **场景拓展**：第六章和第七章展示 Claude Code 在不同领域的应用方式
4. **进阶实战**：建议在学完基础后，选择感兴趣的章节进行实操练习
