### 最小实现题

**题目5**
假设有以下简化RAG系统三个文件：

```python
# tools.py
def search_vectors(query, max_results=5):
    if max_results <= 0:
        return []
    return [f"chunk_{i}" for i in range(min(max_results, 10))]

# rag.py
def query_rag(question: str, max_results: int = 5) -> list:
    chunks = search_vectors(question, max_results)
    return chunks

# ai_generator.py
def generate_response(question: str, max_results: int = 5) -> str:
    chunks = query_rag(question, max_results)
    if not chunks:
        return "抱歉，没有找到相关信息"
    return f"基于{len(chunks)}个相关文档片段回答：..."
```

用pytest编写一个测试，验证：当max_results=0时，系统应返回“抱歉，没有找到相关信息”。
这个测试验证的是外部行为还是内部状态？它对应知识框架中的哪个核心原则？
