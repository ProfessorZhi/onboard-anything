## 最小实现题

### 题目1：基础Hook配置补全
以下是一个Hook配置片段，请补全 matcher 部分，使其仅在执行 git push 命令时触发：

```json
{
  "hooks": {
    "": {
      "command": "echo '开始推送检查'"
    }
  }
}
```

参考答案：

```json
{
  "hooks": {
    "pre-tool-use": {
      "matcher": {
        "tool": "Bash",
        "command": "git push"
      },
      "command": "echo '开始推送检查'"
    }
  }
}
```

## 最小替代练习

### Claude Code Hook机制替代方案
课程演示了通过 .claude/settings.local.json 配置本地Hooks。如果在不使用Claude Code的情况下实现类似效果，可以使用 **Husky** 配合 Git 原生 hooks 目录。

请调研并简要说明：Husky 的 .husky/pre-push 文件与 Claude pre-tool-use Hook 在功能定位上有何相似之处。
