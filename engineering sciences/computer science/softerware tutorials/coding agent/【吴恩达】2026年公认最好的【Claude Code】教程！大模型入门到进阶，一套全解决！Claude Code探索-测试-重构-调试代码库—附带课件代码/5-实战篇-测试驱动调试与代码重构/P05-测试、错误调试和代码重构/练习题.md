## 四、最小替代练习

### 题目7：用Cursor替代Claude Code进行测试驱动开发

**练习目标**：不使用Claude Code，用Cursor（或其他AI编程工具如Copilot）完成以下流程：

1. 给定一个包含Bug的Python函数
2. 先让AI编写测试用例描述预期行为
3. 运行测试，确认失败
4. 让AI根据测试结果定位并修复Bug
5. 再次运行测试，确认通过

**验收标准**：

- 测试文件独立于源码文件
- 至少包含一个`assert`语句
- 修复后运行`pytest`输出`PASSED`
