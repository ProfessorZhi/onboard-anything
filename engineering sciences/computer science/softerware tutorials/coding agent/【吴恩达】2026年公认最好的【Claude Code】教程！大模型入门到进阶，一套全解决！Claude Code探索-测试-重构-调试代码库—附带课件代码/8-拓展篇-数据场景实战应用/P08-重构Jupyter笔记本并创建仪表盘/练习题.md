## 三、最小实现题

### 题目4：Streamlit 仪表盘最小实现

使用以下模拟数据，编写一个最小的 Streamlit 仪表盘：

```python
import pandas as pd
import streamlit as st

# 模拟数据
data = {
    "月份": ["1月", "2月", "3月", "4月"],
    "销售额": [1200, 1500, 900, 1800],
    "客户数": [45, 52, 38, 67]
}
df = pd.DataFrame(data)
```

**要求**：

1. 添加页面标题"销售仪表盘"
2. 添加一个侧边栏年份选择器（选项：2023、2024）
3. 主区域用折线图展示销售额趋势
4. 主区域显示原始数据表格

**考查目标**：掌握 Streamlit 基本组件的使用方式
