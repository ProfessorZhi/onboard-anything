# Claude Code 实战教程——章节目录

---

## 第一章 入门导引：课程全景与认知奠基

**对应分P**：P01 Introduction介绍

**本章要解决什么问题**：在开始技术细节之前，建立对 Claude Code 的整体定位认知，理解 AI 辅助编码工具的发展脉络，明确课程将要覆盖的内容范围和学习路径。

---

## 第二章 工具解剖：智能代理架构与操作入口

**对应分P**：P02 什么是Claude Code？

**本章要解决什么问题**：理解 Claude Code 作为智能代理的核心构成——模型、工具链、运行环境三者如何协同工作；区分 CLI 和 Sonar 两种操作入口的适用场景；认识内置工具集、MCP 可扩展性和记忆系统。

---

## 第三章 启动准备：环境配置与快速上手

**对应分P**：P03 设置和代码库理解

**本章要解决什么问题**：完成 Claude Code 的安装与初始化；学会用 Claude Code 高效理解陌生代码库——用高层次提问替代逐文件浏览；掌握 CLAUDE.md 的持久化记忆机制和 /init、/help、/clear、/compact 等辅助命令。

---

## 第四章 功能开发：上下文管理与计划模式

**对应分P**：P04 添加功能

**本章要解决什么问题**：掌握功能开发的完整工作流程——如何通过 @ 标签精准引用文件、为什么大修改前要先用 Shift+Tab 开启计划模式、如何通过截图反馈法做视觉迭代；理解上下文质量对代码输出的决定性影响。

---

## 第五章 工程保障：测试驱动调试与架构重构

**对应分P**：P05 测试、错误调试和代码重构

**本章要解决什么问题**：建立系统化的调试方法论——先编写测试再定位 Bug，避免复制粘贴到 AI 的无效循环；理解 pytest + mock 测试框架的使用；如何在复杂需求下通过子代理并行探索方案，并完成从单次工具调用到多轮迭代的架构重构。

---

## 第六章 规模化开发：多会话并行与冲突管理

**对应分P**：P06 同时添加多个功能

**本章要解决什么问题**：当需要同时推进多个功能时，如何让多个 Claude Code 实例并行工作而不产生文件覆盖冲突；掌握 Git Worktree 创建隔离工作目录、自定义命令定义可复用指令、以及合并时的冲突解决流程。

---

## 第七章 团队协作：GitHub 集成与操作生命周期控制

**对应分P**：P07 GitHub集成与Hooks探索

**本章要解决什么问题**：突破终端边界，让 Claude Code 在 GitHub 平台参与 PR 审查、Issue 响应等团队协作场景；理解 Claude GitHub App 的安装与配置；掌握 Hooks 生命周期钩子机制，在工具执行前后注入自定义逻辑实现测试触发、权限控制等高级功能。

---

## 第八章 数据工作流：Jupyter 笔记本重构与仪表盘构建

**对应分P**：P08 重构Jupyter笔记本并创建仪表盘

**本章要解决什么问题**：当应用场景从 Web 后端扩展到数据分析时，如何用 Claude Code 处理 Jupyter 笔记本；通过职责分离（数据加载、指标计算、可视化模块化）将混乱笔记本重构为可维护代码；最终无缝转换为可部署的 Streamlit 仪表盘。

---

## 第九章 设计到代码：MCP 协议集成与多工具协同

**对应分P**：P09 基于Figma原型创建Web应用

**本章要解决什么问题**：理解 MCP（Model Context Protocol）作为连接 Claude Code 与外部工具的中间层标准；通过 Figma MCP Server 让 AI 直接“读”设计稿，Playwright MCP Server 做自动化验证，Next.js 构建目标应用，实现“设计 → 生成 → 验证”的完整闭环工作流。

---

## 第十章 技能内化：核心能力闭环与工程方法论

**对应分P**：P10 Conclusion结论

**本章要解决什么问题**：将课程所学提炼为可复用的工程方法论——Explore（探索）、Test（测试）、Refactor（重构）、Debug（调试）四大核心技能的适用场景；理解“清晰指令 + 明确上下文 + 指向相关文件”三条使用原则；掌握通过 CLAUDE.md 配置持久化规范、MCP 扩展能力边界的实践路径。
