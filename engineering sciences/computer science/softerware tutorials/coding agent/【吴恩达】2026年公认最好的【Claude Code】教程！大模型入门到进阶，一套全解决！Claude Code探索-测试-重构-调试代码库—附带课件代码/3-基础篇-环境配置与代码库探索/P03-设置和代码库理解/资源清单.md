## 命令与操作方式

### /init
- 类型：Claude Code内置命令
- 在本P中的作用：自动分析代码库并生成CLAUDE.md的入口命令
- 来源：字幕直接提到
- 后续可替代练习方向：在不同类型的开源项目中运行/init，对比生成的CLAUDE.md质量

### /help
- 类型：Claude Code内置命令
- 在本P中的作用：显示所有命令的快速描述和总结
- 来源：基于上下文推断
- 后续可替代练习方向：输入/help查看完整命令列表，逐个尝试理解功能

### /clear
- 类型：Claude Code内置命令
- 在本P中的作用：完全清除对话历史，从头开始
- 来源：字幕直接提到
- 后续可替代练习方向：在长对话中测试/clear，对比前后的上下文状态

### /compact
- 类型：Claude Code内置命令
- 在本P中的作用：清除历史但保留摘要，保留上下文基础上继续工作
- 来源：字幕直接提到
- 后续可替代练习方向：对比/clear和/compact的效果差异，理解何时选择哪个命令

### Escape键
- 类型：操作方式 / 快捷键
- 在本P中的作用：中断正在执行的命令，主动控制对话节奏

- 后续可替代练习方向：在Claude Code执行耗时任务时练习使用Escape，体会即时中断的反馈

### git add

- 来源：字幕直接提到

### git commit
