## 核心工具

**Claude Code**
- 类型：AI编码助手
- 在本P中的作用：课程核心工具，Andrew Ng称其为“目前最喜欢的编码工具”，代表AI辅助编码的重大飞跃
- 来源：字幕直接提到
- 后续可替代练习方向：安装并完成首个简单任务，理解其基本工作流程

**MCP服务器（Model Context Protocol）**
- 类型：扩展生态/协议
- 在本P中的作用：Claude Code连接外部工具的桥梁，本P提及Figma MCP服务器作为集成示例
- 来源：字幕直接提到
- 后续可替代练习方向：探索MCP生态，了解可集成的工具类型
